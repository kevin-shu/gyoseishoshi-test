# 練習問題格式規範

本文件定義 `docs/practice/` 下練習問題檔案的完整格式。

---

## 檔案架構

```
docs/practice/book3-obligations/
├── ch01-general.md
├── ch02-contract.md
└── ...
```

命名規則與對應教材目錄一致：`chXX-[slug].md`。

---

```markdown

    重要テーマ：[關鍵詞，用中黒「・」連接]

ア　[記述ア，日文]

イ　[記述イ，日文]

ウ　[記述ウ，日文]

エ　[記述エ，日文]

オ　[記述オ，日文]

1. ア・イ
2. ア・ウ
3. イ・エ
4. ウ・オ
5. エ・オ

??? success "解答・解説"
    **正解：X（[組合せ]）**

    ア　○/✗ [解說]

    イ　○/✗ [解說]

    ウ　○/✗ [解說]

    エ　○/✗ [解說]

    オ　○/✗ [解說]

    **中文要點**: ...

    📚 **関連条文**: ...
```

## 解答ブロック詳細規則

### 折疊語法
```
??? success "解答・解説"
    [內容縮排4格]
```

- `???` 為預設折疊（collapsed）
- `!!!` 為預設展開（expanded）
- 練習問題**一律使用 `???`**（避免直接看到答案）

### ○/✗ 標記規範

| 符號 | 意義 |
|------|------|
| `○` | 選項正確（妥当） |
| `✗` | 選項錯誤（妥当でない） |

格式：`1. ✗ [解說文] 。[引用：第XX条、最判XX.X.X]`

判例引用格式：`最判昭40.5.4`、`最判平12.4.14`、`最大判昭XX.X.X`（最高裁大法廷）

### 中文要點寫作原則

- 用繁體中文撰寫
- 整理本題「最重要的1-3個考點」
- 常見形式：
  - 成立要件清單（①②③）
  - 比較表（A vs B）
  - 重要判例要旨（一句話）
  - 常考陷阱提醒（「注意…」「容易混淆…」）

### 条文連結錨點規則

MkDocs 對日文標題產生錨點時，**只保留數字，去掉所有非ASCII字元**：

| 標題 | 錨點 |
|------|------|
| `## 第388条（法定地上権）` | `#388` |
| `### 第3条の2（意思能力）` | `#32` |
| `## 第415条〜第422条の2 債務不履行` | `#415422` |
| `## 第1章 総則` | `#_1`（有時保留底線） |

**建議**：執行 `mkdocs build` 後，到 `site/` 目錄下的 HTML 文件中確認實際錨點值。

連結格式（相對路徑，從 `docs/practice/bookX/chXX.md` 出發）：
```markdown
[第388条〜第389条](../../civil-code/book2-property/ch07-mortgage.md#388)
```

## 題目來源標注規則

| 年號 | 公元 | 格式 |
|------|------|------|
| 令和X年度 | 2019+X-1 | `令和X年度 問XX` |
| 平成XX年度 | 1989+XX-1 | `平成XX年度 問XX` |

- **必須標注真實出處**（年度＋問號）
- 若題目為自行出題（非過去問），標注：`[出題例（非過去問）]`

## 每章題目數量建議

| 重要度 | 建議題數 |
|--------|----------|
| 最重要（毎年出題） | 3-4題 |
| 重要（2-3年に1回） | 2-3題 |
| 補足 | 1-2題 |

行政書士試験の物権分野例：
- 抵当権（毎年）→ 3題
- 占有権・所有権 → 各2題
- 用益物権・質権 → 各2題
