## 2. 試験科目・配点・出題形式

### 法令等科目（46問 / 244点）

| 科目 | 5肢択一式 | 多肢選択式 | 記述式 | 合計 | 全体比率 |
|------|-----------|-----------|--------|------|----------|
| **行政法** | 19問（76点） | 2問（16点） | 1問（20点） | **112点** | **37.3%** |
| **民法** | 9問（36点） | — | 2問（40点） | **76点** | **25.3%** |
| **憲法** | 5問（20点） | 1問（8点） | — | **28点** | 9.3% |
| **商法・会社法** | 5問（20点） | — | — | **20点** | 6.7% |
| **基礎法学** | 2問（8点） | — | — | **8点** | 2.7% |

> 行政法＋民法＝**188点（62.7%）**。この2科目だけで合格点180点を超える配点がある。

### 基礎知識科目（14問 / 56点）— すべて5肢択一式

| 科目 | 配点 |
|------|------|
| 一般知識（政治・経済・社会） | 各4点 |
| 行政書士法等関連諸法令 | 各4点 |
| 情報通信・個人情報保護 | 各4点 |
| 文章理解 | 各4点 |

> 2024年度から旧「一般知識等」が「基礎知識」に名称変更。「行政書士法等の諸法令」が新設された。

### 出題形式まとめ

| 形式 | 問数 | 1問あたり | 合計 |
|------|------|-----------|------|
| 5肢択一式 | 54問 | 4点 | 216点 |
| 多肢選択式 | 3問 | 8点 | 24点 |
| 記述式 | 3問 | 20点 | 60点 |

## 3. 行政法の出題範囲（詳細）

行政法は最重要科目のため、出題範囲を細分化して把握する。

| 分野 | 主な内容 | 出題傾向 |
|------|----------|----------|
| **行政法総論** | 行政行為、行政裁量、行政指導など | 基礎理論として毎年出題 |
| **行政手続法** | 申請に対する処分、不利益処分、行政指導の手続 | 5肢択一で例年約3問 |
| **行政不服審査法** | 審査請求、再調査の請求、裁決 | 条文知識が頻出 |
| **行政事件訴訟法** | 取消訴訟、義務付け訴訟、差止め訴訟 | 記述式でも頻出、判例問題多い |
| **国家賠償法** | 1条責任、2条責任 | 判例中心の出題 |
| **地方自治法** | 議会、長の権限、住民訴訟、条例 | 例年3〜4問出題 |

## 4. 学習戦略

### 必要勉強時間の目安

| 条件 | 時間 |
|------|------|
| 一般的な目安 | 600〜1000時間 |
| 法律初学者 | 800〜1000時間 |
| 法学既修者 | 500〜600時間 |

### 科目別の学習優先順位と時間配分

| 優先度 | 科目 | 配分目安 | 理由 |
|--------|------|----------|------|
| ★★★ | 行政法 | 35% | 最高配点112点、記述式あり |
| ★★★ | 民法 | 25% | 76点、記述式2問あり |
| ★★☆ | 憲法 | 10% | 28点、多肢選択式あり |
| ★★☆ | 基礎知識 | 15% | 56点だが足切りあり、対策必須 |
| ★☆☆ | 商法・会社法 | 10% | 20点、深入りしすぎない |
| ★☆☆ | 基礎法学 | 5% | 8点、範囲広く費用対効果低い |

### 推奨学習順序

```
民法 → 行政法 → 憲法・基礎法学 → 商法・会社法 → 基礎知識
```

民法を最初に学ぶ理由：法律学習の基礎となる概念（権利・義務、契約、不法行為など）を含み、行政法や商法の理解にも役立つ。

## 5. 学習スケジュール（2026年2月〜11月：約9ヶ月）

現在（2026年2月）から試験日（11月8日）まで約9ヶ月。

### Phase 1: 基礎固め（2月〜4月｜3ヶ月）

**目標**: 民法と行政法の基本テキストを1周し、全体像を把握する。

| 月 | 科目 | 学習内容 |
|----|------|----------|
| **2月** | 民法（総則・物権） | 基本テキスト精読 + 対応する過去問 |
| **3月** | 民法（債権・親族相続） | 基本テキスト精読 + 対応する過去問 |
| **4月** | 行政法（総論・行政手続法・行政不服審査法） | 基本テキスト精読 + 対応する過去問 |

- 1日の学習時間目安：平日2〜3時間、休日5〜6時間
- 月あたり：約80〜90時間

### Phase 2: 主要科目の深掘り（5月〜7月｜3ヶ月）

**目標**: 行政法の残り分野を完了し、憲法・商法に着手。過去問を本格的に回す。

| 月 | 科目 | 学習内容 |
|----|------|----------|
| **5月** | 行政法（行政事件訴訟法・国家賠償法・地方自治法） | テキスト精読 + 過去問演習 |
| **6月** | 憲法 + 基礎法学 | テキスト精読 + 過去問演習 |
| **7月** | 商法・会社法 + 基礎知識 | テキスト精読 + 過去問演習 |

- 民法・行政法の2周目復習を並行して行う
- 7月下旬：**試験申込み手続きを忘れずに行う**

### Phase 3: 演習・記述対策（8月〜9月｜2ヶ月）

**目標**: 過去問の反復演習、記述式対策を集中的に行う。

| 月 | 内容 |
|----|------|
| **8月** | 全科目の過去問3周目 + 記述式対策（行政法1問・民法2問） |
| **9月** | 肢別問題集の反復 + 記述式答案練習 + 弱点科目の補強 |

- 記述式は40字程度の論述。正確な条文知識と論点整理が必要。
- 基礎知識科目の足切り対策（文章理解は確実に得点する）

### Phase 4: 直前期・模試（10月〜11月初旬｜約5週間）

**目標**: 模擬試験で実戦力を養い、弱点を最終補強する。

| 時期 | 内容 |
|------|------|
| **10月上旬〜中旬** | 市販模試 or 予備校模試を2〜3回受験 |
| **10月下旬** | 模試の復習 + 間違えた論点の集中復習 |
| **11月1日〜7日** | 最終チェック：条文の確認、記述式の型の確認、時間配分の確認 |
| **11月8日** | **本試験** |

| 科目 | 配点 | 目標点 | 得点率 |
|------|------|--------|--------|
| 行政法 | 112点 | 80点 | 71% |
| 民法 | 76点 | 50点 | 66% |
| 憲法 | 28点 | 20点 | 71% |
| 商法・会社法 | 20点 | 8点 | 40% |
| 基礎法学 | 8点 | 4点 | 50% |
| 基礎知識 | 56点 | 32点 | 57% |
