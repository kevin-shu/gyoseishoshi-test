## 3. 行政法の出題範囲（詳細）

行政法は最重要科目（112点 / 300点 = 37.3%）のため、出題範囲を細分化して把握する。
「行政法」という名前の単一の法律は存在せず、複数の法律・法理論の総称である。

> 公式出題範囲（行政書士試験研究センター）：**行政法の一般的な法理論**並びに**行政手続法**、**行政不服審査法**、**行政事件訴訟法**、**国家賠償法**及び**地方自治法**を中心とする

### 法律別の出題数内訳

| 分野 | 5肢択一 | 多肢選択 | 記述式 | 出題傾向 |
|------|---------|---------|--------|----------|
| **行政法総論** | ~3問 | 頻出 | — | 行政行為・裁量・行政立法の判例 |
| **行政手続法** | ~3問 | — | 稀に出題 | 条文知識中心（全46条） |
| **行政不服審査法** | ~3問 | — | — | 条文知識中心（全87条） |
| **行政事件訴訟法** | ~3-4問 | 時々 | **ほぼ毎年** | 判例＋条文、記述式の最頻出 |
| **国家賠償法** | ~2問 | — | — | 判例中心（全6条） |
| **地方自治法** | ~3-4問 | — | — | 条文知識中心（300条超） |
| その他（情報公開法等） | 0-2問 | — | — | 年度により変動 |

### 行政法の重要度ランキング

**Tier 1: 最重要（Must Master）**

1. **行政事件訴訟法**（昭和37年法律第139号）— 択一 + 記述で推定32〜36点。処分性・原告適格・訴えの利益の「訴訟要件ビッグ3」は必須。
2. **行政手続法**（平成5年法律第88号）— 全46条と比較的短く、条文素読が有効。
3. **行政不服審査法**（平成26年法律第68号）— 行政手続法との対比学習が有効。

**Tier 2: 重要（High Priority）**

4. **行政法総論** — 行政行為の分類、裁量、行政立法等。判例知識が多い。
5. **地方自治法**（昭和22年法律第67号）— 条文数が非常に多いため頻出条文の絞り込みが重要。

**Tier 3: 基礎（Foundation）**

6. **国家賠償法**（昭和22年法律第125号）— 全6条と極めて短いが、1条・2条の判例が頻出。

**Tier 4: 補助（Supplementary）**

7. **情報公開法・公文書管理法等** — 0〜2問、年度により変動。

### 行政法 記述式の出題傾向

記述式 問44 は行政事件訴訟法からの出題が圧倒的に多い：

| 年度 | 出題法律 | テーマ |
|------|----------|--------|
| 令和7年 | 行政事件訴訟法 | 建築確認拒否処分・審査請求裁決の手続的瑕疵 |
| 令和6年 | 行政事件訴訟法 | 無線局免許の競願関係・取消訴訟 |
| 令和5年 | 行政事件訴訟法 | 議会懲罰に対する差止訴訟 |
| 令和4年 | 行政事件訴訟法 | 義務付け訴訟の訴訟要件 |
| 令和3年 | 行政手続法 | 勧告の法的性質・行政指導 |
| 令和元年 | 行政事件訴訟法 | 被告適格・訴訟類型 |
| 平成30年 | 行政事件訴訟法 | 不作為に対する訴訟類型 |

記述式では**被告適格**、**訴訟類型の選択**、**訴訟要件**を40字程度で記述させる問題が典型パターン。

## 3.5 基礎知識科目の出題傾向と対策

基礎知識科目は配点56点だが**足切り（24点未満で不合格）**があるため、戦略的な対策が必須。R6（2024年）から「一般知識等」が「基礎知識」に名称変更され、「行政書士法等関連諸法令」が新設された。

各分類の詳細な核心知識点と考古題は以下の専用ページを参照：

| 分類 | 核心知識点 | 練習問題 |
|------|-----------|---------|
| 一般知識（政治・経済・社会） | [知識点整理](basic-knowledge/ch01-political-economy.md) | [過去問](practice/basic-knowledge/ch01-political-economy.md) |
| 行政書士法等関連諸法令 | [知識点整理](basic-knowledge/ch02-related-statutes.md) | [過去問](practice/basic-knowledge/ch02-related-statutes.md) |
| 情報通信・個人情報保護 | [知識点整理](basic-knowledge/ch03-info-privacy.md) | [過去問](practice/basic-knowledge/ch03-info-privacy.md) |
| 文章理解 | [解法ガイド](basic-knowledge/ch04-reading-comprehension.md) | [解法指南](practice/basic-knowledge/ch04-reading-comprehension.md) |

> **学習優先順位**: ①文章理解 → ②情報通信・個人情報保護 → ③行政書士法等諸法令 → ④一般知識

### 得点戦略（目標：9/14問＝36点）

| 分類 | 出題数 | 目標正答数 | 戦略 |
|------|:------:|:---------:|------|
| 一般知識 | 5～6 | 1～2 | 日常のニュース習慣で対応、深追いしない |
| 諸法令 | 2 | 1～2 | 条文素読、短期間で仕上がる |
| 情報通信・個人情報保護 | 3～4 | **3** | **最重点対策分野**、法令＋IT用語を体系的に |
| 文章理解 | 3 | **3** | **全問正解必須**、読解練習で安定させる |
| **合計** | **14** | **8～10** | 足切り24点を大きく超える36～40点が目標 |

> **学習優先順位**: ①文章理解 → ②情報通信・個人情報保護 → ③行政書士法等諸法令 → ④一般知識

## 4. 学習戦略

### 必要勉強時間の目安

| 条件 | 時間 |
|------|------|
| 一般的な目安 | 600〜1000時間 |
| 法律初学者 | 800〜1000時間 |
| 法学既修者 | 500〜600時間 |

### 科目別の学習優先順位と時間配分

| 優先度 | 科目 | 配分目安 | 理由 |
|--------|------|----------|------|
| ★★★ | 行政法 | 35% | 最高配点112点、記述式あり |
| ★★★ | 民法 | 25% | 76点、記述式2問あり |
| ★★☆ | 憲法 | 10% | 28点、多肢選択式あり |
| ★★☆ | 基礎知識 | 15% | 56点・足切りあり。情報通信＋文章理解で安定得点を確保（→3.5節参照） |
| ★☆☆ | 商法・会社法 | 10% | 20点、深入りしすぎない |
| ★☆☆ | 基礎法学 | 5% | 8点、範囲広く費用対効果低い |

### 推奨学習順序

```
民法 → 行政法 → 憲法・基礎法学 → 商法・会社法 → 基礎知識
```

民法を最初に学ぶ理由：法律学習の基礎となる概念（権利・義務、契約、不法行為など）を含み、行政法や商法の理解にも役立つ。

## 5. 学習スケジュール（2026年2月〜11月：約9ヶ月）

現在（2026年2月）から試験日（11月8日）まで約9ヶ月。

### Phase 1: 基礎固め（2月〜4月｜3ヶ月）

**目標**: 民法と行政法の基本テキストを1周し、全体像を把握する。

| 月 | 科目 | 学習内容 |
|----|------|----------|
| **2月** | 民法（総則・物権） | 基本テキスト精読 + 対応する過去問 |
| **3月** | 民法（債権・親族相続） | 基本テキスト精読 + 対応する過去問 |
| **4月** | 行政法（総論・行政手続法・行政不服審査法） | 基本テキスト精読 + 対応する過去問 |

- 1日の学習時間目安：平日2〜3時間、休日5〜6時間
- 月あたり：約80〜90時間

### Phase 2: 主要科目の深掘り（5月〜7月｜3ヶ月）

**目標**: 行政法の残り分野を完了し、憲法・商法に着手。過去問を本格的に回す。

| 月 | 科目 | 学習内容 |
|----|------|----------|
| **5月** | 行政法（行政事件訴訟法・国家賠償法・地方自治法） | テキスト精読 + 過去問演習 |
| **6月** | 憲法 + 基礎法学 | テキスト精読 + 過去問演習 |
| **7月** | 商法・会社法 + 基礎知識 | テキスト精読 + 過去問演習 |

- 民法・行政法の2周目復習を並行して行う
- 7月下旬：**試験申込み手続きを忘れずに行う**

### Phase 3: 演習・記述対策（8月〜9月｜2ヶ月）

**目標**: 過去問の反復演習、記述式対策を集中的に行う。

| 月 | 内容 |
|----|------|
| **8月** | 全科目の過去問3周目 + 記述式対策（行政法1問・民法2問） |
| **9月** | 肢別問題集の反復 + 記述式答案練習 + 弱点科目の補強 |

- 記述式は40字程度の論述。正確な条文知識と論点整理が必要。
- 基礎知識科目の足切り対策（文章理解は確実に得点する）

### Phase 4: 直前期・模試（10月〜11月初旬｜約5週間）

**目標**: 模擬試験で実戦力を養い、弱点を最終補強する。

| 時期 | 内容 |
|------|------|
| **10月上旬〜中旬** | 市販模試 or 予備校模試を2〜3回受験 |
| **10月下旬** | 模試の復習 + 間違えた論点の集中復習 |
| **11月1日〜7日** | 最終チェック：条文の確認、記述式の型の確認、時間配分の確認 |
| **11月8日** | **本試験** |
