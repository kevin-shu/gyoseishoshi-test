# 第6章 期間の計算 — 練習問題

!!! info "本章出題傾向"
    期間の計算（第138条〜第143条）に関する独立の出題は少ないが、時効・取消権の行使期間等との複合問題として出題されることがある。
    重要テーマ：初日不算入の原則、暦による計算

---

## 択一式

### 問題 1（平成30年度 問29）

期間の計算に関する次の記述のうち、民法の規定に照らし、正しいものはどれか。

1. 期間を時間によって定めたときは、その期間は翌日から起算する。
2. 期間を日によって定めたときは、期間の初日は常に算入される。
3. 期間を週、月又は年によって定めたときは、暦に従って計算し、期間の末日の終了をもって期間が満了する。
4. 期間を月によって定めた場合において、最後の月に応当する日がないときは、その月の前月の末日に期間が満了する。
5. 期間の末日が日曜日、土曜日、国民の祝日に関する法律に規定する休日その他の休日にあたるときでも、その日をもって期間が満了する。

??? success "解答・解説"
    **正解：3**

    1. ✗ 時間によって期間を定めたときは「**即時から起算する**」（民法139条）。翌日からではない。初日不算入の原則は日・週・月・年で定めた場合に適用される。
    2. ✗ 日によって期間を定めたときは、原則として「**初日は算入しない**」（民法140条本文）。ただし、その期間が午前零時から始まるときは初日を算入する（同条ただし書）。
    3. ○ 民法143条1項・141条。暦に従い計算し、期間の末日の終了（午後12時）をもって満了する。
    4. ✗ 最後の月に応当日がないときは「**その月の末日**」に満了する（民法143条2項）。前月の末日ではない。例：1月31日から1か月 → 2月28日（又は29日）に満了。
    5. ✗ 期間の末日が休日にあたるときは「**その翌日**」に期間が満了する（民法142条）。

    **中文要點**: 期間計算的重點整理：①以時間定期間→即時起算；②以日定期間→初日不算入（原則）；③以月定期間→依曆計算，最後月無對應日時以該月末日為滿了日；④末日為假日→順延至翌日。
