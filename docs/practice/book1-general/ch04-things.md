# 第4章 物 — 練習問題

!!! info "本章出題傾向"
    物（第85条〜第89条）に関する問題は、過去10年間で約2〜3回出題。
    重要テーマ：不動産と動産の区別、主物・従物、天然果実・法定果実

---

## 択一式

### 問題 1（平成29年度 問29）

物に関する次の記述のうち、民法の規定及び判例に照らし、妥当でないものはどれか。

1. 土地及びその定着物は不動産であり、不動産以外の物は動産である。
2. 物の所有者が、その物の常用に供するため、自己の所有に属する他の物をこれに附属させたときは、その附属させた物は従物となり、従物は主物の処分に従う。
3. 物から生じる産出物を天然果実といい、天然果実はその元物から分離する時にこれを収取する権利を有する者に帰属する。
4. 物の使用の対価として受けるべき金銭その他の物を法定果実といい、法定果実はこれを収取する権利の存続期間に応じて日割計算により取得する。
5. 主物と従物の関係は、同一の所有者に属する物の間でしか成立しない。

??? success "解答・解説"
    **正解：5**

    1. ○ 民法86条1項・2項のとおり。
    2. ○ 民法87条1項・2項のとおり。
    3. ○ 民法88条1項・89条1項のとおり。
    4. ○ 民法88条2項・89条2項のとおり。
    5. ✗ 判例上、主物と従物は**異なる所有者に属する場合にも**成立しうる（最判昭44.3.28 ガソリンスタンドの地下タンク事件）。民法87条1項は「自己の所有に属する」と規定しているが、判例はこれを要件として厳格には解していない場面がある。

    **中文要點**: 主物與從物的關係，條文上規定「屬於自己所有的物」（第87條1項），但判例在某些場合下，即使所有者不同也承認從物關係的成立。天然果實在分離時歸屬於有收取權的人，法定果實按日割計算取得。

    📚 **関連条文**: [第85条](../../civil-code/book1-general/ch04-things.md#85)

### 問題 2（令和4年度 問29）

民法上の「物」に関する次の記述のうち、民法の規定及び判例に照らし、正しいものはどれか。

1. 民法上の「物」とは有体物をいい、電気やエネルギーは有体物ではないから、いかなる場合にも「物」として扱われない。
2. 不動産とは土地及びその定着物をいうが、立木法による登記がされた立木は独立の不動産として扱われる。
3. 無記名債権は動産とみなされるという規定が現行民法に存在する。
4. 金銭は動産であり、その所有権は占有と分離して存在しうる。
5. 建物は土地の定着物であるから、土地と一体のものとして一個の不動産を構成する。

??? success "解答・解説"
    **正解：2**

    1. ✗ 確かに民法85条は「有体物」と規定するが、刑法上は電気を財物とみなす規定があり（刑法245条）、管理可能性のあるものは物として扱われうる場面がある。
    2. ○ 立木法に基づき登記された立木は、土地とは独立した不動産として取引の対象となる。
    3. ✗ 2020年改正前の旧86条3項には「無記名債権は動産とみなす」という規定があったが、**改正により削除された**。
    4. ✗ 金銭は動産だが、判例上、金銭の所有権は**占有と一致する**（最判昭39.1.24）。占有者＝所有者。
    5. ✗ 日本法では、**建物は土地とは別個の不動産**である。これはドイツ法等と異なる日本法の特徴。

    **中文要點**: 日本法的特點：建物（建築物）與土地是分開的不動產，可以各自獨立交易。金錢的所有權與占有一致（占有者即所有者）。2020年改正刪除了「無記名債權視為動產」的規定。

    📚 **関連条文**: [第86条](../../civil-code/book1-general/ch04-things.md#86)
