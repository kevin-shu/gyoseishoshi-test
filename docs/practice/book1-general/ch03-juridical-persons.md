# 第3章 法人 — 練習問題

!!! info "本章出題傾向"
    法人（第33条〜第37条）に関する問題は、過去10年間で約3〜4回出題。
    重要テーマ：法人の成立（法人法定主義）、法人の能力、権利能力なき社団

---

## 択一式

### 問題 1（平成26年度 問27）

Ａ、Ｂ、ＣおよびＤは、共同で事業を営む目的で「Ｘ会」という団体を設立した。この場合に関する次の記述のうち、民法の規定および判例に照らし、誤っているものはどれか。

1. Ｘ会が権利能力なき社団であり、Ａがその代表者である場合、Ｘ会の資産として不動産があるときは、その不動産の公示方法として、Ａは、Ａ個人の名義で所有権の登記をすることができる。
2. Ｘ会が民法上の組合である場合、Ｘ会の取引上の債務については、Ｘ会の組合財産がその債務のための責任財産になるとともに、組合員であるＡ、Ｂ、ＣおよびＤも、各自が損失分担の割合に応じて責任を負う。
3. Ｘ会が権利能力なき社団である場合、Ｘ会の取引上の債務については、その構成員全員に1個の債務として総有的に帰属し、Ｘ会の社団財産がその債務のための責任財産になるとともに、構成員であるＡ、Ｂ、ＣおよびＤも各自が連帯して責任を負う。
4. Ｘ会が民法上の組合である場合、組合員であるＡ、Ｂ、ＣおよびＤは、Ｘ会の組合財産につき持分権を有するが、Ｘ会が解散して清算が行われる前に組合財産の分割を求めることはできない。
5. Ｘ会が権利能力なき社団である場合、構成員であるＡ、Ｂ、ＣおよびＤは、全員の同意をもって、総有の廃止その他Ｘ会の社団財産の処分に関する定めのなされない限り、Ｘ会の社団財産につき持分権を有さず、また、社団財産の分割を求めることができない。

??? success "解答・解説"
    **正解：3**

    1. ○ 権利能力なき社団は団体名義での登記ができないため、代表者個人の名義で登記する（最判昭47.6.2）。
    2. ○ 組合の場合、組合財産が責任財産となるほか、各組合員も損失分担の割合に応じて個人責任を負う（民法675条）。
    3. ✗ 権利能力なき社団の債務は構成員全員に総有的に帰属するが、**構成員個人が連帯して責任を負うことはない**（最判昭48.10.9）。責任財産は社団財産に限定される。
    4. ○ 組合財産は組合員の共有（合有）であり、清算前に分割を求めることはできない（民法676条3項）。
    5. ○ 権利能力なき社団の財産は構成員の総有に属し、持分権・分割請求権は認められない（最判昭32.11.14）。

    **中文要點**: 權利能力無社團（無法人格的團體）與民法上的組合的最大區別：組合成員對組合債務負個人責任（按損失分擔比例），但權利能力無社團的成員**不負個人責任**，責任財產僅限於社團財產。

    📚 **関連条文**: [第33条](../../civil-code/book1-general/ch03-juridical-persons.md#33)

### 問題 2（令和元年度 問28）

法人に関する次の記述のうち、民法の規定に照らし、正しいものはどれか。

1. 法人は、法律の規定によらなければ成立しないが、法人の設立にあたっては必ず主務官庁の許可を得なければならない。
2. 法人は、定款その他の基本約款で定められた目的の範囲内においてのみ権利を有し義務を負う。
3. 外国法人は、国、国の行政区画および商事会社を除き、日本において認許されることはない。
4. 法人の不法行為能力は否定されており、法人の代表者が職務を行うにつき他人に損害を加えた場合でも、法人自体は損害賠償責任を負わない。
5. 法人の目的の範囲は、定款に明示された目的そのものに限られ、その目的を遂行するために必要な行為は含まれない。

??? success "解答・解説"
    **正解：2**

    1. ✗ 法人法定主義（民法33条1項）により法律の規定が必要だが、すべてが許可主義ではない。一般社団法人・一般財団法人は**準則主義**（登記のみで設立可能）。
    2. ○ 民法34条。法人は「法令の規定に従い、定款その他の基本約款で定められた目的の範囲内において、権利を有し、義務を負う。」
    3. ✗ 民法35条1項。外国法人は、国、国の行政区画及び「外国会社」を除き、日本において認許されない。「商事会社」ではなく「外国会社」が正確な表現。
    4. ✗ 一般社団法人法78条等により、法人の代表者がその職務を行うにつき他人に損害を加えた場合、**法人も損害賠償責任を負う**。
    5. ✗ 法人の目的の範囲は、定款記載の目的に限らず、**その目的を遂行するうえで直接または間接に必要な行為も含まれる**（最判昭27.2.15、八幡製鉄事件 最大判昭45.6.24）。

    **中文要點**: 法人的權利能力受「定款中所定目的範圍」的限制（第34條），但判例上此範圍做廣義解釋，包含為達成目的所直接或間接必要的行為。法人對代表人職務上造成的損害負賠償責任。

    📚 **関連条文**: [第34条](../../civil-code/book1-general/ch03-juridical-persons.md#34)

### 問題 3（平成28年度 問27）

権利能力なき社団に関する次の記述のうち、判例に照らし、妥当でないものはどれか。

1. 権利能力なき社団が成立するためには、団体としての組織を備え、多数決の原則が行われ、構成員の変更にもかかわらず団体そのものが存続し、その組織において代表の方法、総会の運営、財産の管理その他団体としての主要な点が確定していることが必要である。
2. 権利能力なき社団の財産は、構成員の総有に属する。
3. 権利能力なき社団は、民事訴訟法上、当事者能力を有し、その名において訴え、または訴えられることができる。
4. 権利能力なき社団の代表者が社団の名においてした取引上の債務は、その社団の構成員全員に一個の義務として総有的に帰属するとともに、社団の構成員各自に個人的にも帰属する。
5. 権利能力なき社団の資産たる不動産については、社団の代表者が自己の個人名義に所有権の登記をすることができる。

??? success "解答・解説"
    **正解：4**

    1. ○ 最判昭39.10.15。権利能力なき社団の成立要件に関する判例の定式。
    2. ○ 最判昭32.11.14。構成員の共有でも合有でもなく「総有」に属する。
    3. ○ 民事訴訟法29条。権利能力なき社団は代表者の定めがある場合、当事者能力を有する。
    4. ✗ 社団の債務は構成員全員に総有的に帰属するが、**構成員各自に個人的には帰属しない**（最判昭48.10.9）。
    5. ○ 最判昭47.6.2。団体名義の登記はできないため、代表者個人名義で登記する。

    **中文要點**: 權利能力無社團的四要件（最判昭39.10.15）：①有團體的組織；②實行多數決原則；③成員變更不影響團體存續；④代表方法、總會運營、財產管理等主要事項已確定。債務不歸屬於成員個人。

    📚 **関連条文**: [第33条](../../civil-code/book1-general/ch03-juridical-persons.md#33)
