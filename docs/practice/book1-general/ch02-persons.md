# 第2章 人 — 練習問題

!!! info "本章出題傾向"
    人（第3条〜第32条の2）に関する問題は、過去10年間で約8〜10回出題。
    重要テーマ：行為能力、制限行為能力者制度、未成年者・成年被後見人・被保佐人・被補助人、意思能力、住所

---

## 択一式

### 問題 1（令和2年度 問27）

制限行為能力者に関する次の記述のうち、民法の規定および判例に照らし、誤っているものはどれか。

1. 未成年者について、親権を行う者が管理権を有しないときは、後見が開始する。
2. 保佐人は、民法が定める被保佐人の一定の行為について同意権を有するほか、家庭裁判所が保佐人に代理権を付与する旨の審判をしたときには特定の法律行為の代理権も有する。
3. 家庭裁判所は、被補助人の特定の法律行為につき補助人の同意を要する旨の審判、および補助人に代理権を付与する旨の審判をすることができる。
4. 被保佐人が保佐人の同意を要する行為をその同意を得ずに行った場合において、相手方が被保佐人に対して、一定期間内に保佐人の追認を得るべき旨の催告をしたが、その期間内に回答がなかったときは、当該行為を追認したものと擬制される。
5. 制限行為能力者が、相手方に制限行為能力者であることを黙秘して法律行為を行った場合であっても、それが他の言動と相まって相手方を誤信させ、または誤信を強めたものと認められるときは、詐術にあたる。

??? success "解答・解説"
    **正解：4**

    1. ○ 民法838条1号。親権者が管理権を有しない場合は未成年後見が開始する。
    2. ○ 民法13条1項（同意権）、876条の4（代理権付与の審判）。
    3. ○ 民法17条1項（同意を要する旨の審判）、876条の9（代理権付与の審判）。
    4. ✗ 被保佐人に対して催告した場合、期間内に回答がなかったときは「**取り消したものとみなす**」（民法20条4項）。追認ではなく取消しである。
    5. ○ 最判昭44.2.13。単なる黙秘も、他の言動と相まって相手方を誤信させたときは詐術にあたる（民法21条）。

    **中文要點**: 對被保佐人催告時，若期限內未回答，視為「撤銷」而非「追認」。注意：對制限行為能力者本人催告時視為取消；對法定代理人等催告時未回答則視為追認（民法20條）。催告對象不同，擬制效果相反。

    📚 **関連条文**: [第3条の2](../../civil-code/book1-general/ch02-persons.md#32)

### 問題 2（令和4年度 問27）

未成年者に関する次の記述のうち、民法の規定に照らし、正しいものはどれか。

1. 未成年者が法律行為をするには、その法定代理人の同意を得なければならないが、単に権利を得る行為についても同意が必要である。
2. 法定代理人が目的を定めて処分を許した財産は、その目的の範囲内で未成年者が自由に処分することができるが、目的を定めないで処分を許した財産についてはこの限りでない。
3. 未成年者が婚姻をしたときは、これによって成年に達したものとみなされるが、離婚した場合には再び未成年者に復帰する。
4. 法定代理人が未成年者に営業を許可した場合、その営業に関する行為については未成年者は行為能力者として扱われるが、許可を取り消すことはできない。
5. 未成年者が法定代理人の同意を得ずに行った法律行為は、未成年者本人のみが取り消すことができ、法定代理人はこれを取り消すことができない。

??? success "解答・解説"
    **正解：2**

    1. ✗ 「単に権利を得、又は義務を免れる法律行為」については法定代理人の同意は不要（民法5条1項ただし書）。
    2. ○ 民法5条3項のとおり。目的を定めた場合はその範囲内で、目的を定めない場合は自由に処分できる。
    3. ✗ 2022年4月施行の改正民法で婚姻適齢は男女とも18歳となり、成年年齢も18歳に引き下げられたため、婚姻による成年擬制の規定（旧753条）は削除された。
    4. ✗ 法定代理人は営業の許可を取り消し、または制限することができる（民法6条2項）。
    5. ✗ 取消権は、制限行為能力者本人だけでなく、法定代理人も行使できる（民法120条1項）。

    **中文要點**: 法定代理人允許未成年人處分的財產：(1)指定用途的→在該用途範圍內自由處分；(2)未指定用途的→完全自由處分。2022年改正後，成年年齡降至18歲，婚姻擬制成年制度已廢除。

    📚 **関連条文**: [第3条](../../civil-code/book1-general/ch02-persons.md#3)

### 問題 3（令和元年度 問27）

制限行為能力者の相手方の保護に関する次の記述のうち、民法の規定に照らし、正しいものはどれか。

1. 制限行為能力者の相手方は、被保佐人に対し、1か月以上の期間を定めて、その期間内にその保佐人の追認を得るべき旨の催告をすることができ、被保佐人がその期間内にその追認を得た旨の通知を発しないときは、その行為を追認したものとみなされる。
2. 制限行為能力者の相手方は、成年被後見人に対し、1か月以上の期間を定めて、その期間内にその法定代理人の追認を得るべき旨の催告をすることができる。
3. 制限行為能力者の相手方が、制限行為能力者が行為能力者とならない間に法定代理人に催告した場合、その法定代理人が期間内に確答を発しないときは、その行為を取り消したものとみなされる。
4. 制限行為能力者の相手方が、制限行為能力者が行為能力者となった後、その者に催告した場合、その者が期間内に確答を発しないときは、その行為を追認したものとみなされる。
5. 制限行為能力者が行為能力者であることを信じさせるため詐術を用いたときは、その行為を取り消すことができるが、相手方は損害賠償を請求することができる。

??? success "解答・解説"
    **正解：4**

    1. ✗ 被保佐人本人に催告した場合、期間内に回答がなければ「**取り消したものとみなす**」（民法20条4項）。追認ではない。
    2. ✗ 成年被後見人は催告の相手方にできない。催告は**法定代理人に対して**行う（民法20条1項）。
    3. ✗ 法定代理人に催告して確答がない場合は「**追認したものとみなす**」（民法20条2項）。取消しではない。
    4. ○ 行為能力者となった後に本人に催告して確答がないときは「追認したものとみなす」（民法20条1項）。
    5. ✗ 詐術を用いた場合は「取り消すことが**できない**」（民法21条）。取消権が否定される。

    **中文要點**: 催告的擬制效果整理：①對法定代理人催告→未回答視為追認；②行為能力恢復後對本人催告→未回答視為追認；③對被保佐人・被補助人本人催告→未回答視為取消。使用詐術的制限行為能力者喪失取消權。

    📚 **関連条文**: [第3条の2](../../civil-code/book1-general/ch02-persons.md#32)

### 問題 4（平成29年度 問27）

成年後見制度に関する次の記述のうち、民法の規定に照らし、正しいものはどれか。

1. 後見開始の審判は、本人、配偶者、四親等内の親族、後見人、保佐人、補助人、検察官のほか、市町村長も請求することができる。
2. 成年被後見人が日用品の購入その他日常生活に関する行為をした場合であっても、成年後見人はこれを取り消すことができる。
3. 被保佐人が保佐人の同意を得なければならない行為は、民法13条1項に列挙された行為に限られ、家庭裁判所がそれ以外の行為について保佐人の同意を要する旨の審判をすることはできない。
4. 補助開始の審判をするには、本人以外の者の請求による場合であっても、本人の同意を要しない。
5. 任意後見契約法に基づく任意後見契約が登記されている場合であっても、家庭裁判所は、本人の利益のため特に必要があると認めるときに限り、後見開始の審判等をすることができる。

??? success "解答・解説"
    **正解：5**

    1. ✗ 後見開始の審判は記述の者が請求できるが、**市町村長**の請求権は民法ではなく老人福祉法・知的障害者福祉法・精神保健福祉法に基づく。民法の規定に照らした場合は誤り。ただし設問の文脈次第では正解にもなりうるが、本問では5が正解。
    2. ✗ 日用品の購入その他日常生活に関する行為は取り消すことが**できない**（民法9条ただし書）。
    3. ✗ 家庭裁判所は、民法13条1項所定の行為以外についても保佐人の同意を要する旨の審判をすることが**できる**（民法13条2項）。
    4. ✗ 補助開始の審判には**本人の同意を要する**（民法15条2項）。
    5. ○ 任意後見契約法10条1項。任意後見契約が登記されていても、本人の利益のため特に必要があると認めるときは法定後見の審判ができる。

    **中文要點**: 成年被後見人的日常生活行為（日用品購入等）不可撤銷。補助開始的審判必須取得本人同意。即使有任意後見契約，為了本人利益特別必要時，仍可進行法定後見。

    📚 **関連条文**: [第3条の2](../../civil-code/book1-general/ch02-persons.md#32)

### 問題 5（平成27年度 問27）

意思能力および行為能力に関する次の記述のうち、民法の規定に照らし、妥当でないものはどれか。

1. 法律行為の当事者が意思表示をした時に意思能力を有しなかったときは、その法律行為は無効である。
2. 意思能力を欠く状態でなされた法律行為の無効は、誰でも主張することができる。
3. 成年被後見人の法律行為は取り消すことができるが、日用品の購入その他日常生活に関する行為については取り消すことができない。
4. 制限行為能力者の相手方は、制限行為能力者が行為能力者とならない間に、その法定代理人に対し催告をすることができるが、成年被後見人に対しては直接催告をすることはできない。
5. 被保佐人が、保佐人の同意を要する行為について、保佐人の同意またはこれに代わる家庭裁判所の許可を得ないでした行為は、取り消すことができる。

??? success "解答・解説"
    **正解：2**

    1. ○ 民法3条の2（2020年改正で明文化）。
    2. ✗ 意思能力を欠く法律行為の無効は、意思能力を欠いていた者（またはその代理人・承継人）の保護を目的とするため、**主張できる者は限定される**。2020年改正後の通説では、意思無能力者側のみが無効を主張できるとされる。
    3. ○ 民法9条本文・ただし書。
    4. ○ 民法20条。成年被後見人には催告能力がないため、直接の催告はできない。
    5. ○ 民法13条4項。

    **中文要點**: 意思無能力的法律行為雖然「無效」，但2020年改正後的通說認為，只有意思無能力者一方才能主張無效（相對的無效）。這與公序良俗違反（任何人都可以主張的絕對無效）不同。

    📚 **関連条文**: [第3条の2](../../civil-code/book1-general/ch02-persons.md#32)

### 問題 6（令和5年度 問27）

住所に関する次の記述のうち、民法の規定及び判例に照らし、正しいものはどれか。

1. 民法上の住所は、市区町村に届け出た住民登録上の住所と常に一致する。
2. 人は同時に二つ以上の住所を有することはできない。
3. 各人の生活の本拠をその者の住所とする。
4. 住所が知れない場合には、本籍地をもって住所とみなす。
5. 日本に住所を有しない者について、日本における最後の住所をもって住所とみなすことはない。

??? success "解答・解説"
    **正解：3**

    1. ✗ 住民登録上の住所と民法上の住所は必ずしも一致しない。民法上の住所は実質的な生活の本拠による（最判昭35.3.22）。
    2. ✗ 生活の本拠が複数ある場合、同時に二つ以上の住所を有することも認められうる（通説）。
    3. ○ 民法22条のとおり。「各人の生活の本拠をその者の住所とする。」
    4. ✗ 住所が知れない場合は「**居所**」を住所とみなす（民法23条1項）。本籍地ではない。
    5. ✗ 日本に住所を有しない者は、「日本における最後の住所」をもって住所とみなすことがある（民事訴訟法等）。

    **中文要點**: 民法上的住所是「生活的本據」（第22條），與住民登記上的住所不一定一致。住所不明時以「居所」視為住所，不是本籍地。

    📚 **関連条文**: [第3条](../../civil-code/book1-general/ch02-persons.md#3)

## 記述式

### 問題 1（令和3年度 問45）

Aは17歳の未成年者であり、法定代理人Bの同意を得ずに、自己が所有する腕時計をCに売却した。Cは、Aが制限行為能力者であることを知らなかった。この場合において、Cは、Aに対してどのような催告をすることができ、Aが確答を発しなかったときはどのような効果が生じるか。40字程度で記述しなさい。

??? success "解答・解説"
    **模範解答**

    > 「Cは、Aが成年に達した後に1か月以上の期間を定めて追認するかどうか催告でき、確答がなければ追認したものとみなされる。」（55字）

    **別解**

    > 「Cは、法定代理人Bに対し1か月以上の期間を定めて追認の催告ができ、確答なき場合は追認とみなされる。」（48字）

    **解説**

    未成年者に直接催告するのではなく、①法定代理人Bに催告する方法（民法20条2項→確答なければ追認擬制）と、②Aが成年に達してから本人に催告する方法（民法20条1項→確答なければ追認擬制）の2通りがある。

    未成年者本人（行為能力を有しない間）に催告することはできない。成年被後見人に対しても同様である。

    **中文要點**: 對未成年人的催告有兩種方式：(1)對法定代理人催告；(2)等未成年人成年後對本人催告。兩者在未回答時都視為追認。不能直接對尚未成年的本人催告。

    📚 **関連条文**: [第3条の2](../../civil-code/book1-general/ch02-persons.md#32)
