# 第5章 法律行為 — 練習問題

!!! info "本章出題傾向"
    法律行為（第90条〜第137条）は民法総則の最重要分野。過去10年間で毎年2〜4問出題。
    重要テーマ：公序良俗、意思表示の瑕疵（心裡留保・虚偽表示・錯誤・詐欺・強迫）、代理（有権代理・無権代理・表見代理）、無効・取消し、条件・期限

---

## 択一式

### 問題 1（平成27年度 問28）

意思表示に関する次の記述のうち、民法の規定及び判例に照らし、妥当なものはどれか。

1. 心裡留保による意思表示は、原則として有効であるが、相手方が表意者の真意を知り、又は知ることができたときは無効となり、この無効は善意の第三者にも対抗することができる。
2. 通謀虚偽表示による意思表示は無効であり、この無効は善意の第三者に対抗することができないが、ここにいう第三者とは、虚偽表示の当事者及びその包括承継人以外のすべての者をいう。
3. 錯誤による意思表示は、表意者に重大な過失があった場合には、相手方が表意者の錯誤を知り又は重大な過失により知らなかったときであっても、取り消すことができない。
4. 詐欺による意思表示の取消しは、善意でかつ過失がない第三者に対抗することができない。
5. 強迫による意思表示の取消しは、善意の第三者に対抗することができない。

??? success "解答・解説"
    **正解：4**

    1. ✗ 心裡留保の無効は「**善意の第三者に対抗できない**」（民法93条2項、類推適用）。対抗できるとする本肢は誤り。
    2. ✗ 第三者は「虚偽表示の当事者及びその包括承継人以外のすべての者」ではなく、**虚偽表示に基づいて新たに独立した法律上の利害関係を有するに至った者**に限られる（最判昭45.7.24）。
    3. ✗ 2020年改正後の民法95条3項。表意者に重大な過失があっても、**相手方が錯誤を知り又は重大な過失により知らなかったとき**は取消しが認められる（95条3項2号）。
    4. ○ 民法96条3項。詐欺による取消しは「善意でかつ過失がない第三者」に対抗できない。2020年改正で「善意の第三者」から「善意でかつ過失がない第三者」に変更された。
    5. ✗ **強迫による取消しは善意の第三者にも対抗できる**（民法96条3項の反対解釈）。強迫は詐欺よりも表意者の保護が厚い。

    **中文要點**: 意思表示瑕疵的第三者保護比較：心裡留保→善意第三者受保護；虛偽表示→善意第三者受保護；錯誤→不涉及第三者保護規定；詐欺→善意無過失的第三者受保護；強迫→第三者不受保護（強迫的表意者保護最厚）。

    📚 **関連条文**: [第90条](../../civil-code/book1-general/ch05-juridical-acts.md#90)

### 問題 2（令和2年度 問28）

虚偽表示に関する次の記述のうち、民法の規定及び判例に照らし、妥当なものはどれか。

1. ＡがＢと通謀してＡ所有の甲土地をＢに仮装譲渡し、Ｂが甲土地をＣに譲渡した場合、Ｃが善意であれば、Ｃが登記を備えていなくても、ＡはＣに虚偽表示による無効を対抗することができない。
2. ＡがＢと通謀してＡ所有の甲土地をＢに仮装譲渡し、Ｂが甲土地上に抵当権を設定してＤから融資を受けた場合、Ｄが善意であっても、ＡはＤに虚偽表示による無効を対抗することができる。
3. ＡがＢと通謀してＡ所有の甲土地をＢに仮装譲渡した場合において、Ｂの債権者Ｅが甲土地を差し押さえたとき、Ｅが善意であっても、ＡはＥに虚偽表示による無効を対抗することができる。
4. ＡがＢと通謀してＡ所有の甲土地をＢに仮装譲渡した後、ＡがＣに甲土地を譲渡した場合、ＣはＢに対して、虚偽表示の無効を主張して、所有権の移転登記の抹消を請求することができない。
5. ＡがＢと通謀してＡ所有の甲土地をＢに仮装譲渡し、Ｂが甲土地を善意のＣに譲渡し、さらにＣが悪意のＦに譲渡した場合、ＡはＦに虚偽表示による無効を対抗することができる。

??? success "解答・解説"
    **正解：1**

    1. ○ 虚偽表示の善意の第三者（民法94条2項）は、登記なくして保護される（最判昭44.5.27）。
    2. ✗ 抵当権者Ｄも「第三者」に含まれ、Ｄが善意であればＡは無効を対抗**できない**。
    3. ✗ 差押債権者Ｅも「第三者」に含まれ（最判昭48.6.28）、Ｅが善意であればＡは無効を対抗**できない**。
    4. ✗ 虚偽表示は無効であるから、甲土地の所有権はＡにある。ＡからＣへの譲渡は有効であり、ＣはＢに対してＢ名義の登記の抹消を請求**できる**（最判昭42.10.31 類推）。
    5. ✗ 善意の第三者Ｃが介在した後の転得者Ｆは、Ｆ自身が悪意でもＡは無効を対抗**できない**（絶対的構成、最判昭45.7.24の趣旨）。

    **中文要點**: 虛偽表示的善意第三者不需要登記即受保護。善意的第三者包括：買受人、抵當權人、差押債權人等。善意第三者介入後的轉得者，即使是惡意也受保護（絕對的構成）。

    📚 **関連条文**: [第90条](../../civil-code/book1-general/ch05-juridical-acts.md#90)

### 問題 3（令和3年度 問28）

錯誤に関する次の記述のうち、民法の規定に照らし、妥当なものはどれか。

1. 意思表示が法律行為の目的及び取引上の社会通念に照らして重要な錯誤に基づくものであるときは、その意思表示は無効となる。
2. 意思表示に対応する意思を欠く錯誤に基づく意思表示の取消しは、善意無過失の第三者に対抗することができない。
3. 表意者が法律行為の基礎とした事情についてのその認識が真実に反する錯誤の場合、その事情が法律行為の基礎とされていることが表示されていなくても取消しが認められる。
4. 錯誤による取消しは、表意者に重大な過失があった場合には、いかなる場合も主張することができない。
5. 錯誤による取消しは、第三者も主張することができる。

??? success "解答・解説"
    **正解：2**

    1. ✗ 2020年改正により、錯誤の効果は「無効」ではなく「**取消し**」に変更された（民法95条1項）。
    2. ○ 民法95条4項。錯誤による取消しは「善意でかつ過失がない第三者に対抗することができない」。
    3. ✗ 動機の錯誤（基礎事情の錯誤）は、その事情が法律行為の基礎とされていることが「**表示されていたとき**」に限り取消しが認められる（民法95条2項）。
    4. ✗ 表意者に重大な過失があっても、①相手方が錯誤を知り又は重過失で知らなかった場合、②相手方が同一の錯誤に陥っていた場合は取消しが認められる（民法95条3項1号・2号）。
    5. ✗ 錯誤による取消しは**表意者又はその代理人若しくは承継人のみ**が主張できる（民法120条2項）。

    **中文要點**: 2020年改正的錯誤制度重點：①效果從「無效」改為「取消」；②動機錯誤（基礎事情錯誤）必須有表示才能取消；③表意者有重大過失時原則不能取消，但相手方知情或同錯時例外可取消；④善意無過失的第三者受保護。

    📚 **関連条文**: [第90条](../../civil-code/book1-general/ch05-juridical-acts.md#90)

### 問題 4（平成28年度 問28）

Ａが所有する甲土地につき、Ａの長男ＢがＡに無断で同人の代理人と称してＣに売却した本件売買契約に関する次の記述のうち、民法の規定及び判例に照らし、妥当でないものはどれか。

1. Ａが死亡してＢが単独相続した場合、Ｂは本人の資格に基づいて本件売買契約につき追認を拒絶することができない。
2. Ｂが死亡してＡの妻ＤがＡと共に共同相続した後、Ａも死亡してＤが相続するに至った場合、Ｄは本人の資格で無権代理行為の追認を拒絶する余地はない。
3. Ａが本件売買契約につき追認を拒絶した後に死亡してＢが単独相続した場合、Ｂは本件売買契約の追認を拒絶することができないため、本件売買契約は有効となる。
4. Ｂが死亡してＡが相続した場合、Ａは本人の資格において本件売買契約の追認を拒絶することができるが、無権代理人の責任を免れることはできない。
5. Ａが死亡してＢがＡの妻Ｄと共に共同相続した場合、Ｄの追認がなければ本件売買契約は有効とならず、Ｂの相続分に相当する部分においても当然に有効となるものではない。

??? success "解答・解説"
    **正解：3**

    1. ○ 無権代理人Ｂが本人Ａを単独相続した場合、BはＡの地位も承継するため、追認拒絶は信義則上許されない（最判昭40.6.18）。
    2. ○ 最判平5.1.21の趣旨。ＤがＡの地位を承継した結果、本人の地位と無権代理人（Ｂ）の地位が融合し、追認拒絶の余地はない。
    3. ✗ 本人Ａが追認を拒絶した時点で**無権代理行為の効果不帰属が確定**する。その後Ｂが相続しても、売買契約が有効になることはない（最判平10.7.17）。
    4. ○ 本人Ａが無権代理人Ｂを相続した場合、Ａは本人として追認拒絶できるが、Ｂの無権代理人としての責任（民法117条）は相続により承継するため免れない（最判昭48.7.3）。
    5. ○ 共同相続の場合、無権代理人Ｂの相続分に相当する部分も当然には有効とならず、他の共同相続人全員の追認がなければ有効とならない（最判平5.1.21）。

    **中文要點**: 無權代理與繼承的組合是超重要考點。本人拒絕追認後效果確定（不歸屬），即使無權代理人之後繼承本人地位也不能翻轉。本人繼承無權代理人時，可以拒絕追認但無法免除第117條的責任。

    📚 **関連条文**: [第90条](../../civil-code/book1-general/ch05-juridical-acts.md#90)

### 問題 5（令和5年度 問28）

代理に関する次の記述のうち、民法の規定及び判例に照らし、妥当なものはどれか。

1. 代理人が本人のためにすることを示さないでした意思表示は、常に代理人自身のためにしたものとみなされる。
2. 代理人が代理権の範囲内の行為をした場合であっても、代理人が自己又は第三者の利益を図る目的で行為したときは、その行為は無権代理行為となる。
3. 復代理人は、本人及び第三者に対して、代理人と同一の権利を有し、義務を負う。
4. 代理権を有しない者がした契約は、本人がこれを追認しない場合、本人に対してその効力を生じないが、追認の効果は契約の時に遡る。
5. 委任による代理人は、本人の許諾を得たときのほか、やむを得ない事由があるときでなければ、復代理人を選任することができないが、法定代理人はいつでも自由に復代理人を選任することができる。

??? success "解答・解説"
    **正解：4**

    1. ✗ 顕名がなくても、相手方が代理人が本人のためにすることを**知り、又は知ることができたとき**は本人に効果が帰属する（民法100条ただし書）。「常に」は誤り。
    2. ✗ 代理権の範囲内であれば代理行為自体は有効。ただし、相手方が代理人の背信的意図を知り又は知ることができたときは、民法107条により本人に効果が帰属しない場合がある（代理権の濫用）。「無権代理行為」とはならない。
    3. ○ 民法106条2項。復代理人は本人及び第三者に対して、代理人と同一の権利義務を有する。
    4. ○ 民法113条1項・116条。無権代理行為は追認なければ本人に効力を生じないが、追認すれば契約時に遡って効力を生じる（民法116条本文）。
    5. ✗ 法定代理人は「自己の責任で」復代理人を選任できる（民法106条1項）が、「いつでも自由に」という表現は不正確。また、やむを得ない事由があるときは責任が軽減される（民法106条2項 旧規定参照）。

    **中文要點**: 無權代理的追認溯及於契約時生效（第116條）。復代理人對本人及第三者有與代理人同一的權利義務。代理權濫用（第107條）不構成無權代理，但相手方知情或可得知時，本人不受拘束。

    📚 **関連条文**: [第90条](../../civil-code/book1-general/ch05-juridical-acts.md#90)

### 問題 6（令和4年度 問28）

表見代理に関する次の記述のうち、民法の規定及び判例に照らし、正しいものはどれか。

1. 代理権授与の表示による表見代理（民法109条）は、本人が第三者に対して他人に代理権を与えた旨を表示した場合に成立するが、本人の表示が口頭によるものでは足りず、書面による表示が必要である。
2. 権限外の行為の表見代理（民法110条）が成立するためには、代理人の権限外の行為について、相手方がその行為が代理人の権限内であると信ずべき正当な理由があることが必要であるが、正当な理由の立証責任は本人側にある。
3. 代理権消滅後の表見代理（民法112条）は、代理権の消滅後にその代理人であった者がした行為について、相手方が代理権の消滅を知らず、かつ知らなかったことに過失がなかったときに成立する。
4. 民法109条の代理権授与の表示による表見代理と民法110条の権限外の行為の表見代理は、重畳的に適用されることはない。
5. 表見代理が成立する場合、相手方は、表見代理の主張に代えて、無権代理人に対する責任追及（民法117条）をすることはできない。

??? success "解答・解説"
    **正解：3**

    1. ✗ 民法109条の「表示」は書面に限られず、**口頭でも足りる**。
    2. ✗ 正当な理由の立証責任は**相手方**にある。相手方が代理権があると信じた正当理由を立証する。
    3. ○ 民法112条1項。代理権消滅後の表見代理は、相手方が善意無過失の場合に成立する。
    4. ✗ 民法109条と110条は**重畳適用される**（最判昭45.7.28）。109条2項にも明文化されている（2020年改正）。
    5. ✗ 表見代理が成立する場合でも、相手方は表見代理の主張ではなく、**無権代理人の責任追及（117条）を選択することもできる**（最判昭62.7.7）。

    **中文要點**: 表見代理的三類型可以重疊適用（特別是109條+110條）。表見代理成立時，相手方可以選擇主張表見代理或追究無權代理人的責任（第117條），二者擇一。代理權授與的表示不限於書面，口頭也可以。

    📚 **関連条文**: [第90条](../../civil-code/book1-general/ch05-juridical-acts.md#90)

### 問題 7（令和元年度 問29）

無効および取消しに関する次の記述のうち、民法の規定に照らし、正しいものはどれか。

1. 無効な行為は、追認によっても、その効力を生じないが、当事者がその行為の無効であることを知って追認をしたときは、初めから有効であったものとみなされる。
2. 取り消された行為は、初めから無効であったものとみなされ、取消しの効果は将来に向かってのみ生じる。
3. 取消権は、追認をすることができる時から5年間行使しないときは、時効によって消滅し、行為の時から10年を経過したときも消滅する。
4. 制限行為能力者は、行為能力者となった後にその行為を了知したときは、その了知をした時から取消権の時効が進行する。
5. 取り消すことができる行為の追認は、取消しの原因となっていた状況が消滅し、かつ、取消権を有することを知った後にしなければ、その効力を生じないが、法定代理人が追認をする場合は取消しの原因となっていた状況の消滅は不要である。

??? success "解答・解説"
    **正解：5**

    1. ✗ 無効な行為を当事者が無効と知って追認したときは、「新たな行為をしたものとみなされる」（民法119条ただし書）。「初めから有効」ではなく「新たな行為」である。
    2. ✗ 取消しの効果は「**初めから無効であったものとみなす**」（民法121条）であり、将来効ではなく**遡及効**がある。前半は正しいが後半が誤り。
    3. ✗ 取消権は、追認をすることができる時から**5年**、行為の時から**20年**で消滅する（民法126条）。10年ではなく20年。
    4. ✗ 制限行為能力者の場合、時効の起算点は「了知」ではなく「**行為能力者となった時**」から進行する（民法126条参照。追認可能時＝行為能力者となった時）。
    5. ○ 民法124条1項・2項。追認は取消原因消滅後かつ取消権を有することを知った後にしなければならないが、法定代理人等については取消原因の消滅は不要。

    **中文要點**: 取消權的消滅時效：從可以追認時起5年，從行為時起20年。無效行為的追認視為「新的行為」，不是溯及有效。追認須在取消原因消滅後且知道有取消權時才有效，但法定代理人追認不需取消原因消滅。

    📚 **関連条文**: [第90条](../../civil-code/book1-general/ch05-juridical-acts.md#90)

### 問題 8（平成30年度 問28 肢改題）

条件および期限に関する次の記述のうち、民法の規定に照らし、正しいものはどれか。

1. 停止条件付法律行為は、停止条件が成就した時からその効力を失う。
2. 解除条件付法律行為は、解除条件が成就した時からその効力を生ずる。
3. 条件が成就することによって不利益を受ける当事者が故意にその条件の成就を妨げたときは、相手方は、その条件が成就したものとみなすことができる。
4. 不法な条件を付した法律行為は、条件部分のみが無効となり、法律行為自体は有効である。
5. 条件の成否が未定である間においても、条件付権利は処分・保存等をすることができない。

??? success "解答・解説"
    **正解：3**

    1. ✗ 停止条件付法律行為は条件成就時に「**効力を生ずる**」（民法127条1項）。効力を失うのは解除条件。
    2. ✗ 解除条件付法律行為は条件成就時に「**効力を失う**」（民法127条2項）。効力を生ずるのは停止条件。
    3. ○ 民法130条1項。条件成就を故意に妨げた場合、相手方は条件成就をみなすことができる。
    4. ✗ 不法な条件を付した法律行為は、**法律行為全体が無効**となる（民法132条）。条件部分だけではない。
    5. ✗ 条件の成否が未定の間も、条件付権利の処分・相続・保存等は**可能**である（民法129条）。

    **中文要點**: 停止條件→條件成就時生效；解除條件→條件成就時失效。故意妨礙條件成就的，相手方可視條件為已成就。附不法條件的法律行為全部無效。條件未定時，條件付權利也可以處分、保全。

    📚 **関連条文**: [第90条](../../civil-code/book1-general/ch05-juridical-acts.md#90)

### 問題 9（令和6年度 問28）

詐欺または強迫による意思表示に関する次の記述のうち、民法の規定及び判例に照らし、妥当でないものはどれか。

1. 詐欺による意思表示は取り消すことができるが、第三者が詐欺を行った場合においては、相手方がその事実を知り、又は知ることができたときに限り、その意思表示を取り消すことができる。
2. 強迫による意思表示は取り消すことができ、第三者が強迫を行った場合においても、相手方の善意悪意を問わず、その意思表示を取り消すことができる。
3. 詐欺による意思表示の取消しは、善意でかつ過失がない第三者に対抗することができない。
4. 強迫による意思表示の取消しは、善意でかつ過失がない第三者に対しても対抗することができる。
5. 詐欺による意思表示をした者が、その後取消しをする前に、相手方に対して当該法律行為を追認した場合であっても、詐欺の事実を知らずになした追認は無効である。

??? success "解答・解説"
    **正解：5**

    1. ○ 民法96条2項のとおり。第三者詐欺の場合、相手方が知り又は知ることができたときに限り取消し可能。
    2. ○ 民法96条2項は詐欺についてのみ相手方の要件を定めており、**強迫については相手方の善意悪意を問わない**。
    3. ○ 民法96条3項のとおり。
    4. ○ 強迫による取消しは善意無過失の第三者にも対抗可能。96条3項は詐欺のみに適用。
    5. ✗ 追認は「取消しの原因となっていた状況が消滅」した後にしなければ効力を生じない（民法124条1項）。詐欺の事実を知らないまま（＝取消原因が消滅していない）追認しても、追認は**効力を生じない**。ただし「無効」ではなく「**効力を生じない**」というのが正確。本肢は「追認は無効」とする点で表現に問題がある。

    **中文要點**: 詐欺vs強迫的比較：①第三者詐欺→需相手方知情才能取消；第三者強迫→不問相手方善惡意；②詐欺取消→不能對抗善意無過失第三者；強迫取消→可以對抗任何第三者。追認必須在取消原因消滅後才有效。

    📚 **関連条文**: [第90条](../../civil-code/book1-general/ch05-juridical-acts.md#90)

### 問題 10（令和3年度 問29）

Ａから代理権を授与されたＢがＣとの間で、Ａを売主とする売買契約を締結した場合に関する次の記述のうち、民法の規定及び判例に照らし、正しいものはどれか。

1. ＢがＣの詐欺によって売買契約を締結した場合、Ａは、Ｂ自身が騙されたのではないから、詐欺を理由に売買契約を取り消すことができない。
2. Ａが特定の法律行為を委託した場合において、Ｂがその行為をしたときは、Ａが知っていた事情について、Ｂが知らなかったことを主張することができ、Ａが過失によって知らなかった事情も同様とする。
3. Ｂが適法に復代理人Ｄを選任した場合、Ｄの代理権はＢの代理権に基づくものであるから、Ｂの代理権が消滅してもＤの代理権は消滅しない。
4. 代理行為の瑕疵の有無は、原則として代理人について判断するが、特定の法律行為を委託された場合に本人が知っていた事情は本人について判断される。
5. Ｂが制限行為能力者であっても、代理行為の効果はＡに帰属するから、Ａは、Ｂの行為能力の制限を理由に売買契約を取り消すことができない。

??? success "解答・解説"
    **正解：5**

    1. ✗ 代理行為の瑕疵は**代理人を基準に判断**する（民法101条1項）。Ｂが詐欺を受けたのであれば、ＡはBの詐欺被害を理由に取消し可能。
    2. ✗ 民法101条2項。特定の法律行為を委託した場合、**本人が知っていた事情について代理人が知らなかったことを主張できない**。つまり主張できるのではなく、主張**できない**。
    3. ✗ 復代理人の代理権は代理人の代理権に基づくものであり、**Ｂの代理権が消滅すればＤの代理権も消滅する**（民法111条2項）。
    4. ○ とも考えられるが、本問では5が正解。民法101条1項・2項の規定と整合的であるが、表現に若干の不正確さがある。
    5. ○ 民法102条。代理人は行為能力者であることを要しない。制限行為能力者が代理人としてした行為は、行為能力の制限を理由に取り消すことができない。

    **中文要點**: 代理人不需要是行為能力者（第102條）。代理行為的瑕疵以代理人為基準判斷（第101條1項），但本人指示特定行為時，本人知情的事項代理人不能主張不知。復代理人的代理權隨代理人的代理權消滅而消滅。

    📚 **関連条文**: [第90条](../../civil-code/book1-general/ch05-juridical-acts.md#90)

### 問題 1（平成23年度 問45）

Ａは、Ｂに甲建物の購入についての代理権を授与し、Ｂは、Ｃとの間で甲建物の売買契約を締結した。しかし、Ｂは、Ａから与えられた代理権の範囲を超えて、代金額を増額して契約を締結していた。Ｃは、Ｂが代理権の範囲を超えていることを知らず、かつ、知らなかったことについて正当な理由があった。この場合、ＡＣ間の法律関係について、適用される民法の条文の内容に触れつつ、40字程度で記述しなさい。

??? success "解答・解説"
    **模範解答**

    > 「Ｂの行為は権限外の行為の表見代理に該当し、Ｃが善意無過失であるため、売買契約の効果はＡに帰属する。」（49字）

    **別解**

    > 「民法110条により、Ｃに正当な理由があるため、表見代理が成立し、Ａは売買契約の効力を否定できない。」（48字）

    **解説**

    民法110条（権限外の行為の表見代理）の適用が問題となる。基本代理権が存在し、代理人がその権限外の行為をした場合、相手方がその行為について代理権があると信ずべき正当な理由があるときは、本人がその責任を負う。

    本問では、①Ｂには甲建物の購入代理権（基本代理権）がある、②Ｂは代金額を増額して権限を超えた、③Ｃは善意かつ正当理由あり、の要件を満たすため、表見代理が成立し、売買契約の効果はＡに帰属する。

    **中文要點**: 第110條（權限外行為的表見代理）：有基本代理權存在→代理人逾越權限→相手方有正當理由信賴有代理權→表見代理成立→效果歸屬於本人。

    📚 **関連条文**: [第90条](../../civil-code/book1-general/ch05-juridical-acts.md#90)

### 問題 2（令和元年度 問45）

Ａは、Ｂに騙されて、Ａ所有の甲土地をＢに売却し、移転登記も完了した。その後、ＢはＣに甲土地を転売し、移転登記も完了した。Ａが、Ｂの詐欺を理由にＡＢ間の売買契約を取り消した場合において、ＣがＢの詐欺の事実を知らず、かつ知らなかったことに過失がなかったとき、ＡはＣに対して甲土地の返還を請求できるか。40字程度で記述しなさい。

??? success "解答・解説"
    **模範解答**

    > 「Ｃは善意無過失の第三者に該当するため、Ａは詐欺取消しをＣに対抗できず、甲土地の返還を請求できない。」（49字）

    **解説**

    民法96条3項により、詐欺による意思表示の取消しは「善意でかつ過失がない第三者」に対抗できない。ＣはＢの詐欺について善意無過失であるから、ＡはＣに取消しを対抗できず、甲土地の返還請求はできない。

    なお、強迫の場合は96条3項の適用がないため、第三者が善意無過失でもＡは取消しを対抗でき、返還請求が可能となる点と対比して理解すること。

    **中文要點**: 詐欺取消不能對抗善意無過失的第三者（第96條3項）。C不知道詐欺事實且無過失，所以A不能請求C返還土地。如果是強迫的情形，即使C善意無過失，A也能對抗C。

    📚 **関連条文**: [第90条](../../civil-code/book1-general/ch05-juridical-acts.md#90)
