# 第1章 通則 — 練習問題

!!! info "本章出題傾向"
    通則（第1条〜第2条）に関する問題は、過去10年間で約2〜3回出題。
    重要テーマ：信義誠実の原則（信義則）、権利濫用の禁止

---

## 択一式

### 問題 1（平成30年度 問28）

民法第1条に規定する基本原則に関する次の記述のうち、民法の規定及び判例に照らし、妥当でないものはどれか。

1. 私権は、公共の福祉に適合しなければならないとされており、個人の権利であっても社会全体の利益と調和する範囲内で行使されなければならない。
2. 権利の行使及び義務の履行は、信義に従い誠実に行わなければならないとする信義則は、私法上の法律関係にのみ適用され、公法上の法律関係には適用されない。
3. 権利の濫用は許されず、形式的には権利の行使であっても、その権利の社会性に反し、権利の行使として認めることが正義に反する場合には、権利の濫用に該当する。
4. 信義則は、契約関係にある当事者間においては、明文の規定がなくても、付随義務（通知義務・保護義務等）を生じさせる根拠となりうる。
5. 権利の濫用にあたるとされた場合、当該権利行使は法律上の効力を有しないとされ、損害賠償の根拠となりうる。

??? success "解答・解説"
    **正解：2**

    1. ○ 民法第1条第1項のとおり。私権は公共の福祉に適合しなければならない。
    2. ✗ 信義則は私法関係のみならず、**公法上の法律関係にも適用される**（最判昭56.1.27 租税法律関係への信義則の適用を認めた判例）。
    3. ○ 民法第1条第3項。宇奈月温泉事件（大判昭10.10.5）が代表例。
    4. ○ 信義則から、契約の明文にない付随義務が導かれることは判例・学説上認められている。
    5. ○ 権利濫用にあたる行為は効力を否定され、不法行為として損害賠償の対象にもなりうる。

    **中文要點**: 信義則（誠實信用原則）不僅適用於私法關係，判例上也承認其適用於公法關係（如租稅法律關係）。這是考試中常見的陷阱。

    📚 **関連条文**: [第1条](../../civil-code/book1-general/ch01-general-rules.md#1)

### 問題 2（令和3年度 問27）

権利濫用に関する次の記述のうち、判例に照らし、正しいものはどれか。

1. 建物の賃貸人が、正当事由がないにもかかわらず、賃借人に対して立退料の提供なしに建物の明渡しを求めることは、常に権利の濫用に該当する。
2. 土地の所有者が、隣地との境界付近に、専ら隣人の日照を妨害する目的で板塀を建てた場合、権利の濫用に該当しうる。
3. 債権者が、消滅時効の完成した債権を自働債権として相殺の意思表示をすることは、権利の濫用にあたる。
4. 権利の濫用にあたるかどうかの判断においては、権利者側の主観的事情のみが考慮され、相手方の事情は考慮されない。
5. 信義則と権利濫用の禁止は、いずれも民法の基本原則であるが、裁判所は当事者の主張がなくても職権でこれを適用することは許されない。

??? success "解答・解説"
    **正解：2**

    1. ✗ 正当事由の有無は立退料の提供も含む諸事情を総合的に考慮して判断される。「常に」権利濫用とはならない。
    2. ○ 信玄公旗掛松事件等の判例の趣旨に照らし、専ら他人を害する目的でなされた権利行使は権利濫用に該当しうる（大判大8.3.3 板塀事件）。
    3. ✗ 消滅時効の完成した債権を自働債権とする相殺は、時効完成前に相殺適状にあった場合には認められる（民法508条）。権利濫用の問題ではない。
    4. ✗ 権利濫用の判断は、権利者側の事情と相手方側の事情の双方を比較衡量して行う。
    5. ✗ 信義則・権利濫用は、裁判所が職権で適用しうるとされている。

    **中文要點**: 權利濫用的判斷需要綜合考量雙方的事情。專門以損害他人為目的的權利行使（如故意遮擋鄰居日照的圍牆），是權利濫用的典型案例。

    📚 **関連条文**: [第1条](../../civil-code/book1-general/ch01-general-rules.md#1)
