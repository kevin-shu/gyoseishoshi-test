# 第4〜6章 用益物権（地上権・永小作権・地役権） — 練習問題

!!! info "本章出題傾向"
    用益物権（地上権・永小作権・地役権）に関する問題は、過去10年間で約2〜3回出題。
    重要テーマ：地上権の特徴（物権性・地代不要・承諾不要の譲渡転貸）、地役権（時効取得の要件・承役地の譲渡）、区分地上権（第269条の2）

---

## 択一式

### 問題 1（令和元年度 問30）

A所有の甲土地とB所有の乙土地が隣接し、甲土地の上にはC所有の丙建物が存在している。この場合における次のア〜オの記述のうち、民法の規定および判例に照らし、妥当なものの組合せはどれか。

ア　Bが、甲土地に乙土地からの排水のための地役権をA・B間で設定し登記していた場合において、CがAに無断で甲土地に丙建物を築造してその建物の一部が乙土地からの排水の円滑な流れを阻害するときは、Bは、Cに対して地役権に基づき丙建物全部の収去および甲土地の明渡しを求めることができる。

イ　A・B間で、乙土地の眺望を確保するため、甲土地にいかなる工作物も築造しないことを内容とする地役権を設定し登記していた場合において、Cが賃借権に基づいて甲土地に丙建物を築造したときは、Bは地役権に基づき建物の収去を求めることができる。

ウ　甲土地が乙土地を通らなければ公道に至ることができない、いわゆる袋地である場合において、Cが、Aとの地上権設定行為に基づいて甲土地に丙建物を建築し乙土地を通行しようとするときは、Cは、甲土地の所有者でないため、Bとの間で乙土地の通行利用のため賃貸借契約を結ぶ必要がある。

エ　Aは、自己の債務の担保として甲土地に抵当権を設定したが、それ以前に賃借権に基づいて甲土地に丙建物を築造していたCからAが当該抵当権の設定後に丙建物を買い受けた場合において、抵当権が実行されたときは、丙建物のために、地上権が甲土地の上に当然に発生する。

オ　Cが、地上権設定行為に基づいて甲土地上に丙建物を築造していたところ、期間の満了により地上権が消滅した場合において、Aが時価で丙建物を買い取る旨を申し出たときは、Cは、正当な事由がない限りこれを拒むことができない。

1. ア・ウ
2. ア・オ
3. イ・エ
4. イ・オ
5. ウ・エ

??? success "解答・解説"
    **正解：4（イ・オ）**

    ア　✗ 地役権侵害に対する妨害排除請求は認められるが、妨害排除の範囲は**地役権の行使を妨害する部分の除去**に限られる。丙建物全部の収去・甲土地全体の明渡しを求めることはできない。

    イ　○ 地役権は物権であり、不作為地役権（一定の工作物を建てないこと）の場合、登記があれば第三者（賃借人C）にも対抗できる。Bは地役権に基づきCに建物収去を求めることができる。

    ウ　✗ 囲繞地通行権（第210条）は、囲繞地の**所有者または使用権者**が公道に出るために隣地を通行する権利であり、地上権者Cも行使できる（最判昭36.3.24）。賃貸借契約の締結は不要。

    エ　✗ 法定地上権（第388条）が成立するためには、抵当権設定時に土地と建物が同一人に帰属していること等が必要。本問では抵当権設定後にAがCから建物を取得しており、設定時の同一人帰属の要件を欠く可能性が高い。

    オ　○ 第269条第2項により、地上権消滅の際に**土地所有者が時価で買い取る旨の申出**をした場合は、地上権者はこれを**拒絶できない**。

    **中文要點**: 地役権は物権 → 登記後は第三者（賃借人含む）に対抗可能。袋地通行権（第210条）は所有者だけでなく地上権者等の使用権者も行使可能。地上権消滅時の建物買取申出（第269条第2項）→ 地上権者は拒絶不可。

    📚 **関連条文**: [第265条〜第283条](../../civil-code/book2-property/ch04-land-rights.md#265), [第269条](../../civil-code/book2-property/ch04-land-rights.md#269)

### 問題 2（平成27年度 問30）

地上権に関する次の記述のうち、民法の規定および判例に照らし、妥当なものはどれか。

1. 地上権者は、その権利の存続期間内であれば、土地所有者の承諾なく、地上権を第三者に譲渡することも、その土地を第三者に賃貸することもできる。
2. 存続期間を定めなかった地上権については、その存続期間を当事者間で協議によっても定めることができない場合、裁判所が定める。
3. 地上権者は、土地を使用する権利を有するため、土地所有者に対して地代を支払う義務を当然に負う。
4. 地上権は物権であるため、その設定・移転については登記が対抗要件となるが、地上権の設定登記がなければ第三者に対抗することができないことは、賃借権と異なる点である。
5. 地上権が消滅した場合に、地上権者が土地の原状回復のために建物を収去するときは、土地所有者がその建物を時価で買い取ると申し出ても、地上権者はこれを拒絶することができる。

??? success "解答・解説"
    **正解：1**

    1. ○ 地上権は物権であるため、賃借権と異なり、**土地所有者の承諾なく第三者に譲渡・転貸**することができる（第612条の賃借権の譲渡・転貸制限は地上権には適用されない）。
    2. ✗ 第268条第1項により、存続期間を定めなかった地上権については、当事者の請求により裁判所が定めることができるが、**当事者間で協議によって定めることも可能**。
    3. ✗ 地上権は地代の支払いを要件としない。**地代は特約がある場合にのみ発生**する（第266条第1項）。地代のない地上権も有効に成立する。
    4. ✗ 賃借権も**登記**によって第三者対抗力を取得できる（第605条）ほか、建物の賃借権は引渡しによっても対抗力を得ることがある（借地借家法第31条等）。「地上権のみが登記で対抗できる」わけではない。
    5. ✗ 第269条第2項により、土地所有者が時価で買い取る旨を申し出た場合、**地上権者はこれを拒絶することができない**。

    **中文要點**: 地上権（物権）と賃借権（債権）の重要な違い：
    - 地上権：地代は特約がなければ不要、土地所有者の承諾なしに譲渡・転貸OK
    - 賃借権：承諾なく譲渡・転貸すると解除事由になる（第612条）

    📚 **関連条文**: [第265条〜第270条](../../civil-code/book2-property/ch04-land-rights.md#265)
