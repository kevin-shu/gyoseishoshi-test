# 第9章 質権 — 練習問題

!!! info "本章出題傾向"
    質権に関する問題は、過去10年間で約2〜3回出題。
    重要テーマ：要物契約性（引渡しが成立要件）・占有改定の禁止（第345条）・流質契約の禁止（第349条）・転質・権利質

---

## 択一式

### 問題 1（令和元年度 問31）

質権に関する次の記述のうち、民法の規定および判例に照らし、妥当でないものはどれか。

1. 動産質権者は、継続して質物を占有しなければ、その質権をもって第三者に対抗することができず、また、質物の占有を第三者によって奪われたときは、占有回収の訴えによってのみ、その質物を回復することができる。
2. 不動産質権は、目的不動産を債権者に引き渡すことによってその効力を生ずるが、不動産質権者は、質権設定登記をしなければ、その質権をもって第三者に対抗することができない。
3. 債務者が他人の所有に属する動産につき質権を設定した場合であっても、債権者は、その動産が債務者の所有物であることについて過失なく信じたときは、質権を即時取得することができる。
4. 不動産質権者は、設定者の承諾を得ることを要件として、目的不動産の用法に従ってその使用収益をすることができる。
5. 質権は、債権などの財産権の上にこれを設定することができる。

??? success "解答・解説"
    **正解：4**

    1. ○ 第352条（動産質権の対抗要件）により、動産質権者は継続して質物を占有しなければ第三者に対抗できない。また占有を奪われた場合は占有回収の訴え（第200条）によってのみ回復できる（第353条）。
    2. ○ 不動産質権は目的物の引渡しによって効力が生じる（要物契約）が、第177条（不動産物権変動の対抗要件）により、第三者への対抗には登記が必要。
    3. ○ 第192条（即時取得）は動産質権にも適用される（第343条）。他人の所有物について、善意無過失で質権を設定した場合、質権を即時取得できる。
    4. ✗ 第356条により、不動産質権者は、**設定者の承諾を要せず**、目的不動産を使用収益することができる。「承諾を得ることを要件として」は誤り。
    5. ○ 第362条第1項により、質権は「財産権」を目的とすることができる（権利質）。債権・株式・知的財産権等が対象となりうる。

    **中文要點**: 不動産質権者は設定者の承諾**なしに**使用収益できる（第356条）。これは抵当権者（占有なし・使用収益なし）との大きな違い。
    質権の成立要件（第344条）：目的物の引渡しが必要（要物契約）。
    占有改定による引渡しは不可（第345条）。

    📚 **関連条文**: [第342条〜第368条](../../civil-code/book2-property/ch06-pledge.md#342)

### 問題 2（平成26年度 問29）

質権に関する次のア〜オの記述のうち、民法の規定および判例に照らし、妥当なものの組合せはどれか。

ア　質権設定契約は要物契約であるため、債権者が質権設定者から目的物の引渡しを受けることによって効力を生ずるが、目的物の引渡しについては、現実の引渡しのほかに、簡易の引渡しおよび指図による占有移転の方法によることも認められる。

イ　動産質権者は、質物から生ずる果実を収取し、被担保債権の弁済に充当することができ、元本よりも先に利息に充当するものとされている。

ウ　流質契約は、質権設定契約に際しては禁止されているが、弁済期前には認められないものの、弁済期後における流質契約は有効である。

エ　動産質権者は、その権利の存続期間中、自己の責任において、質物について、転質をすることができるが、この場合において、転質をしたことによって生じた損失については、不可抗力によるものであっても、転質権者が責任を負う。

オ　質権者は、質権設定者の承諾がない場合、被担保債権の弁済として質物の所有権を取得したり、質物を任意に処分したりすることはできないが、法律の定める方法によって質物を換価し弁済に充てることはできる。

1. ア・イ
2. ア・エ
3. イ・ウ
4. ウ・オ
5. エ・オ

??? success "解答・解説"
    **正解：2（ア・エ）**

    ア　○ 第344条（質権の設定）は要物契約であり、引渡しが成立要件。引渡し方法として現実の引渡し・簡易の引渡し（第182条第2項）・指図による占有移転（第184条）は認められる。ただし**占有改定は不可**（第345条）。

    イ　✗ 第346条により、質権者は果実を収取して被担保債権の弁済に充当できるが、通常の弁済充当の規定（第489条等）に従う。「元本より先に利息に充当する」という特別ルールは規定されていない。

    ウ　✗ 第349条により、流質契約は**弁済期前の設定時において禁止**される（設定者を保護するため）。弁済期後の流質契約についても通説・判例は無効とする立場が有力であり、「弁済期後は有効」という断言は問題がある。

    エ　○ 第348条により、質権者は自己の責任において転質をすることができ、転質をしたことにより生じた損失については、**不可抗力によるものであっても**転質をした質権者が責任を負う（責任転質）。

    オ　✗ 流質（弁済として質物の所有権を取得すること）は第349条で設定者の承諾の有無にかかわらず**禁止**されている（設定時の合意があっても無効）。「設定者の承諾がない場合はできない」という記述は流質禁止の趣旨と整合しない。

    **中文要點**: 転質（第348条）：質権者が自己の責任で質物を再度質に入れること。不可抗力による損失も転質をした質権者が責任負担 → これが「責任転質」の特徴。
    流質（第349条）：弁済期前に「弁済できなければ質物の所有権をもらう」という特約 → **禁止**（設定者保護のため）。

    📚 **関連条文**: [第342条〜第368条](../../civil-code/book2-property/ch06-pledge.md#342)
