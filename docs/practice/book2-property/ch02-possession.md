# 第2章 占有権 — 練習問題

!!! info "本章出題傾向"
    占有権に関する問題は、過去10年間で毎年1〜2問出題される頻出分野。
    重要テーマ：占有の取得・移転（特に占有改定）、即時取得の成立要件、占有の訴え（3種類）、善意・悪意占有者の効果

---

## 択一式

### 問題 1（令和4年度 問28）

占有権に関する次の記述のうち、民法の規定および判例に照らし、妥当でないものはどれか。

1. A所有の動産甲をAから委ねられ占有するBが、甲を自己の物と称してCに売却した場合、甲に対するCの即時取得について、占有開始におけるCの平穏、公然、善意および無過失は推定される。
2. A所有の乙土地をBが20年以上にわたって占有し、所有権の取得時効の成否が問われる場合、Aが、Bによる乙の占有が他主占有権原に基づくことを証明しない限り、Bについての他主占有事情が証明されても、Bの所有の意思が認められる。
3. A所有の丙土地を無権利者であるBがCに売却し、Cが所有権を取得したと信じて丙の占有を開始した場合、Aから本権の訴えがないときは、Cは、丙を耕作することによって得た収穫物を取得することができる。
4. A所有の動産丁をBに寄託しこれに基づいてBが占有していたところ、丁をCに盗取された場合、Bは、占有回収の訴えにより、Cに対して丁の返還を請求することができる。
5. A所有の動産戊をBに寄託しこれをBに引き渡した後、Aは戊をCに譲渡した場合、Aが、Bに対して以後Cの所有物として戊を占有するよう指示し、Cが、これを承諾したときは、戊についてAからCへの引渡しが認められる。

??? success "解答・解説"
    **正解：2**

    1. ○ 第186条第1項により、占有者の平穏・公然・善意・無過失は推定される。即時取得（第192条）の場面でも推定が働く。
    2. ✗ 占有は所有の意思をもってするものと**推定**される（第186条第1項）。他主占有事情（例：賃借人であることを認識していた等）が証明された場合、この推定は**覆される**。よって「他主占有事情が証明されても所有の意思が認められる」は誤り。
    3. ○ 善意占有者は、占有物から生ずる果実（天然果実・法定果実）を取得できる（第189条第1項）。本権の訴えが提起されていない間は善意占有者として果実収取権が認められる。
    4. ○ 占有代理人（B）も占有回収の訴え（第200条）を提起できる。占有を奪った者（C）に対して返還請求が可能。
    5. ○ 「指図による占有移転」（第184条）の説明である。A（委託者）がBに対し「以後CのためにCを占有者として占有せよ」と指示し、Cが承諾したとき引渡しが成立する。

    **中文要點**: 占有は「所有意思あり」と推定（第186条）。他主占有事情が証明されれば推定は覆る → 取得時効の「自主占有」要件を失う。「他主占有事情の証明」≠「他主占有権原の証明」：混同注意。

    📚 **関連条文**: [第186条〜第192条](../../civil-code/book2-property/ch02-possession.md#186), [第200条](../../civil-code/book2-property/ch02-possession.md#200)

### 問題 2（令和2年度 問28）

占有改定等に関する次のア〜オの記述のうち、民法の規定および判例に照らし、妥当でないものの組合せはどれか。

ア　即時取得が成立するためには占有の取得が必要であるが、この占有の取得には、外観上従来の占有事実の状態に変更を来たさない、占有改定による占有の取得は含まれない。

イ　留置権が成立するためには他人の物を占有することが必要であるが、この占有には、債務者を占有代理人とした占有は含まれない。

ウ　先取特権の目的動産が売買契約に基づいて第三取得者に引き渡されると、その後は先取特権を当該動産に対して行使できないこととなるが、この引渡しには、現実の移転を伴わない占有改定による引渡しは含まれない。

エ　質権が成立するためには目的物の引渡しが必要であるが、この引渡しには、設定者を以後、質権者の代理人として占有させる、占有改定による引渡しは含まれない。

オ　動産の譲渡担保権を第三者に対抗するためには目的物の引渡しが必要であるが、この引渡しには、公示性の乏しい占有改定による引渡しは含まれない。

1. ア・イ
2. ア・ウ
3. イ・エ
4. ウ・オ
5. エ・オ

??? success "解答・解説"
    **正解：4（ウ・オ）**

    ア　○ 判例（最判昭35.2.11）により、即時取得（第192条）が成立するための「占有の取得」には、**占有改定は含まれない**。公示性がない占有改定では即時取得が成立しない。

    イ　○ 判例により、債務者を占有代理人とした間接占有は、留置権の「他人の物の占有」には含まれないとされている（留置権は対外的な占有状態の確保が必要）。

    ウ　✗ 先取特権の追及力消滅（第333条）における「引渡し」には、**占有改定を含む**（最判昭54.2.15）。占有改定による引渡しがあれば先取特権の動産への追及力は消滅する。よって「含まれない」は誤り。

    エ　○ 質権設定には目的物の引渡しが必要（第344条）であり、この引渡しには**占有改定は含まれない**（第345条）。設定者を占有代理人として占有させることは明文で禁止されている。

    オ　✗ 判例（最判昭54.2.15）により、動産の譲渡担保における引渡しには**占有改定が含まれる**。現実の引渡しは不要で、占有改定でも第三者対抗要件を備えることができる。よって「含まれない」は誤り。

    **中文要點**:
    - 即時取得（第192条）：占有改定 → **不成立**（公示性なし）
    - 質権設定（第344条）：占有改定 → **不成立**（第345条で明文禁止）
    - 先取特権の追及力消滅（第333条）：占有改定 → **消滅する**（最判昭54.2.15）
    - 動産譲渡担保の対抗要件：占有改定 → **成立する**（最判昭54.2.15）

    📚 **関連条文**: [第192条](../../civil-code/book2-property/ch02-possession.md#192), [第333条](../../civil-code/book2-property/ch05-security-interests.md#333)
