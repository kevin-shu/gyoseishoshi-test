# 第3章 所有権 — 練習問題

!!! info "本章出題傾向"
    所有権に関する問題は、過去10年間で約3〜4回出題。
    重要テーマ：相隣関係（袋地通行権・境界）、共有（意思決定・分割・持分の帰属）、添付（付合・混和・加工）

---

## 択一式

### 問題 1（平成28年度 問29）

A、B及びCが甲土地を共有し、甲土地上には乙建物が存在している。この場合に関する次のア〜オの記述のうち、民法の規定および判例に照らし、正しいものの組合せはどれか。

ア　DがA、B及びCに無断で甲土地上に丙建物を建てて甲土地を占有使用している場合、Aは、Dに対し、単独で建物の収去および土地の明渡しならびに土地の占拠により生じた損害全額の賠償を求めることができる。

イ　Eが、A、B及びCが共有する乙建物をAの承諾のもとに賃借して居住し、甲土地を占有使用する場合、B及びCは、Eに対し当然には乙建物の明渡しを請求することはできない。

ウ　Fが賃借権に基づいて甲土地上に丙建物を建てた場合において、A、B及びCが甲土地の分割協議を行うとするときは、Fに対して分割協議を行う旨を通知しなければならず、通知をしないときは、A、B及びCの間でなされた分割の合意は、Fに対抗することができない。

エ　Aが乙建物を所有し居住している場合において、Aが、B及びCに対して甲土地の分割請求をしたときは、甲土地をAに単独所有させ、Aが、B及びCに対して持分に相当する価格の賠償を支払う、いわゆる全面的価額賠償の方法によって分割しなければならない。

オ　A、B及びCが乙建物を共有する場合において、Aが死亡して相続人が存在しないときは、Aの甲土地および乙建物の持分は、B及びCに帰属する。

1. ア・イ
2. ア・ウ
3. イ・オ
4. ウ・エ
5. エ・オ

??? success "解答・解説"
    **正解：3（イ・オ）**

    ア　✗ 共有者の一人（A）は、不法占有者に対する損害賠償請求は**自己の持分割合に応じた限度でしか**請求できない（最判昭51.9.7）。建物収去・土地明渡しの請求は単独でできるが、損害賠償「全額」の請求はできない。

    イ　○ 共有者の一人（A）が承諾して賃貸した場合、その承諾を得た賃借人（E）に対して他の共有者（B・C）は当然には明渡し請求できない（最判昭63.5.20）。賃貸借が管理行為として有効か否かの問題となる。

    ウ　✗ 共有物の分割は持分権者間の問題であり、賃借人に分割を通知する義務はない。共有者間の分割合意は、通知がなくても賃借人に対して効力を有する。

    エ　✗ 共有物分割の方法は、現物分割・代金分割（競売）・全面的価格賠償の複数の方法があり、**必ずしも全面的価格賠償による必要はない**。現物分割が不可能または著しく困難な場合等に他の方法が採られる（第258条）。

    オ　○ 共有者が死亡して**相続人がなく**、かつ特別縁故者への財産分与もない場合、その者の共有持分は他の共有者に帰属する（第255条）。

    **中文要點**: 共有物の「保存行為」は各共有人が単独でできる（不法占有者への収去・明渡し請求など）。しかし損害賠償請求は保存行為ではなく**持分比例**で請求するに留まる。

    📚 **関連条文**: [第249条〜第264条](../../civil-code/book2-property/ch03-ownership.md#249), [第255条](../../civil-code/book2-property/ch03-ownership.md#255)

### 問題 2（令和5年度 問28）

所有権に関する次の記述のうち、民法の規定および判例に照らし、妥当なものはどれか。

1. 土地の所有権は、その土地の上下に及ぶが、地下または空間の利用が他人によって法律上制限されている場合、当該土地の所有権はその部分には及ばない。
2. Aが所有する土地の竹木の枝が境界線を越えて隣地Bに越境している場合、BはAに対して枝の切除を催告したとしても、Aが相当の期間内に切除しないときであっても、BがAに代わってその枝を切ることはできない。
3. 土地の所有者は、隣接する他の土地が公道に通じていない場合でも、その他の土地の所有者の承諾がなければ、当該他の土地を通行することができない。
4. 付合により生じた合成物の所有権は、主たる動産の所有者に帰属し、従たる動産の所有者は、その合成物の所有権を当然には取得しない。
5. 建物の賃借人が当該建物に有益費を支出した場合、賃貸借が終了するときには、建物の所有者は、賃借人の選択に従い、支出した金額または増価額を償還しなければならない。

??? success "解答・解説"
    **正解：4**

    1. ✗ 第207条により、土地の所有権はその土地の上下に及ぶ。法律上の制限があっても所有権の範囲が縮小するわけではなく、利用を制限されているに過ぎない。
    2. ✗ 令和3年改正後の第233条第3項により、竹木の枝が境界を越えた場合、竹木の所有者に催告したが相当期間内に切除されないとき等は、越境を受けた土地の所有者が自ら枝を切り取ることができるようになった。
    3. ✗ 第210条（公道に至るための他の土地の通行権）により、囲繞地（公道に通じない土地）の所有者は、その土地を囲んでいる他の土地を通行する権利（袋地通行権）を有する。隣地所有者の承諾は不要。
    4. ○ 第243条（動産の付合）により、所有者を異にする数個の動産が付合して一個の物となったときは、合成物の所有権は**主たる動産の所有者**に帰属する。従たる動産の所有者は、償金請求権（第248条）を有するに止まる。
    5. ✗ 第608条第2項により、賃借人が有益費を支出した場合、賃貸借終了時に**賃貸人**の選択に従い、支出金額または増価額を償還する。選択権は賃借人ではなく賃貸人にある。

    **中文要點**: 動産付合（第243条）→ 主たる動産の所有者が合成物を取得。従の所有者には償金請求権（第248条）のみ。「賃借人の選択」ではなく「賃貸人の選択」が正しい（第608条第2項）。

    📚 **関連条文**: [第206条〜第243条](../../civil-code/book2-property/ch03-ownership.md#206)
