# 第2章 審査請求 — 練習問題

!!! info "本章出題傾向"
    審査請求（第9条〜第53条）は行政不服審査法の最重要章。過去10年間でほぼ毎年3〜5問出題。
    重要テーマ：審理員制度（第9条）、審査請求期間（第18条）、執行停止（第25条）、口頭意見陳述（第31条）、行政不服審査会への諮問（第43条）、裁決の種類（第45条・第46条）、不利益変更の禁止（第48条）、裁決の拘束力（第52条）

---

## 択一式

### 問題 1（令和4年度 問14）

行政不服審査法に規定する審理員に関する次の記述のうち、妥当でないものはどれか。

1. 審査庁は、審査庁に所属する職員のうちから審理手続を行う審理員を指名するとともに、その旨を審査請求人及び処分庁等に通知しなければならない。
2. 審査請求に係る処分に関与した者は、審理員に指名することができない。
3. 審査請求人の配偶者、四親等内の親族又は同居の親族は、審理員に指名することができない。
4. 審査庁が内閣府設置法又は国家行政組織法に規定する委員会である場合には、審理員を指名する必要はない。
5. 審査請求が不適法であっても、審理手続を経ないでする却下裁決の場合を含め、審理員を必ず指名しなければならない。

??? success "解答・解説"
    **正解：5**

    1. ○ 第9条第1項本文のとおり。審理員の指名と通知は審査庁の義務。
    2. ○ 第9条第2項第1号のとおり。処分に関与した者は除斥事由に該当し、審理員になれない。公正性確保のための重要な規定。
    3. ○ 第9条第2項第3号のとおり。親族関係にある者も除斥される。
    4. ○ 第9条第1項ただし書第1号のとおり。委員会等の合議制機関が審査庁である場合は、合議制自体に公正性の担保があるため、審理員の指名は不要。
    5. ✗ 第9条第1項ただし書により、第24条の規定により**審理手続を経ないで却下する場合**には、審理員を指名する必要はない。不適法な審査請求を門前払いする場合にまで審理員を指名させる意味はないからである。

    **中文要點**: 審理員制度是平成26年改正的核心。審理員必須是與處分無關的人（7類除斥事由）。例外不需指名審理員的情形：合議制機關為審查機關、條例有特別規定、不合法審查請求的直接駁回（第24條）。

    📚 **関連条文**: [第9条](../../admin-law/appeal/ch02-review-request.md#9)

### 問題 2（令和2年度 問15）

行政不服審査法に規定する審査請求期間に関する次の記述のうち、正しいものはどれか。

1. 処分についての審査請求は、処分があったことを知った日の翌日から起算して6月を経過したときは、することができない。
2. 処分についての審査請求は、処分があった日の翌日から起算して1年を経過したときは、正当な理由がある場合であってもすることができない。
3. 審査請求書を郵便で提出した場合における審査請求期間の計算については、送付に要した日数は算入しない。
4. 不作為についての審査請求は、当該不作為が継続している場合であっても、申請から3月を経過した後は、することができない。
5. 処分についての審査請求期間は、行政事件訴訟法における取消訴訟の出訴期間と同一である。

??? success "解答・解説"
    **正解：3**

    1. ✗ 主観的期間は「**3月**」であり、6月ではない（第18条第1項）。6月は行政事件訴訟法の取消訴訟の出訴期間（行訴法14条第1項）であり、混同に注意。
    2. ✗ 客観的期間は1年であるが（第18条第2項）、「**正当な理由があるとき**」は例外として認められる（同項ただし書）。
    3. ○ 第18条第3項のとおり。審査請求書を郵便又は信書便で提出した場合は、送付に要した日数は算入しない（**発信主義**）。
    4. ✗ 不作為についての審査請求には第18条の期間制限は適用されない。不作為が継続する限り、いつでも審査請求が可能。
    5. ✗ 行政不服審査法の主観的期間は**3月**、行訴法の主観的期間は**6月**であり、同一ではない。客観的期間はいずれも1年で同じ。

    **中文要點**: 審查請求期間：主觀3個月/客觀1年（行訴法是主觀6個月/客觀1年）。郵寄提出時採發信主義（郵寄日數不算入）。不作為之審查請求無期間限制。正當理由例外適用於主觀期間與客觀期間兩者。

    📚 **関連条文**: [第18条](../../admin-law/appeal/ch02-review-request.md#18)

### 問題 3（平成30年度 問15）

行政不服審査法に規定する執行停止に関する次の記述のうち、妥当なものはどれか。

1. 審査請求がされたときは、処分の効力、処分の執行又は手続の続行は、当然に停止する。
2. 処分庁の上級行政庁又は処分庁である審査庁は、審査請求人の申立てによってのみ執行停止をすることができ、職権で執行停止をすることはできない。
3. 処分庁の上級行政庁又は処分庁のいずれでもない審査庁は、審査請求人の申立てにより、処分庁の意見を聴取した上、執行停止をすることができる。
4. 審査請求人の申立てがあった場合において、処分により生ずる重大な損害を避けるために緊急の必要があると認めるときでも、審査庁は執行停止をするかどうかを裁量により判断する。
5. 行政事件訴訟法に基づく執行停止においても、裁判所は、申立てによるほか、職権で執行停止をすることができる。

??? success "解答・解説"
    **正解：3**

    1. ✗ 行政不服審査法第25条第1項により、審査請求は処分の効力等を妨げない（**執行不停止原則**）。
    2. ✗ 第25条第2項により、処分庁の上級行政庁又は処分庁である審査庁は、「審査請求人の申立てにより**又は職権で**」執行停止をすることができる。職権発動も可能である。
    3. ○ 第25条第3項のとおり。上級行政庁でも処分庁でもない審査庁は、申立てにより、処分庁の意見を聴取した上で執行停止ができる。ただし、処分の効力等の停止以外の措置はとれない。
    4. ✗ 第25条第4項により、重大な損害を避けるために緊急の必要があると認めるときは、審査庁は「執行停止をしなければならない」（**義務的執行停止**）。裁量ではなく義務である。ただし、公共の福祉に重大な影響を及ぼすおそれがあるとき等は例外。
    5. ✗ 行政事件訴訟法第25条に基づく執行停止は、「**申立てによってのみ**」可能であり、裁判所は職権で執行停止をすることはできない。行審法との大きな違いである。

    **中文要點**: 執行停止比較：行審法允許職權啟動（上級機關/處分機關），行訴法僅限申立。義務的執行停止（第4項）：有重大損害且緊急必要時，審查機關必須停止執行。非上級亦非處分機關之審查機關則需聽取處分機關意見。

    📚 **関連条文**: [第25条](../../admin-law/appeal/ch02-review-request.md#25)

### 問題 4（令和5年度 問14）

行政不服審査法に規定する審理手続及び行政不服審査会への諮問に関する次の記述のうち、妥当でないものはどれか。

1. 審査請求人又は参加人の申立てがあった場合には、審理員は、原則として、申立人に口頭で意見を述べる機会を与えなければならない。
2. 口頭意見陳述において、申立人は、審理員の許可を得て、審査請求に係る事件に関し、処分庁等に対して質問を発することができる。
3. 審査庁は、審理員意見書の提出を受けたときは、原則として行政不服審査会等に諮問しなければならない。
4. 審査請求人が行政不服審査会への諮問を希望しない旨の申出をした場合には、参加人から反対する旨の申出がされている場合であっても、諮問を要しない。
5. 審査請求が不適法であり却下する場合には、行政不服審査会への諮問を要しない。

??? success "解答・解説"
    **正解：4**

    1. ○ 第31条第1項本文のとおり。申立てがあった場合には、原則として口頭意見陳述の機会を与えなければならない。ただし、申立人の所在等により困難な場合は例外。
    2. ○ 第31条第5項のとおり。平成26年改正で新設された規定。申立人は審理員の許可を得て処分庁等に質問できる。
    3. ○ 第43条第1項本文のとおり。審理員意見書の提出を受けた審査庁は、原則として諮問義務がある。
    4. ✗ 第43条第1項第4号により、審査請求人が諮問を希望しない旨の申出をした場合は諮問不要であるが、「**参加人から反対する旨の申出がされている場合を除く**」。参加人が反対している場合には、審査請求人の希望にかかわらず諮問が必要。
    5. ○ 第43条第1項第6号のとおり。不適法な審査請求を却下する場合は、実体的な判断を伴わないため、諮問は不要。

    **中文要點**: 口頭意見陳述：申立時原則上必須允許。諮問例外的8種情形中，第4號（請求人不希望諮問）的但書很重要：若參加人提出反對，則仍須諮問。不合法駁回、全部撤銷（無反對意見）、全部認容（無反對意見）等情形也不需諮問。

    📚 **関連条文**: [第31条](../../admin-law/appeal/ch02-review-request.md#31)、[第43条](../../admin-law/appeal/ch02-review-request.md#43)

### 問題 5（令和元年度 問14）

行政不服審査法に規定する裁決に関する次の記述のうち、妥当なものはどれか。

1. 処分についての審査請求が不適法である場合には、審査庁は裁決で当該審査請求を棄却する。
2. 処分が違法又は不当であるが、これを取り消すことにより公の利益に著しい障害を生ずる場合には、審査庁は裁決で審査請求を棄却することができるが、この場合に審査庁は裁決の主文で当該処分が違法又は不当であることを宣言する必要はない。
3. 審査庁が処分庁の上級行政庁又は処分庁のいずれでもない場合であっても、審査庁は認容裁決において処分を変更することができる。
4. 審査庁は、審査請求人の不利益に当該処分を変更する裁決をすることはできない。
5. 裁決は、審査請求人に対してのみ効力を有し、関係行政庁を拘束するものではない。

??? success "解答・解説"
    **正解：4**

    1. ✗ 不適法な審査請求は「**却下**」であり、「棄却」ではない（第45条第1項）。却下は手続的な門前払い、棄却は実体審理の結果の否定。
    2. ✗ いわゆる事情裁決（第45条第3項）では、審査請求を棄却することができるが、裁決の主文で当該処分が「**違法又は不当であることを宣言しなければならない**」。宣言義務がある。
    3. ✗ 第46条第1項ただし書により、審査庁が処分庁の上級行政庁又は処分庁の**いずれでもない場合**には、処分を**変更することはできない**。取消しのみ可能。
    4. ○ 第48条のとおり。**不利益変更の禁止**。審査庁は認容裁決において、審査請求人の不利益に処分を変更し、又は事実上の行為を変更することはできない。
    5. ✗ 第52条第1項により、裁決は**関係行政庁を拘束する**（拘束力）。処分庁は裁決の趣旨に従い改めて処分をしなければならない。

    **中文要點**: 裁決三類型：却下（程序不合法）、棄却（無理由）、認容（撤銷/變更）。事情裁決時必須宣告違法不當。變更裁決僅限上級機關或處分機關本身。不利益變更禁止原則保護審查請求人。裁決具有拘束力，相關行政機關必須遵守。

    📚 **関連条文**: [第45条](../../admin-law/appeal/ch02-review-request.md#45)、[第46条](../../admin-law/appeal/ch02-review-request.md#46)、[第48条](../../admin-law/appeal/ch02-review-request.md#48)、[第52条](../../admin-law/appeal/ch02-review-request.md#52)
