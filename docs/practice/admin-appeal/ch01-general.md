# 第1章 総則 — 練習問題

!!! info "本章出題傾向"
    総則（第1条〜第8条）に関する問題は、行政不服審査法の目的規定と行政事件訴訟法との比較、審査請求の対象（処分・不作為）、適用除外が中心。過去10年間で約3〜5回出題。
    重要テーマ：目的規定の比較（行審法 vs 行訴法）、不服申立て類型、適用除外（第7条）

---

## 択一式

### 問題 1（令和3年度 問14）

行政不服審査法の目的及び対象に関する次の記述のうち、妥当なものはどれか。

1. 行政不服審査法は、国民の権利利益の救済を図ることを目的とするが、行政の適正な運営を確保することはその目的に含まれていない。
2. 行政不服審査法は、行政庁の違法な処分に対する不服申立ての制度を定めるものであり、不当な処分については審査の対象とならない。
3. 行政不服審査法は、国民が簡易迅速かつ公正な手続の下で広く行政庁に対する不服申立てをすることができるための制度を定めるものである。
4. 行政不服審査法に基づく審査請求において、審査庁は処分の違法性のみを審査し、裁量権の行使の当否については審査することができない。
5. 行政事件訴訟法は、国民の権利利益の救済を図るとともに行政の適正な運営を確保することを目的としている点で、行政不服審査法と同一の目的を有する。

??? success "解答・解説"
    **正解：3**

    1. ✗ 行政不服審査法第1条は「国民の権利利益の救済を図る」と「**行政の適正な運営を確保する**」の**両方**を目的として掲げている。
    2. ✗ 行政不服審査法は「違法又は**不当**な処分」を対象とする（第1条第1項）。不当な処分も審査の対象に含まれる点が行訴法との大きな違いである。
    3. ○ 行政不服審査法第1条第1項のとおり。「簡易迅速かつ公正な手続」がキーワード。
    4. ✗ 行政不服審査法は違法性だけでなく**不当性**（裁量権の行使の当否）も審査対象とする。行訴法では裁量権の逸脱濫用のみが違法として審査される。
    5. ✗ 行政事件訴訟法の目的は「国民の権利利益の**実効的**救済」であり、「行政の適正な運営の確保」は含まれていない。

    **中文要點**: 行政不服審查法有兩大目的：國民權利利益之救濟＋行政之適正運作。行政事件訴訟法僅有「國民權利利益之實效的救濟」，不含行政適正運作。此外，行審法的審查範圍涵蓋違法與不當，而行訴法僅審查違法（裁量逸脫濫用）。

    📚 **関連条文**: [第1条](../../admin-law/appeal/ch01-general.md#1)

### 問題 2（令和元年度 問15）

行政不服審査法に規定する不服申立ての類型及び適用除外に関する次の記述のうち、妥当でないものはどれか。

1. 行政庁の処分に不服がある者は、法律に特別の定めがある場合を除き、審査請求をすることができる。
2. 法令に基づき行政庁に対して処分についての申請をした者は、相当の期間が経過したにもかかわらず不作為がある場合には、当該不作為についての審査請求をすることができる。
3. 再調査の請求は、法律に再調査の請求をすることができる旨の定めがある場合に限り、することができる。
4. 再審査請求は、法律に再審査請求をすることができる旨の定めがある場合に限り、することができる。
5. 国会の両院若しくは一院又は議会の議決によってされる処分についても、行政不服審査法に基づく審査請求をすることができる。

??? success "解答・解説"
    **正解：5**

    1. ○ 第2条のとおり。行政庁の処分に不服がある者は審査請求をすることができる。行政不服審査法は一般法としての地位を有する。
    2. ○ 第3条のとおり。不作為（法令に基づく申請に対して何らの処分もしないこと）についても審査請求が可能。なお、不作為の審査請求には第18条の期間制限は適用されない。
    3. ○ 第5条第1項のとおり。再調査の請求は「法律に定めがある場合」に限られる。
    4. ○ 第6条第1項のとおり。再審査請求も「法律に定めがある場合」に限られる。
    5. ✗ 第7条第1項第1号により、国会の両院若しくは一院又は議会の議決によってされる処分は、行政不服審査法の**適用除外**とされている。その他、裁判所の裁判、人事院の判定等も適用除外。

    **中文要點**: 不服申訴有三種類型：審查請求（一般類型）、再調查之請求（法律有規定時）、再審查請求（法律有規定時）。國會議決所為之處分屬於適用除外，不得提起審查請求。此外，不作為之審查請求沒有期間限制。

    📚 **関連条文**: [第2条](../../admin-law/appeal/ch01-general.md#2)、[第7条](../../admin-law/appeal/ch01-general.md#7)
