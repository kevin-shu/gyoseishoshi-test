# 第5章 届出・意見公募手続・補則 — 練習問題

!!! info "本章出題傾向"
    届出（第37条）及び意見公募手続（パブリックコメント、第38条〜第45条）に関する問題は、過去10年間で約4〜5回出題。
    最頻出テーマ：届出の到達主義（第37条）、意見公募手続の対象（命令等を定める場合）、意見公募の期間（30日以上）、結果の公示義務。
    届出と申請の区別は第2条の定義とセットで問われることが多い。

---

## 択一式

### 問題 1（令和元年度 問13）

行政手続法に規定する届出及び意見公募手続に関する次の記述のうち、妥当でないものはどれか。

1. 届出が届出書の記載事項に不備がないこと等の法令に定められた形式上の要件に適合している場合は、当該届出が届出の提出先とされている機関の事務所に到達したときに、届出をすべき手続上の義務が履行されたものとされる。
2. 届出は、行政庁に対し一定の事項の通知をする行為であり、行政庁の諾否の応答を求めるものではない点で、申請と異なる。
3. 命令等制定機関は、命令等を定めようとする場合には、原則として、当該命令等の案及びこれに関連する資料をあらかじめ公示して、意見の提出先及び意見の提出のための期間を定めて広く一般の意見を求めなければならない。
4. 意見公募手続において、意見提出期間は、公示の日から起算して20日以上でなければならない。
5. 命令等制定機関は、意見公募手続を実施して命令等を定めた場合には、当該命令等の公布と同時期に、提出意見及びこれに対する当該命令等制定機関の考えを公示しなければならない。

??? success "解答・解説"
    **正解：4**

    1. ○ 第37条のとおり。届出は形式要件を満たして提出先に到達した時点で手続上の義務が履行されたとされる（到達主義）。行政庁の受理・不受理の判断は不要。
    2. ○ 第2条第3号（申請）と第7号（届出）の定義の違い。申請は行政庁の「諾否の応答」を求めるものであるのに対し、届出は一方的な「通知」行為であり、諾否の応答は求められない。
    3. ○ 第39条第1項のとおり。命令等を定めようとするときは、原則として意見公募手続を実施しなければならない。
    4. ✗ 意見提出期間は、公示の日から起算して**30日以上**でなければならない（第39条第3項）。20日ではない。
    5. ○ 第43条第1項のとおり。命令等を定めた場合には、命令等の公布と同時期に、提出意見、これに対する考え、及び命令等の案の修正内容を公示しなければならない。

    **中文要點**: 届出採「到達主義」，形式要件符合且到達提出機關即完成義務，不需行政機關的受理行為。意見公募手續的意見提出期間為「30日以上」，不是20日。命令等制定後，須同時公示提出的意見及行政機關的見解。

    📚 **関連条文**: [第37条](../../admin-law/procedure/ch05-public-comments.md#37)、[第39条](../../admin-law/procedure/ch05-public-comments.md#39)

### 問題 2（平成28年度 問14）

行政手続法に規定する意見公募手続に関する次の記述のうち、正しいものはどれか。

1. 意見公募手続の対象となる「命令等」には、審査基準及び処分基準は含まれるが、行政指導指針は含まれない。
2. 命令等制定機関は、意見公募手続を実施したが提出された意見がなかった場合には、その旨を公示すれば足り、命令等の案の修正の要否を検討する義務はない。
3. 命令等制定機関は、30日以上の意見提出期間を定めなければならないが、やむを得ない理由があるときは、30日を下回る期間を定めることができ、この場合にはその理由を公示しなければならない。
4. 意見公募手続において提出された意見は、命令等制定機関を法的に拘束し、多数の反対意見があった場合には当該命令等を制定してはならない。
5. 地方公共団体の機関が条例に基づき命令等を定める場合にも、行政手続法の意見公募手続に関する規定が直接適用される。

??? success "解答・解説"
    **正解：3**

    1. ✗ 「命令等」には、法律に基づく命令・規則、審査基準、処分基準に加えて、**行政指導指針**も含まれる（第2条第8号ニ）。
    2. ✗ 提出された意見がなかった場合でも、第43条第1項により結果の公示義務がある。また、意見の有無にかかわらず、命令等の案について必要に応じ修正を検討することが求められる。「提出された意見がなかった場合」の取扱いについて「足り」とする記述は不正確。
    3. ○ 第39条第3項本文により意見提出期間は30日以上が原則。ただし同項ただし書により、やむを得ない理由があるときは30日を下回る期間を定めることができる。この場合、第40条第2項により当該理由を公示しなければならない。
    4. ✗ 意見公募手続は、国民の意見を広く聴取する手続であるが、提出された意見に法的拘束力はない。多数の反対意見があっても、命令等を制定すること自体は妨げられない。
    5. ✗ 第3条第3項により、地方公共団体の機関が命令等を定める行為については、行政手続法第2章〜第6章の規定は適用されない。意見公募手続は第6章に規定されているため、直接適用されない。

    **中文要點**: 意見公募手續（公眾評論）的意見提出期間原則為30日以上，但有不得已的理由時可縮短（須公示理由）。「命令等」的範圍包含審查基準、處分基準和行政指導指針。提出的意見不具法律拘束力。地方公共團體依條例制定命令等時，行政手續法的意見公募規定不直接適用（第3條第3項的適用除外）。

    📚 **関連条文**: [第39条](../../admin-law/procedure/ch05-public-comments.md#39)、[第43条](../../admin-law/procedure/ch05-public-comments.md#43)、[第3条](../../admin-law/procedure/ch01-general.md#3)
