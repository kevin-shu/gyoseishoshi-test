# 第2章 申請に対する処分 — 練習問題

!!! info "本章出題傾向"
    申請に対する処分（第5条〜第11条）に関する問題は、過去10年間で約8〜10回出題。
    最頻出テーマ：審査基準の設定・公表義務（第5条）と処分基準（第12条）の比較、標準処理期間の設定義務の性質（第6条）、申請拒否の理由提示（第8条）。
    第5条の審査基準（法定義務）と第12条の処分基準（努力義務）の混同を狙った選択肢が非常に多い。

---

## 択一式

### 問題 1（令和2年度 問11）

行政手続法に規定する審査基準と処分基準に関する次の記述のうち、正しいものはどれか。

1. 審査基準の設定は法定義務であるが、処分基準の設定も同様に法定義務である。
2. 審査基準を定めるに当たっては、許認可等の性質に照らしてできる限り具体的なものとしなければならないが、処分基準についてはそのような定めはない。
3. 審査基準の公表は法定義務であるが、処分基準の公表は努力義務にとどまる。
4. 審査基準は行政上特別の支障があるときは公にしないことができるが、処分基準についてはそのような例外は認められていない。
5. 審査基準は命令等に該当するが、処分基準は命令等に該当しない。

??? success "解答・解説"
    **正解：3**

    1. ✗ 審査基準の設定は**法定義務**（第5条第1項「定めるものとする」）だが、処分基準の設定は**努力義務**（第12条第1項「努めなければならない」）である。
    2. ✗ 処分基準についても「不利益処分の性質に照らしてできる限り具体的なものとしなければならない」と規定されている（第12条第2項）。
    3. ○ 審査基準の公表は法定義務（第5条第3項「公にしておかなければならない」）であり、処分基準の公表は努力義務（第12条第1項「公にしておくよう努めなければならない」）である。
    4. ✗ 処分基準の公表はそもそも努力義務なので「例外が認められていない」という記述は不適切。なお、審査基準の公表の例外は「行政上特別の支障があるとき」（第5条第3項ただし書）。
    5. ✗ 審査基準も処分基準も、いずれも「命令等」に該当する（第2条第8号ロ・ハ）。

    **中文要點**: 審查基準的設定和公開都是「法定義務」，處分基準的設定和公開都僅是「努力義務」。這個對比是行政手續法中最常被考的陷阱之一。記住口訣：「審查＝法定、處分＝努力」。

    📚 **関連条文**: [第5条](../../admin-law/procedure/ch02-application.md#5)、[第12条](../../admin-law/procedure/ch03-adverse.md#12)

### 問題 2（令和4年度 問12）

行政手続法に規定する標準処理期間に関する次の記述のうち、妥当なものはどれか。

1. 標準処理期間の設定は法定義務であり、行政庁は必ずこれを定めなければならない。
2. 標準処理期間を定めたときは、これを公にしておくよう努めなければならない。
3. 標準処理期間は、申請が行政庁の事務所に到達してから処分をするまでの期間をいい、申請の提出先が行政庁と異なる場合の送付期間は含まれない。
4. 標準処理期間の設定は努力義務であるが、これを定めたときは公にしておかなければならない。
5. 標準処理期間を経過しても処分がなされない場合、当該申請は自動的に許可されたものとみなされる。

??? success "解答・解説"
    **正解：4**

    1. ✗ 標準処理期間の設定は**努力義務**である（第6条「定めるよう努める」）。法定義務ではない。
    2. ✗ 標準処理期間を定めたときの公表は**法定義務**である（第6条「公にしておかなければならない」）。「努めなければならない」ではない。
    3. ✗ 申請の提出先が行政庁と異なる場合、提出先から行政庁に到達するまでの期間も「併せて」標準処理期間に含まれる（第6条かっこ書）。
    4. ○ 設定は努力義務だが、定めた場合の公表は法定義務。第6条の正確な理解。
    5. ✗ 標準処理期間はあくまで「標準的な」期間であり、法的拘束力はない。経過しても申請が自動許可されることはない。

    **中文要點**: 標準處理期間的設定是「努力義務」，但一旦制定後，公開則是「法定義務」。這個「設定＝努力、公開＝法定」的組合是考試常見考點。另外，標準處理期間不具有法律拘束力，超過期限也不會自動許可。

    📚 **関連条文**: [第6条](../../admin-law/procedure/ch02-application.md#6)

### 問題 3（平成30年度 問13）

行政手続法に規定する申請に対する処分の理由の提示に関する次の記述のうち、妥当でないものはどれか。

1. 行政庁は、申請により求められた許認可等を拒否する処分をする場合は、申請者に対し、同時に、当該処分の理由を示さなければならない。
2. 申請により求められた許認可等を拒否する処分を書面でするときは、理由も書面により示さなければならない。
3. 法令に定められた許認可等の要件又は公にされた審査基準が数量的指標その他の客観的指標により明確に定められている場合であって、当該申請がこれらに適合しないことが申請書の記載又は添付書類から明らかであるときは、申請者の求めがあったときにこれを示せば足りる。
4. 理由の提示は、いかなる場合であっても、申請者が処分と同時に知りうるようにしなければならず、事後的に理由を補充することは一切認められない。
5. 申請に対する処分の理由の提示については、判例は、いかなる資料に基づきいかなる法規を適用して拒否されたかを、申請者において了知し得るものでなければならないとしている。

??? success "解答・解説"
    **正解：4**

    1. ○ 第8条第1項本文のとおり。申請拒否処分には同時に理由を提示しなければならない。
    2. ○ 第8条第2項のとおり。書面による処分には書面で理由を示す。
    3. ○ 第8条第1項ただし書のとおり。客観的指標により不適合が明らかな場合は、申請者の求めがあったときに示せば足りる。
    4. ✗ 第8条第1項ただし書は、客観的指標に明確に適合しない場合の例外を定めている。また、判例上も理由の事後的補充が全く認められないわけではなく、処分時に一定の理由を示している場合に訴訟段階で補充的理由を追加することが認められる余地がある。「一切認められない」は誤り。
    5. ○ 旅券発給拒否事件（最判昭60.1.22）の趣旨。理由の提示は、申請者がいかなる事実関係に基づきいかなる法規を適用して拒否されたかを了知しうるものでなければならない。

    **中文要點**: 申請拒否處分的理由提示是法定義務，原則上須同時提示。例外為客觀指標明確且不適合明顯時，可於申請人請求時提示。理由提示的程度需讓申請人能了解是基於何種事實適用何種法規而被拒否（旅券發給拒否事件判例）。

    📚 **関連条文**: [第8条](../../admin-law/procedure/ch02-application.md#8)
