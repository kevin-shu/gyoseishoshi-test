# 第1章 総則 — 練習問題

!!! info "本章出題傾向"
    総則（第1条〜第4条）に関する問題は、過去10年間で約5〜6回出題。
    最頻出テーマ：第3条第3項の地方公共団体の条例・規則に基づく処分の適用除外、第4条第1項の「固有の資格」。
    第3条第1項の適用除外16号は個別に問われることは少ないが、分類理解は必要。

---

## 択一式

### 問題 1（令和3年度 問11）

行政手続法の適用除外に関する次の記述のうち、正しいものはどれか。

1. 地方公共団体の機関がする処分であって、その根拠となる規定が法律に置かれているものについても、行政手続法第2章から第6章までの規定は適用されない。
2. 地方公共団体の機関がする処分であって、その根拠となる規定が条例に置かれているものについては、行政手続法第2章から第6章までの規定は適用されないが、第1章の規定は適用される。
3. 地方公共団体の機関に対する届出であって、その根拠となる規定が条例に置かれているものについては、行政手続法第2章から第4章の2までの規定のみが適用されない。
4. 地方公共団体の機関が命令等を定める行為については、その根拠が条例であるか法律であるかを問わず、行政手続法第6章の規定は適用されない。
5. 地方公共団体の機関がする行政指導であって、その根拠となる規定が条例に置かれているものについては、行政手続法の規定は一切適用されない。

??? success "解答・解説"
    **正解：2**

    1. ✗ 根拠規定が**法律**に置かれている処分は、地方公共団体の機関がする場合であっても行政手続法が適用される。適用除外となるのは「条例又は規則」に根拠があるものに限る（第3条第3項）。
    2. ○ 第3条第3項により、条例に根拠を置く処分には第2章〜第6章が適用されないが、第1章（総則）は適用除外の対象外であり適用される。
    3. ✗ 届出についても第2章〜第6章の全体が適用除外となる（第3条第3項）。「第2章から第4章の2まで」に限定されるのは第3条第1項の処分・行政指導の適用除外。
    4. ✗ 地方公共団体の機関が命令等を定める行為の適用除外は、第3条第3項により「条例又は規則」に根拠がある場合に限られる。法律に根拠がある場合は適用される。
    5. ✗ 第3条第3項により第2章〜第6章が適用除外となるが、「一切適用されない」は誤り。第1章（総則）は適用される。

    **中文要點**: 地方公共團體之機關依「條例或規則」為根據所為之處分、行政指導、届出及命令等制定行為，不適用第2章至第6章，但第1章（總則）仍然適用。依「法律」為根據者，行政手續法全面適用。這是第3條第3項的核心考點。

    📚 **関連条文**: [第3条](../../admin-law/procedure/ch01-general.md#3)

### 問題 2（平成29年度 問12）

行政手続法第4条に規定する国の機関等に対する処分等の適用除外に関する次の記述のうち、妥当でないものはどれか。

1. 国の機関に対する処分であって、当該機関がその固有の資格において処分の名あて人となるものについては、行政手続法の規定は適用されない。
2. 地方公共団体がその固有の資格において届出をすべきこととされているものについては、行政手続法の規定は適用されない。
3. 「固有の資格」とは、一般私人が立ちえない立場、すなわち国又は地方公共団体のみが立ちうる立場をいう。
4. 国立大学法人が建築基準法に基づく建築確認を申請する場合は、国の機関がその固有の資格において申請するものに該当するため、行政手続法の規定は適用されない。
5. 法律により直接に設立された法人に対する処分であって、当該法人の監督に関する法律の特別の規定に基づいてされるものについては、原則として第2章及び第3章の規定は適用されないが、当該法人の解散を命ずる処分については適用される。

??? success "解答・解説"
    **正解：4**

    1. ○ 第4条第1項のとおり。国の機関が「固有の資格」で処分の名あて人となる場合、行政手続法は適用されない。
    2. ○ 第4条第1項後段のとおり。地方公共団体がその固有の資格において届出をすべき場合も適用除外。
    3. ○ 「固有の資格」の解釈として確立された理解であり、行政事件訴訟法や行政不服審査法の解釈とも共通する。
    4. ✗ 国立大学法人は独立の法人格を有し、建築確認の申請は一般私人も行いうる行為であるため、「固有の資格」には該当しない。したがって行政手続法は適用される。
    5. ○ 第4条第2項本文及びかっこ書のとおり。特殊法人に対する監督法律上の処分は原則として第2章・第3章の適用除外だが、解散命令・設立認可取消・役員解任命令は除外されず、行政手続法が適用される。

    **中文要點**: 「固有的資格」是指只有國家或地方公共團體才能處於的地位。國立大學法人申請建築確認屬於一般人也能進行的行為，不屬於「固有的資格」，因此行政手續法仍然適用。這是區分適用與否的關鍵概念。

    📚 **関連条文**: [第4条](../../admin-law/procedure/ch01-general.md#4)
