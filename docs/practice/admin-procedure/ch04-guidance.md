# 第4章 行政指導・処分等の求め — 練習問題

!!! info "本章出題傾向"
    行政指導（第32条〜第36条の2）及び処分等の求め（第36条の3）に関する問題は、過去10年間で約5〜7回出題。
    最頻出テーマ：行政指導の一般原則（第32条）の任意性、不利益取扱いの禁止、中止等の求め（第36条の2）、処分等の求め（第36条の3）の要件。
    2014年改正で追加された第36条の2・第36条の3は近年の出題増加傾向。

---

## 択一式

### 問題 1（令和3年度 問13）

行政手続法に規定する行政指導に関する次の記述のうち、妥当なものはどれか。

1. 行政指導は処分に該当するため、行政指導に不服がある場合には、行政不服審査法に基づく審査請求をすることができる。
2. 行政指導に携わる者は、その相手方が行政指導に従わなかったことを理由として、不利益な取扱いをすることが許される場合がある。
3. 行政指導が口頭でされた場合において、その相手方から行政指導の趣旨・内容・責任者を記載した書面の交付を求められたときは、行政上特別の支障がない限り、これを交付しなければならない。
4. 同一の行政目的を実現するため一定の条件に該当する複数の者に対し行政指導をしようとするときは、行政機関は行政指導指針を定めなければならず、かつ、これを公表しなければならない。
5. 行政指導の相手方は、当該行政指導が法律に根拠を有するものでなければ、これに従う義務を負わない旨を主張して拒否することができる。

??? success "解答・解説"
    **正解：3**

    1. ✗ 行政指導は「処分に該当しないもの」と定義されている（第2条第6号）。処分ではないため、行政不服審査法に基づく審査請求の対象とならない。
    2. ✗ 第32条第2項により、行政指導に従わなかったことを理由とする不利益取扱いは**禁止**されている。例外はない。
    3. ○ 第35条第3項のとおり。口頭の行政指導について、相手方から書面交付を求められた場合、行政上特別の支障がない限り交付義務がある。
    4. ✗ 行政指導指針の策定については、第36条が「あらかじめ、事案に応じ、行政指導指針を定め、かつ、行政上特別の支障がない限り、これを公表しなければならない」と規定しているが、「定めなければならず、かつ、公表しなければならない」という無条件の義務ではない。公表には「行政上特別の支障がない限り」の留保がある。
    5. ✗ 行政指導はそもそも処分ではなく法的拘束力がないため、法律に根拠がなくても行うこと自体は可能。相手方は法律に根拠がないことを主張するまでもなく、任意に従わないことができる（第32条第1項の任意性の原則）。

    **中文要點**: 行政指導不屬於處分，不具有法律拘束力，其實現僅靠相對方的任意協力。口頭行政指導時，相對方可請求交付書面。行政指導指針的公開有「行政上特別的障礙」的留保。最重要的原則是：不得以不遵從行政指導為由對相對方為不利益對待。

    📚 **関連条文**: [第32条](../../admin-law/procedure/ch04-guidance.md#32)、[第35条](../../admin-law/procedure/ch04-guidance.md#35)

### 問題 2（令和5年度 問12）

行政手続法第36条の2に規定する行政指導の中止等の求めに関する次の記述のうち、妥当でないものはどれか。

1. 法令に違反する行為の是正を求める行政指導の相手方は、当該行政指導が当該法律に規定する要件に適合しないと思料するときは、行政指導の中止その他必要な措置をとることを求めることができる。
2. 行政指導の中止等の求めは、書面で行わなければならない。
3. 行政指導の中止等の求めを受けた行政機関は、必要な調査を行い、当該行政指導が法律に規定する要件に適合しないと認めるときは、当該行政指導の中止その他必要な措置をとらなければならない。
4. 行政指導の中止等の求めは、事実上の行為としての行政指導に対するものであるため、この求めに対する行政機関の応答は処分には該当しない。
5. 行政指導の中止等の求めの制度は、平成26年の行政手続法改正により新設されたものである。

??? success "解答・解説"
    **正解：1**

    1. ✗ 第36条の2の行政指導の中止等の求めは、「法律に**基づく**行政指導」（法律の規定に基づくもの）に限られる。単に「法令に違反する行為の是正を求める行政指導」一般ではなく、**法律の根拠を持つ行政指導**が対象。また、「法律」であって「法令」ではない点にも注意。
    2. ○ 第36条の2第2項により、中止等の求めは書面（行政指導の内容、根拠法律の条項、適合しないと思料する理由等を記載）で行わなければならない。
    3. ○ 第36条の2第3項のとおり。行政機関は必要な調査を行い、要件不適合と認めるときは中止等の措置をとらなければならない。
    4. ○ 行政指導の中止等の求めに対する応答は、行政指導が処分でない以上、処分性を有しない。
    5. ○ 平成26年（2014年）改正により、第36条の2（中止等の求め）及び第36条の3（処分等の求め）が新設された。

    **中文要點**: 行政指導中止等之請求（第36條之2）的對象限於「基於法律」的行政指導，而非所有行政指導。此制度與處分等之請求（第36條之3）同為2014年改正新設。行政機關收到請求後須調查，認定不符要件時須採取中止等措施。

    📚 **関連条文**: [第36条の2](../../admin-law/procedure/ch04-guidance.md#362)

### 問題 3（平成30年度 問14）

行政手続法第36条の3に規定する処分等の求めに関する次の記述のうち、正しいものはどれか。

1. 処分等の求めは、行政庁に対し、当該処分又は行政指導がされるべきであることを求める申請であり、行政庁は諾否の応答義務を負う。
2. 処分等の求めをすることができるのは、法令に違反する事実を発見した国民に限られ、法人は処分等の求めをすることができない。
3. 何人も、法令に違反する事実がある場合において、その是正のためにされるべき処分又は行政指導がされていないと思料するときは、処分等の求めをすることができる。
4. 処分等の求めは、口頭でも書面でもすることができ、方式に制限はない。
5. 処分等の求めを受けた行政庁は、必要な調査を行い、その結果に基づき必要があると認めるときは処分又は行政指導をしなければならず、当該調査の結果を求めをした者に通知しなければならない。

??? success "解答・解説"
    **正解：3**

    1. ✗ 処分等の求めは「申請」（第2条第3号）には該当しない。行政庁に対する「端緒の提供」的性格であり、諾否の応答義務を負うものではない。
    2. ✗ 第36条の3は「何人も」と規定しており、自然人に限らず法人も含まれる。また、自己の権利利益が侵害されている必要もない。
    3. ○ 第36条の3第1項のとおり。何人も、法令違反の事実がある場合に、是正のための処分又は行政指導がされていないと思料するときは、処分等の求めができる。
    4. ✗ 処分等の求めは**書面**でしなければならない（第36条の3第2項）。口頭ではできない。
    5. ✗ 行政庁は必要な調査を行い、必要があると認めるときは処分又は行政指導をしなければならないが、**調査の結果を求めをした者に通知する義務は規定されていない**（第36条の3第3項）。

    **中文要點**: 處分等之請求（第36條之3）是2014年改正新設的制度，「任何人」（何人も）都可以提出，不限於自然人或利害關係人。但處分等之請求不是「申請」，行政機關沒有准否回應義務。請求須以書面為之。行政機關須調查並於必要時採取措施，但不負有將調查結果通知請求人的義務。

    📚 **関連条文**: [第36条の3](../../admin-law/procedure/ch04-guidance.md#363)
