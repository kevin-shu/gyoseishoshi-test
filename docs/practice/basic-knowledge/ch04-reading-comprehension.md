# 文章理解 — 解法ガイド

!!! info "本章出題傾向"
    毎年3問固定出題（問58～60）。要旨把握・空欄補充・文章並替が各1問。正答率が非常に高く（R6：93～96%）、確実に全問正解すべき分野。**著作権の関係で過去問は非公開**のため、本ページでは解法テクニックと練習方法を解説する。

---

## 著作権に関する注意

!!! warning "過去問の非公開について"
    文章理解の問題は評論文や論説文の一部を引用して出題されるため、**著作権の関係で原文が公開されていない**。市販の過去問集でも、文章理解の部分は省略されているか、著者の許諾を得た問題のみ収録されている。

    本ページでは、実際の過去問の代わりに**解法テクニック**と**練習指針**を提供する。

## 題型1: 要旨把握

### 解法のポイント

本文の筆者の主張を正しく選ぶ問題。正答率が最も高い題型。

**Step 1**: 選択肢を先に読む（30秒）

- 5つの選択肢のキーワードを把握
- 何が論点かを事前に理解しておく

**Step 2**: 本文を読む際のマーカー

| 注目すべき表現 | 意味 | 例 |
|--------------|------|-----|
| 逆接の接続詞 | その後に筆者の主張 | しかし、だが、ところが |
| 結論の接続詞 | 最終的な結論 | したがって、よって、つまり |
| 強調表現 | 筆者が伝えたい核心 | ～こそ、～にほかならない、～べきである |
| 否定の後の肯定 | 筆者の真の主張 | ～ではなく、～ |

**Step 3**: 筆者の主張は**文末付近**に集中

- 「したがって」「つまり」の後の文が結論であることが多い
- 冒頭は問題提起や一般論であることが多い

**Step 4**: 誤答の典型パターン

| パターン | 内容 |
|---------|------|
| **言い過ぎ** | 本文の内容を誇張している（「すべて」「常に」「絶対」） |
| **すり替え** | 本文の言葉を使いつつ意味を変えている |
| **無関係** | 本文に言及されていない内容 |
| **事実の記述** | 筆者の「主張」ではなく「事実」の記述 |

## 題型2: 空欄補充

### 解法のポイント

空欄に入る接続詞やフレーズを選ぶ問題。前後の論理関係の判断が鍵。

**Step 1**: 空欄の前後の文を精読

**Step 2**: 論理関係を判断

| 前後の関係 | 入る接続詞 |
|-----------|-----------|
| 原因 → 結果 | だから、したがって、そのため |
| 予想に反する展開 | しかし、だが、ところが |
| 前文の具体例 | たとえば、具体的には |
| 前文と同じ内容の言い換え | つまり、すなわち、要するに |
| 後文が前文の理由 | なぜなら、というのも |
| 情報の追加 | また、さらに、しかも |
| 二つの事柄の比較 | 一方、他方、それに対して |
| 話題の転換 | さて、ところで |

**Step 3**: 迷ったら段落全体の流れを確認

- 空欄の前後だけでなく、段落全体の論理構造を把握する
- 空欄が複数ある場合は、確定できるものから埋めていく

### 接続詞の分類（暗記必須）

| 分類 | 代表的な接続詞 |
|------|--------------|
| **順接** | だから、したがって、そのため、よって、それゆえ、ゆえに |
| **逆接** | しかし、だが、ところが、けれども、にもかかわらず、とはいえ |
| **並列・添加** | また、そして、さらに、しかも、その上、加えて |
| **対比** | 一方、他方、それに対して、むしろ、逆に |
| **補足・説明** | なぜなら、というのも、ただし、もっとも |
| **言い換え** | つまり、すなわち、要するに、いわば、換言すれば |
| **例示** | たとえば、具体的には、いわば、一例として |
| **転換** | さて、ところで、では、それでは、次に |

## 題型3: 文章並替

### 解法のポイント

バラバラになった段落を正しい順序に並べ替える問題。三題型中最も難度が高い。

**Step 1**: 冒頭と末尾の段落を確定する

| 位置 | 特徴 |
|------|------|
| **冒頭** | 問題提起、テーマの提示、一般論の導入、「～とは何か」 |
| **末尾** | 結論、まとめ、筆者の主張、「～すべきである」「～にほかならない」 |

**Step 2**: 指示語の連鎖で順序を決定

| 手がかり | 判断方法 |
|---------|---------|
| **この・その・それ** | 指す内容が直前の段落にある |
| **こうした・そうした** | 直前の段落の内容全体を指す |
| **前者・後者** | 直前に二つの事項が挙げられている |

**Step 3**: 接続詞を手がかりにする

| 接続詞 | 前段落との関係 |
|--------|-------------|
| しかし | 前段落と対立する内容が続く |
| たとえば | 前段落の具体例が続く |
| つまり | 前段落の言い換えが続く |
| さらに | 前段落に情報を追加する |

**Step 4**: 一般論 → 具体例の順序を活用

- 抽象的な記述が先、具体的な記述が後
- 固有名詞が初出する段落は、その固有名詞を説明なく使う段落より前に来る

**Step 5**: 通読して自然な流れか確認

## 時間配分ガイド

| 問題 | 推奨時間 | 解法の流れ |
|------|---------|-----------|
| 空欄補充 | **4分** | 前後の論理関係 → 接続詞の分類で判断 |
| 要旨把握 | **5分** | 選択肢→本文→マーカー→照合 |
| 文章並替 | **6分** | 冒頭末尾の確定→指示語連鎖→通読確認 |
| **合計** | **15分** | |

!!! tip "解答順序のアドバイス"
    空欄補充（最も短時間で解ける）→ 要旨把握 → 文章並替（最も時間がかかる）の順で解くのが効率的。

## 練習方法

### 推奨教材

| 教材 | 特徴 | 入手方法 |
|------|------|---------|
| **公務員試験 文章理解** | 出題形式がほぼ同一、問題数豊富 | 書店・Amazon |
| **大学入試 現代文問題集** | 読解力の基礎訓練 | 書店 |
| **新聞社説の要約練習** | 要旨把握力の向上 | 新聞各紙 |
| **予備校の対策講座** | 行政書士試験特化の解法指導 | 各予備校 |

### 日常の練習ルーティン

1. **毎日1問**（5分）: 公務員試験の文章理解を1問解く
2. **接続詞マーカー**: 文章を読む際、接続詞に線を引く習慣をつける
3. **間違い分析**: なぜその選択肢が正解かを論理的に説明できるまで復習
4. **時間計測**: 1問5分の制限時間を守って練習
