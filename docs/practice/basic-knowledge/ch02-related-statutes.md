# 行政書士法等関連諸法令 — 練習問題

!!! info "本章出題傾向"
    R6（2024年）から新設。毎年2問出題。行政書士法・戸籍法・住民基本台帳法から条文知識を直接問う出題が多い。条文素読が最も効率的な対策。

---

## 択一式

### 問題 1（令和6年度 問52）

行政書士法に関する次の記述のうち、妥当なものはどれか。

1. 行政書士は、その事務所の見やすい場所に、その業務に関し受ける報酬の額を掲示しなければならない。
2. 行政書士は、自ら作成した官公署に提出する書類に係る許認可等に関する審査請求について、その手続を代理することはできない。
3. 国または地方公共団体の公務員として行政事務を担当した期間が通算して2年以上になる者は、行政書士となる資格を有する。
4. 破産手続開始の決定を受けた場合、復権をした後においても行政書士となる資格を有しない。
5. 地方公務員が懲戒免職の処分を受けた場合、無期限に行政書士となる資格を有しない。

??? success "解答・解説"
    **正解：1**

    1. ○ 行政書士法10条の2のとおり。行政書士は事務所の見やすい場所に報酬額を掲示する義務がある。
    2. ✗ 行政書士は、自ら作成した書類に係る許認可等に関する審査請求について、その手続を代理することが**できる**（行政書士法1条の3第1号）。
    3. ✗ 公務員特認で行政書士資格を得るには、行政事務を担当した期間が通算して**17年以上**（中卒の場合は20年以上）必要。2年では不足。
    4. ✗ 破産手続開始の決定を受けても、**復権を得た者**は行政書士となる資格を有する。欠格事由は復権により解消される。
    5. ✗ 懲戒免職処分を受けた場合は、処分の日から**3年間**行政書士となる資格を有しない。無期限ではない。

    **中文要點**: 行政書士法規定事務所必須揭示報酬額。公務員特認需17年以上行政事務經驗（非2年）。破產者復權後可取得資格。懲戒免職後的限制期間是3年（非無期限）。

    📚 **関連知識**: [行政書士法等関連諸法令](../../basic-knowledge/ch02-related-statutes.md)

### 問題 2（令和6年度 問53）

住民基本台帳法に明示されている住民票の記載事項に関する次の項目のうち、妥当なものはどれか。

1. 前年度の住民税納税額
2. 緊急時に連絡可能な者の連絡先
3. 地震保険の被保険者である者については、その資格に関する事項
4. 海外渡航歴
5. 世帯主についてはその旨、世帯主でない者については世帯主の氏名および世帯主との続柄

??? success "解答・解説"
    **正解：5**

    1. ✗ 住民税の納税額は住民票の記載事項ではない。
    2. ✗ 緊急連絡先は住民票の記載事項ではない。
    3. ✗ 地震保険に関する事項は住民票の記載事項ではない。
    4. ✗ 海外渡航歴は住民票の記載事項ではない。
    5. ○ 住民基本台帳法7条4号に「世帯主についてはその旨、世帯主でない者については世帯主の氏名及び世帯主との続柄」が記載事項として明記されている。

    **中文要點**: 住民票的法定記載事項包括：姓名、出生年月日、性別、戶長及其續柄、戶籍表示、住所等。住民稅額、緊急聯絡人、地震保險、出國紀錄均非記載事項。

    📚 **関連知識**: [行政書士法等関連諸法令](../../basic-knowledge/ch02-related-statutes.md)

## 補足：過去問が少ない分野の対策

!!! warning "R6新設分野のため過去問が限られる"
    行政書士法等関連諸法令はR6（2024年）から新設された分野のため、直接の過去問はR6・R7の計4問のみ。以下の方法で補完する。

    1. **行政書士法**: 平成17年以前の過去問に行政書士法の出題あり（当時は「行政書士の業務に関し必要な法令等」として出題）
    2. **戸籍法・住民基本台帳法**: 条文数が限られるため、条文の素読が最も効率的
    3. **確認すべき重要条文**:
        - 行政書士法: 1条の2～1条の3（業務範囲）、2条（資格）、2条の2（欠格事由）、9条（帳簿）、10条の2（報酬額の掲示）、12条（守秘義務）、14条（懲戒）
        - 戸籍法: 49条（出生届）、86条（死亡届）、10条（戸籍謄本の請求）
        - 住民基本台帳法: 7条（記載事項）、22条（転入届）、24条（転出届）
