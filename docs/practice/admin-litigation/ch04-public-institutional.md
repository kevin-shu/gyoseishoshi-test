# 第4章 民衆訴訟・機関訴訟 — 練習問題

!!! info "本章出題傾向"
    民衆訴訟・機関訴訟（第42条〜第43条）に関する問題は、過去10年間で約2〜3回出題。
    重要テーマ：客観訴訟の法定主義（第42条）、主観訴訟 vs 客観訴訟の区別、民衆訴訟・機関訴訟の具体例。

---

## 択一式

### 問題 1（令和2年度 問18）

行政事件訴訟法における民衆訴訟および機関訴訟に関する次の記述のうち、妥当なものはどれか。

1. 民衆訴訟とは、国又は公共団体の機関相互間における権限の存否又はその行使に関する紛争についての訴訟をいう。
2. 機関訴訟とは、国又は公共団体の機関の法規に適合しない行為の是正を求める訴訟で、選挙人たる資格その他自己の法律上の利益にかかわらない資格で提起するものをいう。
3. 民衆訴訟および機関訴訟は、法律に定める場合において、法律に定める者に限り提起することができ、この点で主観訴訟と異なる。
4. 民衆訴訟の例として、地方自治法に基づく住民訴訟のほか、国家賠償法に基づく損害賠償請求訴訟がある。
5. 機関訴訟は、国の機関相互間の紛争のみを対象とし、地方公共団体の機関相互間の紛争は含まない。

??? success "解答・解説"
    **正解：3**

    1. ✗ この記述は**機関訴訟**（第6条）の定義である。民衆訴訟の定義は第5条に規定されている。
    2. ✗ この記述は**民衆訴訟**（第5条）の定義である。機関訴訟と民衆訴訟の定義が入れ替わっている。
    3. ○ 第42条のとおり。民衆訴訟および機関訴訟は**法定主義**の下にあり、「法律に定める場合において、法律に定める者に限り」提起できる。これは、法律上の利益があれば提起できる主観訴訟（抗告訴訟・当事者訴訟）との根本的な違いである。
    4. ✗ 国家賠償法に基づく損害賠償請求訴訟は通常の民事訴訟であり、民衆訴訟には該当しない。民衆訴訟の例としては、住民訴訟（地方自治法242条の2）や選挙訴訟（公職選挙法203条等）がある。
    5. ✗ 機関訴訟は国の機関相互間に限られず、**地方公共団体の機関相互間**の紛争も含まれる。例えば、地方自治法176条7項の長と議会との間の訴訟がある。

    **中文要點**: 民眾訴訟（第5條）和機關訴訟（第6條）是客觀訴訟，採法定主義（第42條）：只有法律規定的情況下，由法律規定的人才能提起。這與主觀訴訟（有法律上利益即可起訴）根本不同。民眾訴訟的例子：住民訴訟、選舉訴訟。機關訴訟包括國家機關間和地方公共團體機關間的紛爭。

    📚 **関連条文**: [第42条](../../admin-law/litigation/ch04-public-institutional.md#42)

### 問題 2（平成27年度 問18）

行政事件訴訟法における主観訴訟と客観訴訟の区別に関する次の記述のうち、正しいものはどれか。

1. 主観訴訟とは、個人の権利利益の保護を目的とする訴訟をいい、抗告訴訟のみがこれに分類される。
2. 客観訴訟とは、客観的な法秩序の維持を目的とする訴訟をいい、民衆訴訟および機関訴訟がこれに分類される。
3. 主観訴訟は法律に定める場合に限り提起できるが、客観訴訟は法律上の利益があれば自由に提起できる。
4. 住民訴訟は、住民個人の財産的利益を保護するための訴訟であるから、主観訴訟に分類される。
5. 客観訴訟については、行政事件訴訟法の規定は一切適用されず、個別法の規定のみが適用される。

??? success "解答・解説"
    **正解：2**

    1. ✗ 主観訴訟には抗告訴訟のほか**当事者訴訟**も含まれる。当事者訴訟も個人の権利利益の保護を目的とする訴訟である。
    2. ○ 客観訴訟は客観的な法秩序の適正を確保することを目的とし、民衆訴訟（第5条）と機関訴訟（第6条）がこれに該当する。個人の権利利益の保護ではなく、行政の適法性の確保が目的である。
    3. ✗ 逆である。**客観訴訟**が法律に定める場合に限り提起でき（第42条）、**主観訴訟**は法律上の利益があれば提起できる。
    4. ✗ 住民訴訟（地方自治法242条の2）は**民衆訴訟**であり、**客観訴訟**に分類される。住民は自己の個人的な財産的利益ではなく、地方公共団体の財務の適正を確保するために訴訟を提起する。
    5. ✗ 第43条は、民衆訴訟および機関訴訟について、取消訴訟に関する規定の一部を**準用**している。行政事件訴訟法の規定が一切適用されないわけではない。

    **中文要點**: 主觀訴訟（抗告訴訟＋當事者訴訟）：以保護個人權利利益為目的。客觀訴訟（民眾訴訟＋機關訴訟）：以維護客觀法秩序為目的，採法定主義。住民訴訟是民眾訴訟（客觀訴訟），目的不是保護住民個人的財產利益，而是確保地方公共團體的財務適正。

    📚 **関連条文**: [第42条](../../admin-law/litigation/ch04-public-institutional.md#42)
