# 第2章 抗告訴訟 — 練習問題

!!! info "本章出題傾向"
    抗告訴訟（第8条〜第38条）は行政事件訴訟法の最重要分野。過去10年間で毎年3〜5問出題（択一式＋記述式）。
    重要テーマ：処分性（判例多数）、原告適格（第9条2項の考慮要素）、出訴期間（第14条）、執行停止（第25条）、裁量処分の取消し（第30条）、事情判決（第31条）、判決の拘束力（第33条）、義務付けの訴え（第37条の2・3）、差止めの訴え（第37条の4）

---

## 択一式

### 問題 1（令和4年度 問16）

行政事件訴訟法における「処分性」に関する次の記述のうち、最高裁判所の判例に照らし、妥当なものはどれか。

1. ごみ焼却場の設置行為は、周辺住民の生活環境に直接影響を及ぼすものであるから、抗告訴訟の対象となる「処分」に該当する。
2. 医療法に基づく病院開設中止の勧告は、これに従わない場合に保険医療機関の指定を受けることができなくなるという効果を伴うため、抗告訴訟の対象となる「処分」に該当する。
3. 労災就学援護費の支給・不支給の決定は、労働基準監督署長の裁量的判断に基づく事実上の措置にすぎず、抗告訴訟の対象となる「処分」に該当しない。
4. 都市計画法に基づく用途地域の指定は、個々の国民に対する具体的な権利義務の変動を伴うものであるから、抗告訴訟の対象となる「処分」に該当する。
5. 食品衛生法に基づく通知（輸入届出に係る食品が同法に違反する旨の通知）は、単なる観念の通知であるから、抗告訴訟の対象となる「処分」に該当しない。

??? success "解答・解説"
    **正解：2**

    1. ✗ 最判昭39.10.29（ごみ焼却場設置事件）。ごみ焼却場の設置行為は、行政権の主体が事業遂行のためにする行為であり、公権力の行使には該当しないとして、処分性を**否定**した。
    2. ○ 最判平17.7.15（医療法勧告事件）。勧告自体は法的拘束力を持たないが、勧告に従わない場合に保険医療機関の指定を受けられなくなるという不利益を伴うため、実質的には病院開設の許否を決する効果を有し、処分性を**肯定**した。
    3. ✗ 最判平15.9.4（労災就学援護費事件）。労災就学援護費の支給・不支給の決定は、被災労働者又は遺族の権利に直接影響を及ぼす法的効果を有するものであり、処分性を**肯定**した。
    4. ✗ 最判昭57.4.22。用途地域の指定は一般的抽象的な規制であり、直ちに個々人の権利義務に変動を生じさせるものではないとして、処分性を**否定**した。
    5. ✗ 最判平16.4.26。食品衛生法違反通知は、輸入届出にかかる食品の輸入を事実上不可能にする効果を有するため、処分性を**肯定**した。

    **中文要點**: 處分性的判斷重點在於是否對國民的權利義務產生直接的法律效果。醫療法勸告表面上只是「勸告」，但實質上會導致無法取得保險醫療機關指定，因此肯定處分性。垃圾焚燒場設置和用途地域指定則否定處分性。

    📚 **関連条文**: [第3条](../../admin-law/litigation/ch01-general.md#3)

### 問題 2（令和2年度 問16）

行政事件訴訟法第9条の原告適格に関する次の記述のうち、最高裁判所の判例に照らし、妥当でないものはどれか。

1. 主婦連ジュース不当表示事件において、最高裁は、不当景品類及び不当表示防止法に基づく公正取引委員会の不服申立てにつき、一般消費者の利益は同法の保護する個別的利益ではなく、原告適格を否定した。
2. もんじゅ原子炉設置許可事件において、最高裁は、原子炉の周辺に居住する住民について、原子炉事故により生命・身体等に直接的かつ重大な被害を受けることが想定される範囲の住民に原告適格を認めた。
3. 小田急線高架訴訟（大法廷判決）において、最高裁は、都市計画事業の認可に対する取消訴訟につき、事業地の周辺住民であって騒音等による健康又は生活環境に著しい被害を直接的に受けるおそれのある者に原告適格を認めた。
4. サテライト大阪事件において、最高裁は、場外車券発売施設の設置許可の取消しを求める訴訟につき、施設の周辺に居住する住民全員に原告適格を認めた。
5. 行政事件訴訟法第9条2項は、処分の相手方以外の者の原告適格の判断にあたり、当該法令の趣旨及び目的並びに当該処分において考慮されるべき利益の内容及び性質を考慮すべきことを定めている。

??? success "解答・解説"
    **正解：4**

    1. ○ 最判昭53.3.14（主婦連ジュース事件）。一般消費者の利益は法律が保護する反射的利益にすぎず、不服申立ての利益（原告適格）を否定した。
    2. ○ 最判平4.9.22（もんじゅ事件）。原子炉設置許可の際に安全審査で考慮される範囲の住民には、個別的利益として原告適格が認められるとした。
    3. ○ 最大判平17.12.7（小田急高架訴訟）。第9条2項の趣旨を踏まえ、関係法令の趣旨・目的を参酌して、騒音等の被害を受けるおそれのある周辺住民の原告適格を肯定した。
    4. ✗ 最判平21.10.15（サテライト大阪事件）。場外車券発売施設の周辺住民**全員**に原告適格を認めたわけではなく、自転車競技法および関連法令の趣旨・目的を考慮し、位置基準（文教施設・医療施設から一定距離内）に基づく**限定的な範囲**の者にのみ原告適格を認めた。
    5. ○ 第9条2項のとおり。2004年改正で追加された規定であり、原告適格の判断基準を明文化したものである。

    **中文要點**: 原告適格的判斷以「法律上的利益」為基準。第9條2項（2004年改正新設）要求法院不僅看法令文義，還要考慮法令的旨趣目的及利益的內容性質。衛星大阪案並非承認周邊住民全員的原告適格，而是限定在位置基準範圍內的人。

    📚 **関連条文**: [第9条](../../admin-law/litigation/ch02-appeal-litigation.md#9)

### 問題 3（平成30年度 問17）

行政事件訴訟法における被告適格に関する次の記述のうち、正しいものはどれか。

1. 取消訴訟は、処分をした行政庁を被告として提起しなければならない。
2. 処分をした行政庁が国に所属する場合、国を被告として取消訴訟を提起する。
3. 処分をした行政庁が地方公共団体に所属する場合、当該処分をした行政庁の長を被告として取消訴訟を提起する。
4. 裁決の取消しの訴えは、裁決をした行政庁が所属する国又は公共団体を被告として提起するのが原則であるが、当事者の合意により当該行政庁を被告とすることもできる。
5. 処分をした行政庁が国又は公共団体に所属しない場合は、当該行政庁を被告として取消訴訟を提起する。

??? success "解答・解説"
    **正解：5**

    1. ✗ 2004年改正により、被告は処分をした行政庁ではなく、処分をした行政庁が所属する**国又は公共団体**となった（第11条1項）。旧法の規定と混同しないこと。
    2. ○ とも考えられるが、本問では5がより正確。第11条1項のとおり、国に所属する行政庁の処分は国を被告とする。ただし、本肢はこの点のみ述べており、正しい。
    3. ✗ 地方公共団体に所属する場合は**当該地方公共団体**を被告とする（第11条1項）。行政庁の長ではない。
    4. ✗ 当事者の合意による被告の変更は認められていない。被告適格は法律で定められた強行規定である。
    5. ○ 第11条2項。処分をした行政庁が国又は公共団体に所属しない場合は、当該行政庁を被告として提起する。

    **中文要點**: 2004年改正の重要ポイント：被告は「行政廳」から「國或公共團體」に變更された（第11條1項）。但行政廳不屬於國或公共團體時，以該行政廳為被告（第11條2項）。

    📚 **関連条文**: [第11条](../../admin-law/litigation/ch02-appeal-litigation.md#11)

### 問題 4（令和元年度 問17）

行政事件訴訟法第14条の出訴期間に関する次の記述のうち、正しいものはどれか。

1. 取消訴訟は、処分又は裁決があったことを知った日から3か月以内に提起しなければならない。
2. 取消訴訟は、処分又は裁決があったことを知った日から6か月を経過したときは提起することができないが、正当な理由があるときはこの限りでない。
3. 取消訴訟は、処分又は裁決の日から3年を経過したときは提起することができない。
4. 出訴期間は、処分又は裁決につき審査請求をすることができる場合において、審査請求があったときは、その審査請求に対する裁決があったことを知った日から起算される。
5. 出訴期間の定めは、不作為の違法確認の訴えにも準用される。

??? success "解答・解説"
    **正解：2**

    1. ✗ 出訴期間は**6か月**以内であり、3か月ではない（第14条1項）。なお、2004年改正前は3か月であった。
    2. ○ 第14条1項のとおり。処分又は裁決があったことを知った日から6か月（主観的出訴期間）。正当な理由があるときは例外が認められる。
    3. ✗ 客観的出訴期間は処分又は裁決の日から**1年**であり、3年ではない（第14条2項）。
    4. ✗ 審査請求を経た場合の出訴期間は、裁決があったことを知った日又は裁決の日から起算される（第14条3項）。ただし、本肢は「審査請求をすることができる場合」とするのみで、実際に審査請求前置が要求され審査請求がされた場合の規律を述べている。記述自体は概ね正しいが、「起算される」ではなく正確には裁決があったことを知った日から6か月の期間制限がある。
    5. ✗ 不作為の違法確認の訴え（第37条）には出訴期間の定めは**準用されない**（第38条参照）。不作為が継続する限り提起可能。

    **中文要點**: 出訴期間是主觀期間6個月（知道處分之日起）＋客觀期間1年（處分之日起）。2004年改正將主觀期間從3個月延長為6個月。不作為違法確認之訴沒有出訴期間限制。

    📚 **関連条文**: [第14条](../../admin-law/litigation/ch02-appeal-litigation.md#14)

### 問題 5（令和5年度 問17）

行政事件訴訟法第25条の執行停止に関する次の記述のうち、正しいものはどれか。

1. 処分の取消しの訴えの提起は、処分の効力、処分の執行又は手続の続行を妨げるものであり、訴え提起により当然に執行が停止される。
2. 執行停止は、本案について理由がないとみえるときであっても、重大な損害を避けるため緊急の必要があるときは認められる。
3. 裁判所は、処分の取消しの訴えの提起があった場合において、処分により生ずる重大な損害を避けるため緊急の必要があるときは、申立てにより、決定をもって、執行停止をすることができる。
4. 裁判所は、執行停止の決定において、処分の効力の停止のほか、処分の内容の変更を命ずることもできる。
5. 執行停止は、公共の福祉に重大な影響を及ぼすおそれがあるときであっても、申立人に生ずる損害が甚大であれば認められる。

??? success "解答・解説"
    **正解：3**

    1. ✗ 取消訴訟の提起は、処分の効力等を妨げ**ない**（**執行不停止原則**、第25条1項）。訴え提起により当然に執行が停止されるわけではない。
    2. ✗ 「本案について理由がないとみえるとき」は、執行停止は認められ**ない**（第25条4項、消極要件）。
    3. ○ 第25条2項のとおり。「重大な損害を避けるため緊急の必要があるとき」に、申立てにより執行停止が認められる。2004年改正で「回復の困難な損害」から「重大な損害」に要件が緩和された。
    4. ✗ 執行停止は処分の効力の停止、処分の執行の停止又は手続の続行の停止に限られる（第25条2項）。処分の内容の**変更**を命ずることは**できない**。
    5. ✗ 「公共の福祉に重大な影響を及ぼすおそれがあるとき」は、執行停止は認められ**ない**（第25条4項、消極要件）。申立人の損害の大きさにかかわらない。

    **中文要點**: 執行不停止原則：提起撤銷訴訟不會當然停止處分的效力。例外的執行停止需要「重大損害＋緊急必要」（積極要件），且不得有「本案無理由」或「嚴重影響公共福祉」（消極要件）。2004年改正將要件從「難以回復的損害」緩和為「重大損害」。

    📚 **関連条文**: [第25条](../../admin-law/litigation/ch02-appeal-litigation.md#25)

### 問題 6（平成28年度 問17）

行政事件訴訟法における裁量処分の取消しおよび事情判決に関する次の記述のうち、妥当でないものはどれか。

1. 行政庁の裁量処分については、裁量権の範囲を超え又はその濫用があった場合に限り、裁判所はその処分を取り消すことができる。
2. 裁量権の逾越・濫用の有無を判断するにあたっては、処分の基礎とされた重要な事実に誤認があること等により、重要な事実の基礎を欠く場合、又は事実に対する評価が明らかに合理性を欠くこと等により、判断の過程において考慮すべき事情を考慮しない場合は、裁量権の逾越・濫用にあたる。
3. 裁判所は、処分が違法であっても、これを取り消すことにより公の利益に著しい障害を生ずる場合において、原告の受ける損害の程度、その損害の賠償又は防止の程度及び方法その他一切の事情を考慮したうえ、処分を取り消すことが公共の福祉に適合しないと認めるときは、請求を棄却することができる。
4. 事情判決をする場合、裁判所は、判決の主文において処分が違法であることを宣言しなければならない。
5. 事情判決は、裁量処分にのみ適用される制度であり、覊束処分には適用されない。

??? success "解答・解説"
    **正解：5**

    1. ○ 第30条のとおり。裁量処分の取消しは裁量権の逾越・濫用があった場合に限られる。
    2. ○ 最判昭52.12.20（神戸税関事件）等の判例法理を踏まえた記述。判断過程審査として重要な事実の基礎の欠缺や考慮すべき事情の不考慮は裁量権の逾越・濫用にあたる。
    3. ○ 第31条1項のとおり（事情判決）。処分が違法であっても、取消しが公共の福祉に適合しない場合は請求を棄却できる。
    4. ○ 第31条1項後段。事情判決をする場合、主文で処分が違法であることを宣言しなければならない（違法宣言）。
    5. ✗ 事情判決（第31条）は裁量処分に限定されず、**覊束処分を含むすべての違法な処分**について適用される。条文上も「処分又は裁決が違法であるとき」と規定されており、裁量処分に限定する文言はない。

    **中文要點**: 事情判決（第31條）不限於裁量處分，對羈束處分也適用。處分違法但撤銷會嚴重損害公共利益時，法院可駁回請求，但必須在判決主文中宣告處分違法。裁量處分的撤銷限於裁量權的逾越或濫用（第30條）。

    📚 **関連条文**: [第30条](../../admin-law/litigation/ch02-appeal-litigation.md#30)・[第31条](../../admin-law/litigation/ch02-appeal-litigation.md#31)

### 問題 7（令和2年度 問17）

行政事件訴訟法第33条の取消判決の拘束力に関する次の記述のうち、妥当でないものはどれか。

1. 処分を取り消す判決は、その事件について、処分をした行政庁その他の関係行政庁を拘束する。
2. 拘束力は、判決主文に含まれる結論のみならず、判決理由中の違法事由の判断についても生じる。
3. 申請を却下し又は棄却した処分が判決により取り消された場合、その処分をした行政庁は、判決の趣旨に従い、改めて申請に対する処分をしなければならない。
4. 取消判決の拘束力は、第三者効（第32条1項）とは異なり、処分をした行政庁その他の関係行政庁に対してのみ生じる。
5. 取消判決の拘束力により、行政庁は同一事情の下で同一理由に基づく同一処分を繰り返すことが禁止されるが、異なる理由に基づいて再び同一内容の処分をすることは拘束力に反しない。

??? success "解答・解説"
    **正解：2**

    1. ○ 第33条1項のとおり。取消判決は処分をした行政庁その他の関係行政庁を拘束する。
    2. ✗ 拘束力は判決主文が導き出されるのに**必要な理由中の判断**について生じるとされるが、「判決理由中の違法事由の判断」すべてに無限定に生じるわけではない。本肢は拘束力の範囲を過度に広く述べている点で妥当でない。
    3. ○ 第33条2項のとおり（反復禁止効・再度処分義務）。申請拒否処分が取り消された場合、行政庁は判決の趣旨に従い再処分しなければならない。
    4. ○ 第33条1項は拘束力の名宛人を「処分又は裁決をした行政庁その他の関係行政庁」と規定しており、第三者効（第32条1項の対世効）とは区別される。
    5. ○ 拘束力は同一事情の下で同一理由に基づく同一処分の反復を禁止するが、新たな事情又は異なる理由に基づく処分は拘束力に反しない（最判昭和36.2.21等の趣旨）。

    **中文要點**: 拘束力（第33條）使行政廳受判決拘束，不能在同一情況下以同一理由重複同一處分。申請拒否處分被撤銷時，行政廳必須依判決旨趣重新作成處分。拘束力的範圍限於判決主文導出所必要的理由中的判斷，不是理由中所有判斷。

    📚 **関連条文**: [第33条](../../admin-law/litigation/ch02-appeal-litigation.md#33)

### 問題 8（令和3年度 問17）

行政事件訴訟法における義務付けの訴えに関する次の記述のうち、正しいものはどれか。

1. 義務付けの訴えには、非申請型（第37条の2）と申請型（第37条の3）の2類型があり、いずれも2004年改正により新設された。
2. 非申請型義務付けの訴えは、一定の処分がされないことにより重大な損害を生ずるおそれがあり、かつ、その損害を避けるため他に適当な方法があるときであっても提起することができる。
3. 申請型義務付けの訴えは、法令に基づく申請がされた場合に提起できるものであり、当該申請に対する処分が既にされている場合には提起することができない。
4. 非申請型義務付けの訴えにおいては、裁判所は、行政庁がその処分をすべきであることがその処分の根拠となる法令の規定から明らかであると認められ又は行政庁がその処分をしないことがその裁量権の範囲を超え若しくはその濫用となると認められるときは、行政庁がその処分をすべき旨を命ずる判決をする。
5. 申請型義務付けの訴えは、単独で提起することができ、取消訴訟又は無効等確認の訴えを併合提起する必要はない。

??? success "解答・解説"
    **正解：4**

    1. ○ とも考えられるが、本問は正しいものを一つ選ぶ問題であり、4がより正確。なお、1の記述自体は正しい。
    2. ✗ 非申請型義務付けの訴えは、「その損害を避けるため**他に適当な方法がないとき**に限り」提起できる（第37条の2第1項、**補充性の要件**）。
    3. ✗ 申請型義務付けの訴えは、申請に対する処分がされている場合（拒否処分がされた場合）にも提起でき、その場合は取消訴訟又は無効等確認の訴えとの**併合提起**が必要となる（第37条の3第3項2号）。
    4. ○ 第37条の2第5項のとおり。非申請型義務付けの訴えの本案勝訴要件として、法令の規定から処分をすべきことが明らかであること、又は裁量権の逾越・濫用が認められることが必要。
    5. ✗ 申請型義務付けの訴えは、取消訴訟又は無効等確認の訴えを**併合提起しなければならない**（第37条の3第3項）。単独提起は認められない。

    **中文要點**: 義務付之訴分為非申請型（第37條之2）和申請型（第37條之3）。非申請型需要「重大損害＋補充性」。申請型必須與撤銷訴訟或無效確認之訴併合提起。本案勝訴要件：法令上明顯應作成處分，或不作成處分構成裁量權逾越濫用。

    📚 **関連条文**: [第37条の2](../../admin-law/litigation/ch02-appeal-litigation.md#372)・[第37条の3](../../admin-law/litigation/ch02-appeal-litigation.md#373)

### 問題 9（令和5年度 問18）

行政事件訴訟法における差止めの訴え（第37条の4）に関する次の記述のうち、妥当なものはどれか。

1. 差止めの訴えは、行政庁が一定の処分又は裁決をすべきでないにかかわらずこれがされようとしている場合において、一定の処分又は裁決がされることにより生ずる損害を避けるため他に適当な方法があるときでも提起することができる。
2. 差止めの訴えの提起があった場合、処分又は裁決の執行は当然に停止される。
3. 差止めの訴えは、一定の処分又は裁決がされることにより重大な損害を生ずるおそれがある場合に限り提起することができ、その損害の重大性の判断にあたっては、損害の回復の困難の程度を考慮するものとし、損害の性質及び程度並びに処分の内容及び性質をも勘案する。
4. 差止めの訴えの本案勝訴要件は、義務付けの訴えと同じく、行政庁がその処分をすべきでないことがその処分の根拠となる法令の規定から明らかであるか、行政庁がその処分をすることがその裁量権の範囲を超え若しくはその濫用となると認められることである。
5. 差止めの訴えは、2004年改正以前から行政事件訴訟法に規定されていた訴訟類型である。

??? success "解答・解説"
    **正解：3**

    1. ✗ 差止めの訴えは「その損害を避けるため**他に適当な方法がないとき**に限り」提起できる（第37条の4第1項、**補充性の要件**）。
    2. ✗ 差止めの訴えの提起は当然に処分の執行を停止させるものではない。暫定的な救済が必要な場合は、**仮の差止め**（第37条の5第2項）を別途申し立てる必要がある。
    3. ○ 第37条の4第1項・2項のとおり。「重大な損害」の判断にあたっては、損害の回復の困難の程度を考慮し、損害の性質・程度、処分の内容・性質を勘案する。
    4. ✗ 差止めの訴えの本案勝訴要件は、行政庁がその処分をすべきでないことが根拠法令から明らかであるか、行政庁がその処分をすることが**裁量権の逾越・濫用となると認められる**こと（第37条の4第5項）。義務付けの訴えとは「すべき」と「すべきでない」が逆である。
    5. ✗ 差止めの訴えは**2004年改正**により新設されたものである（第3条7項、第37条の4）。

    **中文要點**: 差止之訴（第37條之4）的訴訟要件：①重大損害之虞、②補充性（無其他適當方法）。「重大損害」的判斷考慮損害回復的困難程度、損害的性質程度、處分的內容性質。差止之訴與義務付之訴同為2004年改正新設。

    📚 **関連条文**: [第37条の4](../../admin-law/litigation/ch02-appeal-litigation.md#374)

### 問題 10（令和4年度 問18）

行政事件訴訟法における仮の義務付けおよび仮の差止め（第37条の5）に関する次の記述のうち、妥当でないものはどれか。

1. 義務付けの訴えの提起があった場合において、その義務付けの訴えに係る処分がされないことにより生ずる償うことのできない損害を避けるため緊急の必要があり、かつ、本案について理由があるとみえるときは、裁判所は、申立てにより、仮の義務付けの決定をすることができる。
2. 差止めの訴えの提起があった場合において、その差止めの訴えに係る処分がされることにより生ずる償うことのできない損害を避けるため緊急の必要があり、かつ、本案について理由があるとみえるときは、裁判所は、申立てにより、仮の差止めの決定をすることができる。
3. 仮の義務付けおよび仮の差止めは、公共の福祉に重大な影響を及ぼすおそれがあるときは認められない。
4. 仮の義務付けおよび仮の差止めは、裁判所が職権で行うこともでき、当事者の申立てがなくても発令することができる。
5. 仮の義務付けおよび仮の差止めの要件である「償うことのできない損害」は、執行停止の要件である「重大な損害」よりも厳格な要件である。

??? success "解答・解説"
    **正解：4**

    1. ○ 第37条の5第1項のとおり。仮の義務付けの要件：①償うことのできない損害を避けるための緊急の必要性、②本案について理由があるとみえること。
    2. ○ 第37条の5第2項のとおり。仮の差止めの要件は仮の義務付けと同様。
    3. ○ 第37条の5第3項。公共の福祉に重大な影響を及ぼすおそれがあるときは、仮の義務付け・仮の差止めは認められない（消極要件）。
    4. ✗ 仮の義務付けおよび仮の差止めは、**申立てにより**行われるものであり（第37条の5第1項・2項）、裁判所が**職権で**行うことは**認められていない**。
    5. ○ 仮の義務付け・仮の差止めの「償うことのできない損害」は、執行停止の「重大な損害」（第25条2項）よりも厳格な要件とされている。仮の義務付け・仮の差止めは行政庁に作為・不作為を命じるものであり、より強力な救済手段であるため要件も厳格となる。

    **中文要點**: 假義務付和假差止（第37條之5）需要「無法補償的損害＋緊急必要＋本案有理由」，且不得嚴重影響公共福祉。這比執行停止（重大損害）的要件更為嚴格。假義務付和假差止只能依申立（申請）為之，法院不能職權發動。

    📚 **関連条文**: [第37条の5](../../admin-law/litigation/ch02-appeal-litigation.md#375)

### 問題 1（令和3年度 問44）

Ａは、産業廃棄物処理施設の設置について、Ｂ県知事から許可を受けた事業者であるが、一定の処分がされるべきであるのにされていない場合の訴訟として、行政事件訴訟法が定める非申請型義務付けの訴え（第37条の2）を提起しようとしている。この場合、非申請型義務付けの訴えの訴訟要件（本案勝訴要件を除く。）について、行政事件訴訟法の規定に即して40字程度で記述しなさい。

??? success "解答・解説"
    **模範解答**

    > 「一定の処分がされないことにより重大な損害を生ずるおそれがあり、その損害を避けるため他に適当な方法がないこと。」（52字）

    **別解**

    > 「重大な損害を生ずるおそれがあり、かつ、その損害を避けるため他に適当な方法がないときに限り提起できる。」（49字）

    **解説**

    非申請型義務付けの訴え（第37条の2第1項）の訴訟要件は以下の2つである：

    1. **重大な損害のおそれ**（損害要件）：一定の処分がされないことにより重大な損害を生ずるおそれがあること
    2. **補充性**：その損害を避けるため他に適当な方法がないこと

    なお、本案勝訴要件（第37条の2第5項）は問題で除外されているので記述不要：
    - 法令の規定から処分をすべきことが明らかであること、又は
    - 行政庁がその処分をしないことが裁量権の逾越・濫用にあたること

    **中文要點**: 非申請型義務付之訴（第37條之2）的訴訟要件：①因一定處分未被作成而有重大損害之虞（損害要件），②為避免該損害無其他適當方法（補充性要件）。

    📚 **関連条文**: [第37条の2](../../admin-law/litigation/ch02-appeal-litigation.md#372)

### 問題 2（令和5年度 問44）

Ｃ市の住民Ｄは、Ｃ市長がＤに対して一定の不利益処分をしようとしていることを事前に知ったため、当該処分がされることを事前に阻止するための訴訟を提起しようとしている。行政事件訴訟法が定める差止めの訴え（第37条の4）について、その訴訟要件（本案勝訴要件を除く。）を行政事件訴訟法の規定に即して40字程度で記述しなさい。

??? success "解答・解説"
    **模範解答**

    > 「一定の処分がされることにより重大な損害を生ずるおそれがある場合で、その損害を避けるため他に適当な方法がないこと。」（55字）

    **別解**

    > 「処分により重大な損害を生ずるおそれがあり、かつ損害を避けるため他に適当な方法がないときに限り提起できる。」（51字）

    **解説**

    差止めの訴え（第37条の4第1項）の訴訟要件は以下の2つである：

    1. **重大な損害のおそれ**（損害要件）：一定の処分又は裁決がされることにより重大な損害を生ずるおそれがあること
    2. **補充性**：その損害を避けるため他に適当な方法がないこと

    「重大な損害を生ずるか否か」の判断にあたっては（第37条の4第2項）：
    - 損害の回復の困難の程度を考慮する
    - 損害の性質及び程度並びに処分又は裁決の内容及び性質を勘案する

    本案勝訴要件（第37条の4第5項）は問題で除外されているので記述不要：
    - 行政庁がその処分をすべきでないことが根拠法令から明らかであること、又は
    - 行政庁がその処分をすることが裁量権の逾越・濫用にあたること

    **中文要點**: 差止之訴（第37條之4）的訴訟要件：①因一定處分將被作成而有重大損害之虞（損害要件），②為避免該損害無其他適當方法（補充性要件）。與非申請型義務付之訴的要件結構相同（重大損害＋補充性），但方向相反（義務付＝要求作成處分，差止＝阻止作成處分）。

    📚 **関連条文**: [第37条の4](../../admin-law/litigation/ch02-appeal-litigation.md#374)
