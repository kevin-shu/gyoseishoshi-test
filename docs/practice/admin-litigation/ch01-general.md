# 第1章 総則 — 練習問題

!!! info "本章出題傾向"
    総則（第1条〜第7条）に関する問題は、過去10年間で約2〜3回出題。
    重要テーマ：訴訟類型の4分類（第2条・第3条）、民事訴訟の例による（第7条）。特に第3条の抗告訴訟6類型の定義は頻出。

---

## 択一式

### 問題 1（令和3年度 問16）

行政事件訴訟法に規定する訴訟類型に関する次の記述のうち、正しいものはどれか。

1. 行政事件訴訟法第2条は、行政事件訴訟を、抗告訴訟、当事者訴訟および民衆訴訟の3種類に分類している。
2. 抗告訴訟とは、行政庁の公権力の行使に関する不服の訴訟をいい、第3条に規定される処分の取消しの訴え、裁決の取消しの訴え、無効等確認の訴え、不作為の違法確認の訴え、義務付けの訴えおよび差止めの訴えの6類型が法定されている。
3. 当事者訴訟とは、国民が自己の法律上の利益にかかわりなく、行政の適法性を確保するために提起する訴訟をいう。
4. 民衆訴訟および機関訴訟は、いずれも主観訴訟に分類され、法律上の利益を有する者が提起することができる。
5. 行政事件訴訟法は行政訴訟について網羅的に規定しているため、同法に規定されていない訴訟類型を他の法律で創設することは許されない。

??? success "解答・解説"
    **正解：2**

    1. ✗ 行政事件訴訟は、抗告訴訟、当事者訴訟、民衆訴訟および**機関訴訟**の**4種類**に分類される（第2条）。機関訴訟が欠落している。
    2. ○ 第3条2項〜7項に規定される6類型のとおり。2004年改正で義務付けの訴え（5項）と差止めの訴え（7項）が追加された。
    3. ✗ この記述は**民衆訴訟**（第5条）の定義である。当事者訴訟とは、当事者間の法律関係を確認し又は形成する処分等に関する訴訟で法令の規定により当事者の一方を被告とするもの、及び公法上の法律関係に関する訴訟をいう（第4条）。
    4. ✗ 民衆訴訟および機関訴訟は**客観訴訟**に分類される。法律上の利益の有無にかかわらず、法律に定める場合に限り提起できる（第42条）。
    5. ✗ 第1条は「他の法律に特別の定めがある場合を除くほか」と規定しており、特別法による訴訟類型の創設を排除していない。

    **中文要點**: 行政事件訴訟分為4大類型：抗告訴訟、當事者訴訟（主觀訴訟）及民眾訴訟、機關訴訟（客觀訴訟）。抗告訴訟有6個法定類型，其中義務付之訴和差止之訴是2004年改正新設的。

    📚 **関連条文**: [第2条](../../admin-law/litigation/ch01-general.md#2)・[第3条](../../admin-law/litigation/ch01-general.md#3)

### 問題 2（平成29年度 問16）

行政事件訴訟法第7条が定める「民事訴訟の例による」に関する次の記述のうち、妥当なものはどれか。

1. 行政事件訴訟法第7条は、行政事件訴訟に関しては、民事訴訟法の規定をすべてそのまま適用することを定めたものである。
2. 「民事訴訟の例による」とは、行政事件訴訟法に特別の定めがない事項について、民事訴訟に関する法令の規定に準じて取り扱うことを意味する。
3. 行政事件訴訟法に特別の定めがある事項についても、当事者の合意により民事訴訟法の規定を適用することができる。
4. 「民事訴訟の例による」とされる結果、行政事件訴訟においても私的自治の原則に基づく処分権主義が完全に適用され、裁判所の職権証拠調べは認められない。
5. 「民事訴訟の例による」の規定があるため、行政事件訴訟において弁論主義は一切適用されず、職権探知主義のみが妥当する。

??? success "解答・解説"
    **正解：2**

    1. ✗ 民事訴訟法の規定を「すべてそのまま適用する」のではなく、行政事件訴訟法に**特別の定めがない限り**、民事訴訟に関する法令を準用するという趣旨である。
    2. ○ 第7条は「行政事件訴訟に関し、この法律に定めがない事項については、民事訴訟の例による」と規定しており、補充的に民事訴訟法令の規定を準用するものである。
    3. ✗ 行政事件訴訟法に特別の定めがある事項については、当事者の合意にかかわらず行政事件訴訟法が優先適用される。公益に関わる行政訴訟では当事者の合意による変更は認められない。
    4. ✗ 行政事件訴訟法第24条は**職権証拠調べ**を認めている。これは民事訴訟法の弁論主義の修正であり、行政事件訴訟法の特別の定めに該当する。
    5. ✗ 行政事件訴訟においても基本的に弁論主義が妥当し、職権探知主義は原則として採用されていない。第24条の職権証拠調べは例外的な規定である。

    **中文要點**: 第7條「依民事訴訟之例」意味著行政事件訴訟法沒有特別規定的事項，補充性地準用民事訴訟法。但行訴法有特別規定時（如第24條職權證據調查），優先適用行訴法。

    📚 **関連条文**: [第7条](../../admin-law/litigation/ch01-general.md#7)
