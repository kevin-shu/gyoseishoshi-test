# 第3章 当事者訴訟 — 練習問題

!!! info "本章出題傾向"
    当事者訴訟（第39条〜第41条）に関する問題は、過去10年間で約3〜4回出題。
    重要テーマ：形式的当事者訴訟 vs 実質的当事者訴訟の区別、2004年改正による第4条後段の「公法上の法律関係に関する確認の訴え」の明文化。

---

## 択一式

### 問題 1（令和4年度 問19）

行政事件訴訟法における当事者訴訟に関する次の記述のうち、妥当なものはどれか。

1. 当事者訴訟とは、行政庁の公権力の行使に関する不服の訴訟をいう。
2. 形式的当事者訴訟とは、当事者間の法律関係を確認し又は形成する処分又は裁決に関する訴訟で、法令の規定により当該法律関係の当事者の一方を被告とするものをいい、その典型例として土地収用における損失補償に関する訴訟がある。
3. 実質的当事者訴訟とは、公法上の法律関係に関する確認の訴えその他の公法上の法律関係に関する訴訟をいうが、2004年改正以前から「確認の訴え」が明文で規定されていた。
4. 当事者訴訟においては、取消訴訟と同様に出訴期間の制限があり、処分又は裁決があったことを知った日から6か月以内に提起しなければならない。
5. 当事者訴訟においては、行政事件訴訟法の規定は一切適用されず、民事訴訟法の規定のみが適用される。

??? success "解答・解説"
    **正解：2**

    1. ✗ この記述は**抗告訴訟**（第3条1項）の定義である。当事者訴訟の定義は第4条に規定されている。
    2. ○ 第4条前段のとおり。形式的当事者訴訟の典型例は土地収用法に基づく損失補償額に関する訴訟であり、収用委員会の裁決に不服がある場合に起業者又は被収用者を被告として提起する。
    3. ✗ 実質的当事者訴訟における「公法上の法律関係に関する確認の訴え」の文言は、**2004年改正で追加**されたものである。改正前は「公法上の法律関係に関する訴訟」とのみ規定されていた。
    4. ✗ 実質的当事者訴訟には原則として**出訴期間の制限はない**。形式的当事者訴訟については、個別法に出訴期間の定めがある場合にはそれに従う（第40条）が、6か月という一律の制限はない。
    5. ✗ 当事者訴訟にも行政事件訴訟法の規定が適用される。第41条は取消訴訟に関する規定の一部を準用しており、また第7条により民事訴訟の例によるとされる。

    **中文要點**: 當事者訴訟分為形式的當事者訴訟（第4條前段：處分或裁決所形成的法律關係之當事者一方為被告）和實質的當事者訴訟（第4條後段：公法上法律關係之訴訟）。「公法上法律關係之確認之訴」是2004年改正明文化的。土地收用補償訴訟是形式的當事者訴訟的典型例。

    📚 **関連条文**: [第39条](../../admin-law/litigation/ch03-party-litigation.md#39)

### 問題 2（平成30年度 問19）

2004年（平成16年）の行政事件訴訟法改正による実質的当事者訴訟の活用に関する次の記述のうち、妥当でないものはどれか。

1. 2004年改正により、行政事件訴訟法第4条後段に「公法上の法律関係に関する確認の訴え」が明文で例示された。
2. 実質的当事者訴訟としての確認訴訟は、行政処分の取消訴訟では救済が困難な場合に、公法上の法律関係を直接確認する訴訟として活用が期待されている。
3. 在外日本国民の選挙権に関する最高裁判決（最大判平17.9.14）は、実質的当事者訴訟としての確認訴訟を活用した代表例である。
4. 実質的当事者訴訟においても、取消訴訟と同様に、被告は処分をした行政庁が所属する国又は公共団体となる。
5. 実質的当事者訴訟としての確認訴訟が活用される場面の一つとして、行政立法（省令・通達等）の違法を争う場合がある。

??? success "解答・解説"
    **正解：4**

    1. ○ 2004年改正の重要な改正点の一つ。「確認の訴え」を明示することで、実質的当事者訴訟の活用促進が図られた。
    2. ○ 抗告訴訟では「処分性」が必要であるため、処分性がない行政活動に対しては救済が困難であった。実質的当事者訴訟としての確認訴訟は、処分性の制約を超えて公法上の法律関係を争う手段として活用される。
    3. ○ 最大判平17.9.14（在外邦人選挙権剝奪訴訟）は、選挙権を有する地位の確認を実質的当事者訴訟として認めた画期的判決である。
    4. ✗ 実質的当事者訴訟の被告については、取消訴訟の被告適格（第11条）のような特別規定はなく、**相手方となるべき国、公共団体等が被告**となる。取消訴訟と「同様に」というのは正確ではなく、実質的当事者訴訟は処分の取消しを求める訴えではないため第11条は直接適用されない。もっとも、結果として国又は公共団体が被告となる点では共通するが、根拠規定が異なる。
    5. ○ 通達や省令のように処分性が認められにくい行政活動の違法を争う手段として、実質的当事者訴訟の確認訴訟が活用される場面がある。

    **中文要點**: 2004年改正在第4條後段明文加入「公法上法律關係之確認之訴」，促進實質的當事者訴訟的活用。典型案例是在外邦人選舉權訴訟（最大判平17.9.14）。實質的當事者訴訟不需要「處分性」，因此可以用來爭議行政立法（省令、通達等）的違法。

    📚 **関連条文**: [第39条](../../admin-law/litigation/ch03-party-litigation.md#39)
