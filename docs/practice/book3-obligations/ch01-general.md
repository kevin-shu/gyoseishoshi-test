# 第1章 総則（債権総則）— 練習問題

!!! info "本章出題傾向"
    債権総則は行政書士試験の民法分野で最も出題数が多い重要テーマです。特に**連帯債務の絶対効・相対効**（令和5年問30）、**相殺の要件・禁止事由**（令和5年問31）、**保証の要件**（令和6年問31）、**債権者代位権の要件**（令和3年問32）が近年頻出です。平成29年改正後の新規定が問われることが多く、「改正前後の違い」を意識した学習が必要です。
    重要テーマ：債務不履行・損害賠償・債権者代位権・詐害行為取消権・連帯債務・保証・債権譲渡・相殺

---

## 択一式

### 問題 1（令和3年度 問32）

債権者代位権に関する次の記述のうち、民法の規定に照らし、正しいものはどれか。

1. 債権者は、債務者に属する権利（以下「被代位権利」という。）のうち、債務者の取消権については、債務者に代位して行使することはできない。

2. 債権者は、債務者の相手方に対する債権の期限が到来していれば、自己の債務者に対する債権の期限が到来していなくても、被代位権利を行使することができる。

3. 債権者は、被代位権利を行使する場合において、被代位権利が動産の引渡しを目的とするものであっても、債務者の相手方に対し、その引渡しを自己に対してすることを求めることはできない。

4. 債権者が、被代位権利の行使に係る訴えを提起し、遅滞なく債務者に対し訴訟告知をした場合には、債務者は、被代位権利について、自ら取立てその他の処分をすることはできない。

5. 債権者が、被代位権利を行使した場合であっても、債務者の相手方は、被代位権利について、債務者に対して履行をすることを妨げられない。

??? success "解答・解説"
    **正解：5**

    1. ✗ 取消権（例えば詐欺・強迫による取消権）は一身専属権ではなく、代位行使が可能である（民法423条1項）。行使することができないのは「債務者の一身に専属する権利」（離婚請求権・扶養請求権など）と「差押えを禁じられた権利」に限られる。

    2. ✗ 債権者が被代位権利を行使するには、**自己の債権の期限が到来していること**が原則として必要である（民法423条2項）。相手方の債権（被代位権利）の期限が到来していても、自己の債権の期限が到来していなければ行使できない（保存行為を除く）。

    3. ✗ 民法423条の3により、被代位権利が動産の引渡しを目的とするものである場合、債権者は相手方に対してその引渡しを「自己に対して」することを求めることができる（直接の引渡請求）。

    4. ✗ 民法423条の5により、債権者が被代位権利の行使に係る訴えを提起した場合であっても、**債務者は被代位権利について自ら取立てその他の処分をすることができる**（処分権限は失われない）。

    5. ○ 民法423条の5（後段）により、「この場合においては、相手方も、被代位権利について、債務者に対して履行をすることを妨げられない」と規定されている。債権者が代位権を行使しても、相手方は債務者に対して直接弁済することができる。

    **中文要點**：
    債権者代位権行使後も「債務者の処分権限は失われない」（423条の5）。相手方も債務者に直接弁済できる。これは代位権行使の「処分権限制限効がない」ことを示す。試験では「代位権を行使すると債務者は処分できなくなる」という誤肢に注意。

    📚 **関連条文**: [第423条〜第423条の7](../../civil-code/book3-obligations/ch01-general.md#423)

### 問題 2（令和5年度 問30）

連帯債務者の一人について生じた次のア～オの事由のうち、民法の規定に照らし、他の連帯債務者に対して効力が生じないものの組合せとして、正しいものはどれか。

ア　連帯債務者の一人と債権者との間の混同

イ　連帯債務者の一人がした代物弁済

ウ　連帯債務者の一人が債権者に対して債権を有する場合において、その連帯債務者がした相殺の援用

エ　債権者がした連帯債務者の一人に対する履行の請求

オ　債権者がした連帯債務者の一人に対する債務の免除

1. ア・イ
2. ア・ウ
3. イ・エ
4. ウ・オ
5. エ・オ

??? success "解答・解説"
    **正解：5（エ・オ）**

    ア　✗（他の連帯債務者に効力が生じる＝絶対効）混同（民法440条）は絶対効事由であり、「その連帯債務者は弁済をしたものとみなす」ため、他の連帯債務者の債務も対当額について消滅する。

    イ　✗（絶対効）代物弁済は「弁済と同一の効力」を有するため（民法482条）、弁済と同様に絶対効となる。

    ウ　✗（絶対効）民法439条1項により、相殺を援用した場合、「債権は全ての連帯債務者の利益のために消滅する」と規定されており、絶対効事由である。

    エ　○（他の連帯債務者に効力が生じない＝相対効）民法441条（相対的効力の原則）により、更改・相殺援用・混同の3事由以外は、原則として相対効である。平成29年（2017年）改正前は履行の請求は絶対効であったが、**改正後は相対効に変更された**。

    オ　○（相対効）同様に、債務の免除も改正前は絶対効であったが、**改正後は相対効に変更された**（民法441条）。

    **中文要點**：
    平成29年改正の最重要ポイント：**絶対効は更改・相殺援用・混同の3つのみ**（旧法では履行請求・免除・時效中断も絶対効だったが改正で相対効へ変更）。弁済・代物弁済も絶対効（履行だから）。「免除」「履行請求」「時效完成」は相対効。

    📚 **関連条文**: [第436条〜第445条](../../civil-code/book3-obligations/ch01-general.md#436)

### 問題 3（令和5年度 問31）

相殺に関する次の記述のうち、民法の規定に照らし、誤っているものはどれか。

1. 差押えを受けた債権の第三債務者は、差押え後に取得した債権が差押え前の原因に基づいて生じたものであれば、その第三債務者が、差押え後に他人の債権を取得したときでなければ、その債権による相殺をもって差押債権者に対抗することができる。

2. 時効によって消滅した債権が、その消滅以前に相殺適状にあった場合には、その債権者は、当該債権を自働債権として相殺することができる。

3. 相殺禁止特約のついた債権を譲り受けた者が当該特約について悪意又は重過失である場合には、当該譲渡債権の債務者は、当該特約を譲受人に対抗することができる。

4. 債務者に対する貸金債権の回収が困難なため、債権者がその腹いせに悪意で債務者の物を破損した場合には、債権者は、当該行為による損害賠償債務を受働債権として自己が有する貸金債権と相殺することはできない。

5. 過失によって人の生命又は身体に損害を与えた場合、その加害者は、その被害者に対して有する貸金債権を自働債権として、被害者に対する損害賠償債務と相殺することができる。

??? success "解答・解説"
    **正解：5**

    1. ○ 民法511条2項により、差押え後に取得した債権であっても、差押え前の原因に基づいて生じたものであれば、差押債権者に対して相殺を対抗できる（ただし差押え後に「他人の債権を取得した」場合を除く）。

    2. ○ 民法508条により、時効によって消滅した債権であっても、その消滅以前に相殺適状（相殺に適する状態）にあった場合には、その債権者は相殺をすることができる。相殺への信頼保護のため例外が認められている。

    3. ○ 民法505条2項により、相殺禁止特約（相殺を禁止・制限する旨の意思表示）は、「第三者がこれを知り、又は重大な過失によって知らなかったときに限り、その第三者に対抗することができる」と規定されている。悪意・重過失の譲受人には特約を対抗できる。

    4. ○ 民法509条1号により、「悪意による不法行為に基づく損害賠償の債務」を受働債権（相殺を受ける側の債権）として相殺することはできない。腹いせ目的の悪意不法行為は「悪意による不法行為」に当たるため、その加害者は損害賠償債務を受働債権とした相殺はできない。

    5. ✗ 民法509条2号により、「人の生命又は身体の侵害による損害賠償の債務」の債務者は、**故意・過失を問わず**相殺をもって債権者に対抗することができない。過失による身体侵害でも相殺禁止が適用されるため、選択肢5は誤り。

    **中文要點**：
    相殺禁止（509条）のポイント：①悪意不法行為の損害賠償（1号）②生命・身体侵害の損害賠償（2号、故意・過失不問）。5肢は「過失でも相殺禁止」という改正ポイントを問う。旧法は「不法行為に基づく損害賠償」のみだったが、改正で生命・身体侵害は過失も含まれるよう拡大された。

    📚 **関連条文**: [第505条〜第511条](../../civil-code/book3-obligations/ch01-general.md#505)

### 問題 4（令和6年度 問31）

AはBから金銭を借り受け、CがAの同貸金債務を保証した。次の記述のうち、民法の規定に照らし、誤っているものはどれか。

1. AがBに対し保証人を立てる義務を負う場合において、BがCを指名したときは、Cが弁済をする資力を有しなくなったときでも、Bは、Aに対し、Cに代えて資力を有する保証人を立てることを請求することはできない。

2. AがBに対し保証人を立てる義務を負う場合において、BがCを指名するときは、Cは、行為能力者でなければならない。

3. BのAに対する履行の請求その他の事由による時効の完成猶予および更新は、Cに対しても、その効力を生ずる。

4. Cの保証債務は、Aの債務に関する利息、違約金、損害賠償その他その債務に従たるすべてのものを包含する。

5. Cは、その保証債務についてのみ、違約金または損害賠償の額を約定することができる。

??? success "解答・解説"
    **正解：2**

    1. ○ 民法450条3項により、「前二項の規定は、**債権者が保証人を指名した場合には、適用しない**」と定められている。よって、債権者Bが保証人Cを指名した場合、450条2項（資力喪失時の代替請求）も適用されず、Bは新たな保証人を請求できない。

    2. ✗ 民法450条1項は、「保証人は行為能力者でなければならない」と規定しているが、同条3項により「**債権者が保証人を指名した場合には、第1項・第2項を適用しない**」。よって、BがCを指名した場合、Cが行為能力者である必要はない。

    3. ○ 民法457条1項により、「主たる債務者に対する履行の請求その他の事由による時効の完成猶予及び更新は、保証人に対しても、その効力を生ずる」と規定されている（主たる債務者への事由→保証人にも及ぶ）。

    4. ○ 民法447条1項により、「保証債務は、主たる債務に関する利息、違約金、損害賠償その他その債務に従たるすべてのものを包含する」と規定されている。

    5. ○ 民法447条2項により、「保証人は、その保証債務についてのみ、違約金又は損害賠償の額を約定することができる」と規定されている（保証債務の加重は禁止されているが、保証人自身の違約金を別途定めることは可能）。

    **中文要點**：
    保証人の要件（450条）は債務者が義務的に立てる場合の規定。但し「**債権者が指名した場合は適用除外**」（450条3項）。これが本問の核心。また、主たる債務者への請求→保証人にも時效猶予・更新の効力が及ぶ（457条1項）という附従性も重要。

    📚 **関連条文**: [第446条〜第465条の10](../../civil-code/book3-obligations/ch01-general.md#446)

### 問題 5（[出題例]）

詐害行為取消権に関する次の記述のうち、民法の規定に照らし、誤っているものはどれか。

1. 債権者は、債務者が債権者を害することを知ってした財産権を目的とする行為について、取消しを裁判所に請求することができるが、受益者がその行為の時において債権者を害することを知らなかったときは、この限りでない。

2. 詐害行為取消請求の被告は、受益者に対する請求の場合は受益者、転得者に対する請求の場合はその転得者とする。

3. 詐害行為取消請求を認容する確定判決は、申請した債権者のみならず、債務者及びその全ての債権者に対しても効力を有する。

4. 債権者は、その債権が詐害行為の前の原因に基づいて生じたものである場合に限り、詐害行為取消請求をすることができる。

5. 詐害行為取消請求に係る訴えは、債務者が債権者を害することを知って行為をしたことを債権者が知った時から5年を経過したときは提起することができない。

??? success "解答・解説"
    **正解：5**

    1. ○ 民法424条1項の規定のとおり。詐害行為取消権の要件：①債務者が債権者を害することを知っていた（主観的要件）②受益者も悪意であること（ただし無償行為は受益者の善悪を問わない場合もある）③財産権を目的とする行為であること。

    2. ○ 民法424条の7第1項の規定のとおり。詐害行為取消請求訴訟の被告は、受益者への請求→受益者、転得者への請求→転得者となる（債務者は被告にはならないが、訴訟告知をしなければならない）。

    3. ○ 民法425条の規定のとおり。「詐害行為取消請求を認容する確定判決は、債務者及びその全ての債権者に対してもその効力を有する」。申請した債権者だけでなく全債権者に及ぶ（これは債権者代位権の判決と異なる点）。

    4. ○ 民法424条3項の規定のとおり。「その債権が（詐害）行為の**前**の原因に基づいて生じたものである場合に限り」詐害行為取消請求ができる。詐害行為後に生じた債権者は請求できない。

    5. ✗ 民法426条により、詐害行為取消請求の訴えの期間制限は、「債務者が債権者を害することを知って行為をしたことを債権者が知った時から**2年**」（主観的期間）と「行為の時から**10年**」（客観的期間）である。5年ではなく2年が正しい。

    **中文要點**：
    詐害行為取消請求の時效：主観2年（知った時から）＋客観10年（行為時から）。よく「2年か5年か」を問う問題が出る。また、取消判決の効力は「全債権者」に及ぶ点（425条）を確認。財産権を目的としない行為（離婚、相続放棄など）には適用不可。

    📚 **関連条文**: [第424条〜第426条](../../civil-code/book3-obligations/ch01-general.md#424)

### 問題 6（[出題例]）

債権の譲渡に関する次の記述のうち、民法の規定に照らし、正しいものはどれか。

1. 当事者が債権の譲渡を禁止し、又は制限する旨の意思表示（譲渡制限の意思表示）をした場合、その意思表示に悪意の譲受人に対しては、債権譲渡は無効となる。

2. 債権の譲渡を債務者以外の第三者に対抗するためには、確定日付のある証書による通知又は承諾が必要である。

3. 債権の譲渡は、その意思表示の時に債権が現に発生していることを必要とする。

4. 債務者は、対抗要件具備時より後に取得した譲渡人に対する債権による相殺をもって、常に譲受人に対抗することができる。

5. 譲渡制限の意思表示がされた預貯金債権については、それを知り、または重大な過失によって知らなかった譲受人に対しても、債権譲渡は有効である。

??? success "解答・解説"
    **正解：2**

    1. ✗ 民法466条2項（改正後）により、譲渡制限の意思表示をした場合であっても、「債権の譲渡は、その効力を妨げられない」（有効）。ただし3項により、悪意・重過失の譲受人に対しては、債務者は履行を拒絶でき、かつ譲渡人への弁済で対抗できる。譲渡自体は有効であることに注意。

    2. ○ 民法467条2項の規定のとおり。「通知又は承諾は、確定日付のある証書によってしなければ、債務者以外の第三者に対抗することができない」。確定日付のある証書の例：公正証書、内容証明郵便など。

    3. ✗ 民法466条の6第1項により、「債権の譲渡は、その意思表示の時に債権が現に発生していることを要しない」（将来債権の譲渡も可能）。

    4. ✗ 民法469条1項により、対抗要件具備時「**より前**に取得した」債権による相殺は対抗できる。対抗要件具備時より後に取得した債権による相殺は原則として対抗できない（ただし469条2項各号の例外あり）。

    5. ✗ 民法466条の5第1項により、「**預貯金債権**についての譲渡制限の意思表示は、（466条2項の規定にかかわらず）悪意・重過失の譲受人に対抗することができる」。預貯金債権については特則があり、通常の債権とは異なり、悪意・重過失の第三者への対抗が認められる。

    **中文要點**：
    平成29年改正の重要ポイント：①譲渡制限特約があっても債権譲渡は「有効」（466条2項）→ただし悪意・重過失の第三者へは対抗可（3項）。②**預貯金債権は例外**：善意の譲受人にも対抗できる（466条の5）。③第三者対抗要件は「確定日付のある証書」による通知・承諾（467条2項）。

    📚 **関連条文**: [第466条〜第469条](../../civil-code/book3-obligations/ch01-general.md#466)

### 問題 7（[出題例]）

債務不履行に基づく損害賠償に関する次の記述のうち、民法の規定に照らし、誤っているものはどれか。

1. 確定期限のある債務の債務者は、その期限の到来した時から履行遅滞の責任を負い、別途、債権者からの催告は不要である。

2. 金銭の給付を目的とする債務の不履行による損害賠償の額は、債務者が遅滞の責任を負った最初の時点における法定利率によって定めるが、債権者は実際の損害額を証明することを要しない。

3. 債務の不履行が契約その他の債務の発生原因及び取引上の社会通念に照らして債務者の責めに帰することができない事由によるものであるときは、債権者は損害賠償を請求することができない。

4. 当事者が損害賠償の額を予定した場合、債権者はその予定額を超える損害が生じたことを証明すれば、予定額を超える損害の賠償を請求することができる。

5. 損害賠償額の予定がある場合においても、債権者は履行の請求や解除権の行使を行うことができる。

??? success "解答・解説"
    **正解：4**

    1. ○ 民法412条1項の規定のとおり。確定期限のある債務は「期限の到来した時から」遅滞の責任を負う（催告不要）。不確定期限や期限なし債務は催告が必要。

    2. ○ 民法419条1項・2項の規定のとおり。金銭債務の不履行については、①損害賠償額は法定利率（または約定利率が高い場合は約定利率）で定め、②債権者は損害の証明が不要。また③不可抗力による抗弁も不可（419条3項）。

    3. ○ 民法415条1項ただし書の規定のとおり（平成29年改正）。「債務の不履行が契約その他の債務の発生原因及び取引上の社会通念に照らして債務者の責めに帰することができない事由によるものであるとき」は免責される。

    4. ✗ 民法420条1項により、当事者が損害賠償の額を予定した場合（賠償額の予定）、その予定額が損害賠償額となる。**実際の損害がそれを超えても、予定額を超えて請求することはできない**（また、実際の損害が少なくても、予定額未満になることもない）。

    5. ○ 民法420条2項の規定のとおり。「賠償額の予定は、履行の請求又は解除権の行使を妨げない」。損害賠償額を予定しても、履行請求・解除権行使は別途行使できる。

    **中文要點**：
    損害賠償の額の予定（420条）：予定額が上限・下限両方の拘束力を持つ→実際損害が多くても少なくても予定額通り。当事者の予測可能性を保障する制度。違約金は「賠償額の予定と推定」（3項）。注意：賠償額を予定しても履行請求・解除権の行使は妨げられない。
