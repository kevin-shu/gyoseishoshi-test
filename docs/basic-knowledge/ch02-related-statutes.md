# 行政書士法等関連諸法令

> R6（2024年）から新設された分野。条文知識を直接問う傾向が強く、条文の素読が最も効率的な対策。目標正答数：**2問中1～2問**。

---

## 行政書士法【最重要】

### 業務範囲

#### 知識點整理

| 業務分類 | 内容 | 根拠条文 |
|----------|------|----------|
| **独占業務** | 官公署に提出する書類の作成 | 1条の2第1項 |
| **独占業務** | 官公署への書類提出手続の代理 | 1条の3第1号 |
| **独占業務** | 契約その他に関する書類の作成（権利義務・事実証明） | 1条の2第1項 |
| **非独占業務** | 行政書士が作成できる書類に関する相談業務 | 1条の3第4号 |
| **特定業務** | 行政手続における聴聞・弁明の代理（特定行政書士） | 1条の3第2号 |

**独占業務の例外**: 弁護士、弁理士、税理士、社会保険労務士等が各法律で認められている業務は、行政書士の独占業務から除外される。

#### 中文翻譯

| 業務分類 | 內容 |
|----------|------|
| **獨佔業務** | 製作提交官公署的文件 |
| **獨佔業務** | 代理向官公署提交文件的程序 |
| **獨佔業務** | 製作契約及其他關於權利義務・事實證明的文件 |
| **非獨佔業務** | 關於行政書士可製作之文件的諮詢業務 |
| **特定業務** | 行政程序中聽證・辯明的代理（限特定行政書士） |

#### 出題分析

獨佔業務與非獨佔業務的區別是最核心考點。注意「特定行政書士」才能代理聽證和辯明程序，一般行政書士不可。

### 登録制度

#### 知識點整理

| 項目 | 内容 |
|------|------|
| **登録先** | 日本行政書士会連合会が備える行政書士名簿 |
| **登録申請** | 事務所所在地の都道府県行政書士会を経由して申請 |
| **資格要件** | 行政書士試験合格者、弁護士・弁理士・公認会計士・税理士、公務員特認（17年以上） |
| **欠格事由** | 成年被後見人・被保佐人、禁錮以上の刑（刑の執行終了後3年未満）、懲戒による業務禁止（処分の日から3年未満）等 |
| **登録拒否** | 行政書士の信用を害するおそれがある場合等 |

#### 中文翻譯

| 項目 | 內容 |
|------|------|
| **登錄處** | 日本行政書士會聯合會的行政書士名簿 |
| **申請途徑** | 透過事務所所在地的都道府縣行政書士會提出 |
| **資格要件** | 考試合格者、律師等相關資格者、公務員特認（17年以上） |
| **欠格事由** | 成年被後見人・被保佐人、受禁錮以上刑罰（刑期結束未滿3年）、受懲戒業務禁止（處分日起未滿3年）等 |

#### 出題分析

登錄是向「聯合會」申請（非都道府縣知事）、經由「都道府縣行政書士會」，這個申請流程是重要考點。欠格事由中的「3年」期限也常出題。

### 義務・禁止事項

#### 知識點整理

| 義務 | 内容 | 根拠 |
|------|------|------|
| **守秘義務** | 正当な理由なく業務上知った秘密を漏らしてはならない（退職後も） | 12条 |
| **報酬額の掲示** | 事務所の見やすい場所に報酬額を掲示 | 10条の2 |
| **帳簿の備付け** | 業務に関する帳簿を備え、事件の名称等を記載 | 9条 |
| **帳簿の保存** | 関係書類とともに事件終了後**2年間**保存 | 9条 |
| **依頼に応ずる義務** | 正当な事由がなければ依頼を拒めない | 11条 |
| **会則の遵守** | 所属行政書士会・連合会の会則を守る | 13条の22 |

#### 中文翻譯

| 義務 | 內容 |
|------|------|
| **保密義務** | 無正當理由不得洩漏業務上知悉之秘密（退職後亦同） |
| **報酬額公示** | 在事務所顯眼處揭示報酬額 |
| **帳簿備置** | 備置業務帳簿，記載案件名稱等 |
| **帳簿保存** | 案件結束後保存**2年** |
| **接受委託義務** | 無正當理由不得拒絕委託 |

#### 出題分析

保密義務「退職後も」（離職後仍有效）是常考陷阱。帳簿保存期間「2年」也是數字記憶考點。

### 懲戒制度

#### 知識點整理

| 処分 | 内容 | 処分権者 |
|------|------|----------|
| **戒告** | 注意・警告 | 都道府県知事 |
| **業務停止** | 2年以内の業務停止 | 都道府県知事 |
| **業務禁止** | 行政書士の業務を禁止（最も重い処分） | 都道府県知事 |

**手続**: 都道府県知事は処分前に弁明の機会を付与しなければならない。聴聞は業務停止・業務禁止の場合に実施。

#### 中文翻譯

| 處分 | 內容 | 處分權者 |
|------|------|----------|
| **戒告** | 注意・警告 | 都道府縣知事 |
| **業務停止** | 2年以內的業務停止 | 都道府縣知事 |
| **業務禁止** | 禁止從事行政書士業務（最重處分） | 都道府縣知事 |

#### 出題分析

處分權者是「都道府縣知事」而非行政書士會，這是重要區別。三種處分的輕重順序（戒告→業務停止→業務禁止）及業務停止的上限「2年」需記憶。

### 行政書士会と連合会

#### 知識點整理

| 項目 | 行政書士会 | 日本行政書士会連合会 |
|------|-----------|-------------------|
| **設立単位** | 都道府県ごとに1つ | 全国で1つ |
| **法人格** | 法人 | 法人 |
| **入会義務** | 登録した行政書士は当然入会 | 行政書士会が当然入会 |
| **主な業務** | 会員の指導・連絡・監督 | 行政書士名簿の登録事務、会員の品位保持 |

#### 中文翻譯

| 項目 | 行政書士會 | 日本行政書士會聯合會 |
|------|-----------|-------------------|
| **設立單位** | 每都道府縣1個 | 全國1個 |
| **入會義務** | 登錄即當然入會 | 各行政書士會當然入會 |
| **主要業務** | 會員的指導・聯絡・監督 | 名簿登錄事務、維持會員品位 |

#### 出題分析

「強制入會」制度是核心考點。行政書士登錄後自動成為所在地行政書士會的會員，無須另外申請入會。

## 戸籍法

### 届出の種類と期間

#### 知識點整理

| 届出 | 届出義務者 | 届出期間 | 届出先 |
|------|-----------|----------|--------|
| **出生届** | 父または母 | 出生の日から**14日以内** | 本籍地・出生地・届出人の所在地 |
| **死亡届** | 同居の親族等 | 死亡の事実を知った日から**7日以内** | 死亡地・本籍地・届出人の所在地 |
| **婚姻届** | 当事者双方 | 届出により効力発生（届出婚主義） | 夫婦の本籍地・所在地 |
| **離婚届** | 当事者双方（協議離婚） | 届出により効力発生 | 本籍地・所在地 |
| **転籍届** | 戸籍の筆頭者とその配偶者 | 制限なし | 転籍地 |

#### 中文翻譯

| 屆出 | 義務人 | 期限 | 受理處 |
|------|--------|------|--------|
| **出生屆** | 父或母 | 出生之日起**14日內** | 本籍地・出生地・屆出人所在地 |
| **死亡屆** | 同居親屬等 | 知悉死亡事實起**7日內** | 死亡地・本籍地・屆出人所在地 |
| **婚姻屆** | 當事人雙方 | 屆出即生效（屆出婚主義） | 夫婦本籍地・所在地 |
| **離婚屆** | 當事人雙方（協議離婚） | 屆出即生效 | 本籍地・所在地 |

#### 出題分析

出生屆「14日」和死亡屆「7日」的期限差異是數字記憶重點。婚姻屆是「屆出婚主義」（不是事實婚）也是考點。R7出題了戶籍法。

---

### 戸籍謄本の請求

#### 知識點整理

| 請求権者 | 請求条件 |
|----------|----------|
| **本人・配偶者・直系尊卑属** | 理由を問わず請求可能 |
| **弁護士・司法書士・行政書士等** | 職務上の請求（受任事件に必要な場合） |
| **第三者** | 正当な理由がある場合に限り請求可能 |
| **国・地方公共団体** | 法令の定める事務に必要な場合 |

#### 中文翻譯

本人、配偶者、直系親屬無條件可請求。律師、行政書士等專門職業者可基於職務需要請求。第三人僅在有正當理由時可請求。

#### 出題分析

請求權者的範圍和限制條件是核心考點。注意「兄弟姐妹」不在本人・直系親屬的範圍內。

## 住民基本台帳法

### 届出義務

#### 知識點整理

| 届出 | 届出期間 | 届出先 |
|------|----------|--------|
| **転入届** | 転入の日から**14日以内** | 転入先の市町村長 |
| **転出届** | あらかじめ届出 | 転出元の市町村長 |
| **転居届** | 転居の日から**14日以内** | 市町村長 |
| **世帯変更届** | 変更の日から**14日以内** | 市町村長 |

#### 中文翻譯

| 屆出 | 期限 | 受理處 |
|------|------|--------|
| **遷入屆** | 遷入之日起**14日內** | 遷入地市町村長 |
| **遷出屆** | **事前**屆出 | 遷出地市町村長 |
| **遷居屆** | 遷居之日起**14日內** | 市町村長 |

#### 出題分析

轉入屆「14日以內」vs. 轉出屆「事前」的差異是核心考點。R6出題了住民基本台帳法。

### 住民票の記載事項と交付

#### 知識點整理

**記載事項**：氏名、出生の年月日、男女の別、世帯主の氏名・世帯主との続柄、戸籍の表示、住民となった年月日、届出の年月日、住所（転居した場合は従前の住所）

**住民票の写しの交付請求**：

| 請求者 | 条件 |
|--------|------|
| **本人・同一世帯の者** | 理由を問わず請求可能 |
| **国・地方公共団体** | 法令の定める事務に必要な場合 |
| **第三者** | 正当な理由がある場合（債権回収等） |
| **弁護士・行政書士等** | 受任事件に必要な場合 |

#### 中文翻譯

住民票記載事項包含：姓名、出生年月日、性別、戶長姓名與續柄、戶籍表示、住所等。

交付請求：本人和同一世帯成員可無條件請求。第三人需有正當理由。

#### 出題分析

住民票的記載事項和請求權者範圍是考試重點。注意住民票的請求權者是「同一世帯」（非「親屬」），與戶籍謄本的請求權者（直系親屬）不同。

### マイナンバーとの関係

#### 知識點整理

| 項目 | 住民票コード | マイナンバー（個人番号） |
|------|-------------|----------------------|
| **桁数** | 11桁 | 12桁 |
| **利用範囲** | 住民基本台帳ネットワーク内の事務 | 税・社会保障・災害対策の3分野 |
| **変更可能性** | 本人の請求により変更可能 | 原則変更不可（漏洩等の場合を除く） |
| **住民票への記載** | 記載あり（ただし原則非開示） | 記載あり（ただし原則非開示） |

#### 中文翻譯

住民票代碼（11碼）與個人編號（12碼）的區別：住民票代碼用於住基網路內部事務，個人編號限用於稅務・社會保障・災害對策三領域。個人編號原則上不可變更。

#### 出題分析

住民票代碼與個人編號的位數差異（11桁 vs. 12桁）、利用範圍的限定是考試重點。

## 本章の学習戦略まとめ

!!! info "学習のポイント"
    関連諸法令はR6新設のため過去問が少ないが、条文数が限られるため**条文の素読が最も効率的**。

    1. **行政書士法**: 業務範囲（独占 vs. 非独占）、登録制度、懲戒制度を最優先
    2. **戸籍法**: 届出の種類と期間の数字を正確に記憶
    3. **住民基本台帳法**: 転入届「14日以内」vs. 転出届「事前」の対比
    4. 平成17年以前の過去問に行政書士法の出題があり参考になる
