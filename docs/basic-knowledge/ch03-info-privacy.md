# 情報通信・個人情報保護

> 対策のしやすさと得点効率が最も高い分野。法令科目と同じ要領で条文を学習でき、配分時間あたりの得点効率が最も高い。目標正答数：**3～4問中3問**。

---

## 個人情報保護法【最重要・毎年出題】

### 目的と定義

#### 知識點整理

**目的（1条）**: 個人情報の適正かつ効果的な活用が新たな産業の創出並びに活力ある経済社会及び豊かな国民生活の実現に資するものであることその他の個人情報の有用性に配慮しつつ、**個人の権利利益を保護**することを目的とする。

**重要な定義**:

| 用語 | 定義 | 具体例 |
|------|------|--------|
| **個人情報** | 生存する個人に関する情報で、特定の個人を識別できるもの | 氏名、住所、顔写真 |
| **個人識別符号** | 身体の特徴をデータ化したもの、公的番号 | 指紋データ、マイナンバー、旅券番号 |
| **要配慮個人情報** | 不当な差別・偏見が生じうる情報 | 人種、信条、病歴、犯罪歴、障害 |
| **匿名加工情報** | 個人を識別できないように加工した情報 | 統計データ化した購買履歴 |
| **仮名加工情報** | 他の情報と照合しない限り個人を識別できないように加工 | 氏名を番号に置換したデータ |
| **個人情報取扱事業者** | 個人情報データベース等を事業の用に供している者 | 企業、団体、個人事業主 |

#### 中文翻譯

| 用語 | 定義 |
|------|------|
| **個人資訊** | 與生存之個人相關、可識別特定個人之資訊 |
| **個人識別符號** | 將身體特徵數據化者、公共編號 |
| **需特別照顧之個人資訊** | 可能產生不當歧視或偏見之資訊 |
| **匿名加工資訊** | 加工至無法識別個人之資訊 |
| **假名加工資訊** | 不與其他資訊比對即無法識別個人之資訊 |
| **個人資訊處理業者** | 將個人資訊資料庫用於事業之人 |

#### 出題分析

各定義的區別是最頻出考點。特別是「匿名加工情報」vs.「仮名加工情報」的差異、「要配慮個人情報」的具體例。注意：個人情報限於「生存する個人」，死者不含在內。

### 個人情報取扱事業者の義務

#### 知識點整理

| 義務 | 内容 | 条文 |
|------|------|------|
| **利用目的の特定** | 個人情報の利用目的をできる限り特定 | 17条 |
| **利用目的の通知・公表** | 取得時に利用目的を本人に通知または公表 | 21条 |
| **利用目的による制限** | 利用目的の達成に必要な範囲を超えて取り扱ってはならない | 18条 |
| **適正な取得** | 偽りその他不正な手段により取得してはならない | 20条 |
| **安全管理措置** | 漏洩・滅失・毀損の防止その他の安全管理 | 23条 |
| **従業者の監督** | 従業者に対する必要かつ適切な監督 | 24条 |
| **委託先の監督** | 委託先に対する必要かつ適切な監督 | 25条 |
| **第三者提供の制限** | 本人の同意なく第三者に提供してはならない（原則） | 27条 |
| **保有個人データの開示** | 本人からの請求に応じて保有個人データを開示 | 33条 |
| **漏洩等の報告** | 個人情報保護委員会への報告＋本人への通知 | 26条 |

#### 中文翻譯

核心義務體系：特定利用目的 → 通知本人 → 範圍內利用 → 安全管理 → 限制第三方提供 → 接受開示請求 → 洩漏時報告。

#### 出題分析

第三方提供的限制（原則需本人同意）及其例外是最重要考點。保有個人資料的開示請求權也常出題。

---

### 第三者提供の例外（オプトアウト・法定例外）

#### 知識點整理

**同意不要の場合（法定例外）**:

1. 法令に基づく場合
2. 人の生命・身体・財産の保護に必要で本人の同意取得が困難
3. 公衆衛生の向上・児童の健全育成に特に必要で本人の同意取得が困難
4. 国の機関等の法令事務への協力

**オプトアウト（27条2項）**: 本人の求めにより提供を停止することを条件に、あらかじめ個人情報保護委員会に届け出た上で、本人の同意なく第三者提供が可能。ただし、**要配慮個人情報はオプトアウトによる第三者提供は不可**。

#### 中文翻譯

不需同意即可提供第三方的法定例外：依法令規定、保護生命身體財產且取得同意困難、公共衛生・兒童健全發育所特別需要、配合國家機關法定事務。

Opt-out制度：事先向個人資訊保護委員會報備，以「本人可隨時要求停止」為條件，無需本人同意即可提供第三方。但**需特別照顧之個人資訊不可**使用opt-out方式。

#### 出題分析

法定例外的4種情形和opt-out的要件是高頻考點。「要配慮個人情報不可opt-out」是重要限制。

---

### 適用除外

#### 知識點整理

以下の者が学術研究等の目的で個人情報を取り扱う場合、義務規定の一部が適用除外される：

| 対象 | 備考 |
|------|------|
| **報道機関** | 報道の用に供する目的 |
| **著述を業とする者** | 著述の用に供する目的 |
| **大学その他の学術研究機関** | 学術研究の用に供する目的 |
| **宗教団体** | 宗教活動の用に供する目的 |
| **政治団体** | 政治活動の用に供する目的 |

#### 中文翻譯

報道機關（報導目的）、著作者（著述目的）、學術研究機關（學術目的）、宗教團體（宗教活動目的）、政治團體（政治活動目的）的部分義務得以免除。

#### 出題分析

適用除外的5類主體和各自的目的限定是出題重點。注意是「適用除外」而非「完全不受規範」，仍有一定的安全管理義務。

### 個人情報保護委員会

#### 知識點整理

| 項目 | 内容 |
|------|------|
| **性質** | 内閣府の外局に設置された**独立行政委員会**（合議制） |
| **構成** | 委員長1名＋委員8名（国会同意人事） |
| **権限** | 報告徴収・立入検査、指導・助言、勧告、命令 |
| **特徴** | 事業者に対する一元的な監督機関 |

#### 中文翻譯

個人資訊保護委員會是設置於內閣府之**獨立行政委員會**，由委員長1名及委員8名組成（需國會同意）。對事業者擁有報告徵收、立入檢查、指導、勸告、命令等權限。

#### 出題分析

「獨立行政委員會」的性質和「一元的監督機關」的定位是考試重點。

## 情報公開法（行政機関情報公開法）

### 開示請求権

#### 知識點整理

| 項目 | 内容 |
|------|------|
| **請求権者** | **何人も**（国籍・住所を問わない） |
| **対象** | 行政機関の職員が作成・取得した行政文書 |
| **開示決定の期限** | 請求があった日から**30日以内** |
| **延長** | やむを得ない理由がある場合、**30日以内**の延長可 |
| **手数料** | 実費の範囲内で政令で定める額 |

#### 中文翻譯

任何人（不問國籍・住所）皆可請求公開行政文書。行政機關應於請求之日起30日內做出開示決定，有不得已之理由時得延長30日。

#### 出題分析

「何人も」（任何人皆可請求）是最重要考點。注意是「行政文書」（行政機關職員所製作・取得之文書），不包含個人的筆記等。

---

### 不開示情報

#### 知識點整理

| 不開示情報 | 内容 |
|-----------|------|
| **個人情報** | 特定の個人を識別できる情報（公務員の職務遂行情報は原則開示） |
| **法人情報** | 法人の正当な利益を害するおそれがある情報 |
| **国の安全等に関する情報** | 国の安全が害されるおそれがある情報 |
| **公共の安全に関する情報** | 犯罪の予防・捜査等に支障を及ぼすおそれがある情報 |
| **審議検討情報** | 率直な意見交換が不当に損なわれるおそれがある情報 |
| **事務事業情報** | 行政機関の事務・事業の適正な遂行に支障を及ぼすおそれがある情報 |

#### 中文翻譯

不得公開的資訊6類：個人資訊、法人資訊、國家安全相關、公共安全相關、審議檢討資訊、事務事業資訊。但公務員的職務執行資訊原則上應公開。

#### 出題分析

6類不開示資訊的內容和「公務員的職務遂行情報は原則開示」是核心考點。部分開示（除去不開示部分後開示其餘部分）的制度也常出題。

## 公文書管理法

### 核心制度

#### 知識點整理

| 項目 | 内容 |
|------|------|
| **目的** | 行政文書等の適正な管理、歴史公文書等の適切な保存・利用 |
| **行政文書の作成義務** | 意思決定に至る過程・実績を文書で作成 |
| **保存期間** | 1年未満～30年（文書管理簿に記載） |
| **保存期間満了後** | 国立公文書館等への移管 or 廃棄（廃棄は内閣総理大臣の同意必要） |
| **特定歴史公文書等** | 国立公文書館等が保存、利用請求に応じて一般利用に供する |

#### 中文翻譯

公文書管理法規定行政文書的適正管理體制。保存期間屆滿後，重要文書移管至國立公文書館等保存，廢棄則需**內閣總理大臣同意**。

#### 出題分析

「廢棄需內閣總理大臣同意」是最常出的考點。保存期間屆滿後的「移管 or 廢棄」的處理流程也需掌握。

## プロバイダ責任制限法

### 核心制度

#### 知識點整理

| 項目 | 内容 |
|------|------|
| **正式名称** | 特定電気通信役務提供者の損害賠償責任の制限及び発信者情報の開示に関する法律 |
| **目的** | プロバイダ等の損害賠償責任の制限、発信者情報の開示 |
| **免責要件** | 権利侵害を知らなかった場合、知ることが相当と認められない場合 |
| **発信者情報開示** | 権利侵害が明白な場合、被害者はプロバイダに発信者情報の開示を請求可能 |
| **発信者情報開示命令** | 2022年改正により新設、裁判所への非訟手続 |

#### 中文翻譯

此法規定網路服務提供者（ISP等）的損害賠償責任限制及發信者資訊開示制度。被害者在權利侵害明確時，可請求ISP開示發信者資訊。2022年改正新設了開示命令制度（透過法院的非訟程序）。

#### 出題分析

ISP的免責要件和發信者資訊開示的條件是考試重點。2022年改正的「開示命令」制度也可能出題。

## IT用語・情報通信

### 重要用語

#### 知識點整理

| 用語 | 内容 |
|------|------|
| **生成AI** | テキスト・画像等を自動生成する人工知能。ハルシネーション（幻覚）問題、著作権問題が課題 |
| **デジタル庁** | 2021年設置、デジタル社会形成の司令塔。内閣直属の組織 |
| **マイナンバー** | 12桁の個人番号。税・社会保障・災害対策の3分野で利用 |
| **マイナンバーカード** | ICチップ搭載の本人確認書類。電子証明書を格納 |
| **電子署名** | 電磁的方式による署名。電子署名法により手書き署名と同等の法的効力 |
| **不正アクセス禁止法** | 他人のID・パスワードを無断使用する行為等を禁止 |
| **フィッシング** | 偽サイト等で個人情報を騙し取る行為 |
| **ランサムウェア** | データを暗号化し身代金を要求するマルウェア |
| **IoT** | Internet of Things。モノのインターネット |
| **ブロックチェーン** | 分散型台帳技術。暗号資産（仮想通貨）の基盤技術 |
| **クラウドコンピューティング** | インターネット経由でサーバー・ストレージ等を利用 |
| **5G** | 第5世代移動通信システム。高速・大容量・低遅延・多数同時接続 |

#### 中文翻譯

| 用語 | 內容 |
|------|------|
| **生成式AI** | 自動生成文字・圖像的人工智慧。有幻覺（hallucination）問題 |
| **數位廳** | 2021年設置、數位社會形成的司令塔。直屬內閣 |
| **個人編號** | 12碼。限用於稅務・社會保障・災害對策 |
| **電子簽章** | 電磁方式的簽名，具與手寫簽名同等法律效力 |
| **不正存取禁止法** | 禁止未經授權使用他人帳號密碼等行為 |
| **勒索軟體** | 加密資料並要求贖金的惡意軟體 |

#### 出題分析

生成AI（ChatGPT等）相關的基本概念是近年熱門考點。數位廳「直屬內閣」的定位、個人編號的利用範圍限定（3分野）也是重要考點。IT用語不需要深入技術細節，掌握基本概念和用途即可。

## 本章の学習戦略まとめ

!!! info "学習のポイント"
    情報通信・個人情報保護は**配分時間あたりの得点効率が最も高い**分野。法令科目と同じ要領で対策可能。

    1. **個人情報保護法**: 目的→定義→義務→適用除外の順で体系的に整理
    2. **情報公開法**: 「何人も」の開示請求権、6種の不開示情報を暗記
    3. **公文書管理法**: 廃棄時の内閣総理大臣の同意が最重要ポイント
    4. **IT用語**: 総務省・デジタル庁の用語集を活用、深入りは不要
    5. **時間配分**: 基礎知識科目全体の学習時間の**50%以上**をこの分野に投入すべき
