# 文章理解

> 3問固定出題で正答率が非常に高い（R6：93～96%）。確実に全問正解することで足切り回避の基盤となる。目標正答数：**3問中3問（全問正解必須）**。

---

## 出題形式と特徴

### 三種題型

#### 知識點整理

| 形式 | 問題数 | 内容 | 難易度 |
|------|--------|------|--------|
| **要旨把握** | 1問 | 筆者の主張・結論を正しく選択する | ★★☆ |
| **空欄補充** | 1問 | 文中の空欄に入る接続詞・フレーズを選択 | ★★☆ |
| **文章並替** | 1問 | バラバラになった段落の正しい順序を選択 | ★★★ |

#### 中文翻譯

| 題型 | 題數 | 內容 | 難度 |
|------|------|------|------|
| **主旨掌握** | 1題 | 選出作者的主張・結論 | ★★☆ |
| **填空補充** | 1題 | 選出空格中應填入的接續詞・片語 | ★★☆ |
| **段落排序** | 1題 | 選出打亂段落的正確順序 | ★★★ |

#### 出題分析

三種題型每年各出1題，合計3題12分。歷年正答率極高，是基礎知識科目中最穩定的得分源。只要掌握解題技巧並做足練習，全問正解完全可能。

## 要旨把握の解法

### 解題步驟

#### 知識點整理

**Step 1**: 選択肢を先に読み、論点のキーワードを把握する

**Step 2**: 本文を読みながら、以下のマーカーに注目する

| マーカー | 意味 | 例 |
|----------|------|-----|
| **逆接の接続詞** | 直後に筆者の主張が来ることが多い | しかし、だが、ところが、けれども |
| **結論の接続詞** | 筆者の最終的な結論 | したがって、よって、つまり、すなわち |
| **強調表現** | 筆者が特に伝えたいこと | ～こそ、～にほかならない、～べきである |

**Step 3**: 筆者の主張は**文末付近**にあることが多い（特に「したがって」「つまり」の後）

**Step 4**: 選択肢と本文の主張を照合。本文に書かれていないことを述べている選択肢は排除

#### 中文翻譯

解題步驟：

1. **先讀選項**，掌握論點關鍵字
2. **讀正文時注意標記**：逆接接續詞（しかし等）後常接作者主張；結論接續詞（したがって等）後是最終結論
3. 作者主張多在**文末附近**
4. 排除正文中未提及內容的選項

#### 出題分析

要旨把握的陷阱：① 選項使用了正文的詞彙但意思偷換；② 正文有提及但非作者「主張」的事實陳述。關鍵是區分「事實」和「主張」。

## 空欄補充の解法

### 接続詞分類表

#### 知識點整理

| 分類 | 機能 | 代表的な接続詞 |
|------|------|---------------|
| **順接** | 原因→結果 | だから、したがって、そのため、よって、それゆえ |
| **逆接** | 予想に反する展開 | しかし、だが、ところが、けれども、にもかかわらず |
| **並列・添加** | 情報の追加 | また、そして、さらに、しかも、その上 |
| **対比** | 二つの事柄の比較 | 一方、他方、それに対して、むしろ |
| **補足・説明** | 前述の内容を補う | なぜなら、というのも、つまり、すなわち |
| **転換** | 話題の変更 | さて、ところで、では、それでは |
| **例示** | 具体例の提示 | たとえば、具体的には、いわば |
| **言い換え** | 別の表現で言い直す | つまり、すなわち、要するに、いわば |

#### 中文翻譯

| 分類 | 功能 | 代表接續詞 |
|------|------|-----------|
| **順接** | 原因→結果 | だから、したがって、よって |
| **逆接** | 與預期相反 | しかし、だが、ところが |
| **並列・添加** | 追加資訊 | また、さらに、しかも |
| **對比** | 比較兩事物 | 一方、それに対して、むしろ |
| **補足・說明** | 補充前述內容 | なぜなら、つまり、すなわち |
| **轉換** | 改變話題 | さて、ところで |
| **舉例** | 提示具體例 | たとえば、具体的には |
| **換言** | 用其他說法重述 | つまり、すなわち、要するに |

### 解題步驟

#### 知識點整理

**Step 1**: 空欄の前後の文を読む

**Step 2**: 前後の論理関係を判断する

| 前後関係 | 入る接続詞 |
|----------|-----------|
| 前文が原因、後文が結果 | 順接（したがって、だから） |
| 前文と後文が対立 | 逆接（しかし、だが） |
| 前文の具体例が後文 | 例示（たとえば） |
| 前文と後文が同内容 | 言い換え（つまり、すなわち） |
| 後文が前文の理由 | 補足（なぜなら） |

**Step 3**: 候補が複数ある場合、文脈の流れ全体を確認して絞り込む

#### 出題分析

接續詞の分類を正確に記憶していれば解ける問題がほとんど。特に「逆接」と「順接」の判斷が最重要。空欄の前後だけでなく、段落全体の流れを把握すること。

## 文章並替の解法

### 解題步驟

#### 知識點整理

**Step 1**: 冒頭と末尾の段落を確定する

- **冒頭の特徴**: 問題提起、テーマの提示、一般論の導入
- **末尾の特徴**: 結論、まとめ、筆者の主張

**Step 2**: 指示語・接続詞の連鎖で順序を判断する

| 手がかり | 判断方法 |
|----------|----------|
| **指示語**（この、その、それ、これ） | 指示語が指す内容が前段落にある |
| **接続詞** | 逆接の後には対立する内容、順接の後には結果が来る |
| **具体例** | 一般論→具体例の順で並ぶ |
| **固有名詞の初出** | 説明なしに使われる固有名詞は、先に紹介された段落の後に来る |

**Step 3**: 確定した部分から残りの段落を論理的に配置

**Step 4**: 通読して自然な流れか確認

#### 中文翻譯

解題步驟：

1. **確定首尾段落**：首段多為問題提起・主題提示，末段多為結論・主張
2. **追蹤指示語鏈**：「この」「その」指向前文內容，藉此判定順序
3. **利用接續詞**：逆接後接對立內容、順接後接結果
4. 從已確定部分向外擴展，最後通讀確認

#### 出題分析

段落排序是三種題型中最難的，但有明確的解題策略。關鍵是先確定首尾段落（通常最容易判斷），再用指示語鏈串聯中間段落。

## 時間配分

### 推奨時間配分

#### 知識點整理

| 問題 | 推奨時間 | 備考 |
|------|----------|------|
| 要旨把握 | **5分** | 選択肢→本文→照合 |
| 空欄補充 | **4分** | 前後の論理関係判断 |
| 文章並替 | **6分** | 最も時間がかかる |
| **合計** | **15分** | 基礎知識科目の試験時間目安 |

#### 中文翻譯

文章理解3題合計約15分鐘。建議順序：先做空欄補充（最快解完）→ 要旨把握 → 最後做段落排序（最耗時）。

#### 出題分析

時間管理非常重要。基礎知識科目全14題的建議時間是30分鐘左右，其中文章理解佔15分鐘。剩餘15分鐘分配給其他11題（各約1.5分鐘）。

## 練習方法

### 推奨教材

!!! warning "著作権に関する注意"
    文章理解の過去問は著作権の関係で**公開されていない**。以下の代替教材で練習する。

| 教材 | 特徴 |
|------|------|
| **公務員試験の文章理解** | 出題形式がほぼ同一、問題数が豊富 |
| **大学入試現代文** | 読解力の基礎訓練に有効 |
| **新聞社説の要約練習** | 要旨把握力の向上に有効 |
| **予備校の対策講座** | 行政書士試験に特化した解法指導 |

### 練習のポイント

1. **毎日1問**：短時間でも毎日取り組むことで読解スピードが向上
2. **時間を計る**：1問5分の時間制限を設けて練習
3. **接続詞に印をつける**：文章を読む際、接続詞にマーカーをつける習慣
4. **間違えた問題の分析**：なぜその選択肢が正解なのかを論理的に説明できるまで復習
