# 第四章 民衆訴訟及び機関訴訟（第42条〜第43条）

> 客観訴訟に分類される民衆訴訟と機関訴訟を規定する短い章。法律に定める場合のみ提起可能。

---

## 第42条（訴えの提起）【重要】

### 日文原文

> **第四十二条**
> 民衆訴訟及び機関訴訟は、法律に定める場合において、法律に定める者に限り、提起することができる。

### 中文翻譯

> **第四十二條（訴之提起）**
> 民眾訴訟及機關訴訟，僅於法律所定之情形，由法律所定之人，始得提起。

### 白話講解

本條是客觀訴訟（民眾訴訟和機關訴訟）的核心規定，確立了**法定主義**：

1. **「法律に定める場合」**：僅在法律特別規定的情形下才能提起（訴訟類型法定）
2. **「法律に定める者」**：僅限法律規定有資格的人才能提起（原告法定）

這與主觀訴訟（抗告訴訟、當事者訴訟）有根本性的差異。主觀訴訟中，只要具有法律上的利益就能提起；但客觀訴訟必須有法律的特別授權。

**理由**：客觀訴訟不是為了保護個人權利，而是為了維護客觀的法律秩序。如果沒有法定限制，任何人都可以對任何行政行為提起訴訟，將導致行政運作的不安定。

---

## 第43条（抗告訴訟又は当事者訴訟に関する規定の準用）【重要】

### 日文原文

> **第四十三条**
> 民衆訴訟又は機関訴訟で、処分又は裁決の取消しを求めるものについては、第九条及び第十条第一項の規定を除き、取消訴訟に関する規定を準用する。
>
> ２　民衆訴訟又は機関訴訟で、処分又は裁決の無効の確認を求めるものについては、第三十六条の規定を除き、無効等確認の訴えに関する規定を準用する。
>
> ３　民衆訴訟又は機関訴訟で、前二項に規定する訴訟以外のものについては、第三十九条及び第四十条第一項の規定を除き、当事者訴訟に関する規定を準用する。

### 中文翻譯

> **第四十三條（抗告訴訟或當事者訴訟相關規定之準用）**
> 民眾訴訟或機關訴訟中，請求撤銷處分或裁決者，除第九條及第十條第一項之規定外，準用撤銷訴訟之相關規定。
>
> ２　民眾訴訟或機關訴訟中，請求確認處分或裁決無效者，除第三十六條之規定外，準用無效等確認之訴之相關規定。
>
> ３　民眾訴訟或機關訴訟中，前二項規定以外之訴訟者，除第三十九條及第四十條第一項之規定外，準用當事者訴訟之相關規定。

### 白話講解

本條規定民眾訴訟和機關訴訟的準用關係。重點是**排除適用**的規定：

| 訴訟態樣 | 準用規定 | 排除適用 | 排除理由 |
|----------|----------|----------|----------|
| 請求撤銷處分/裁決 | 撤銷訴訟之規定 | **第9條**（原告適格）、**第10條第1項**（撤銷理由限制） | 客觀訴訟不以個人法律上利益為前提 |
| 請求確認無效 | 無效等確認之訴之規定 | **第36條**（原告適格） | 同上 |
| 其他 | 當事者訴訟之規定 | **第39條**（起訴通知）、**第40條第1項**（逾期起訴） | 性質不同 |

排除第9條和第36條的原因：客觀訴訟的原告適格由各個別法律規定（第42條），不需要適用第9條的「法律上的利益」要件。

## 四大訴訟類型總整理

| | 主觀訴訟 | 客觀訴訟 |
|---|---|---|
| 抗告訴訟 | ✓ | |
| 当事者訴訟 | ✓ | |
| 民衆訴訟 | | ✓ |
| 機関訴訟 | | ✓ |

### 主觀訴訟 vs 客觀訴訟的差異

| 比較項目 | 主觀訴訟 | 客觀訴訟 |
|----------|----------|----------|
| 目的 | 保護個人權利利益 | 維護客觀法律秩序 |
| 原告 | 具有法律上利益之人 | 法律所定之人 |
| 提起條件 | 符合訴訟要件即可 | 法律有特別規定時 |
| 類型 | 抗告訴訟、當事者訴訟 | 民眾訴訟、機關訴訟 |

### 民眾訴訟的具體例

| 法律 | 訴訟類型 | 内容 |
|------|----------|------|
| 公職選舉法204條 | 選舉無效訴訟 | 選舉規定違反的選舉無效 |
| 公職選舉法207條 | 當選無效訴訟 | 當選的效力 |
| 地方自治法242條之2 | 住民訴訟 | 地方公共團體財務會計上違法行為 |

### 機關訴訟的具體例

| 法律 | 訴訟類型 | 内容 |
|------|----------|------|
| 地方自治法176條7項 | 議會議決無效確認 | 首長與議會間的爭議 |
| 地方自治法251條之5 | 國地方爭訟 | 國家與地方公共團體間的權限爭議 |

!!! info "出題傾向"
    本章出題頻率不高，但常與第一章的訴訟類型定義結合出題。常考的知識點：(1) 客觀訴訟的法定主義（第42條）— 法律に定める場合に限り提起可能；(2) 主觀訴訟 vs 客觀訴訟的分類；(3) 第43條的排除適用 — 排除第9條（原告適格）的理由；(4) 住民訴訟和選舉訴訟作為民眾訴訟的具體例。特別注意在五肢擇一問題中，「民衆訴訟は、自己の法律上の利益にかかわらない資格で提起する」這類正確記述的辨識。
