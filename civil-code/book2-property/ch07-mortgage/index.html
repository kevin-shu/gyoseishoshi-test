    第十章 抵当権（第369条〜第398条の二十二）¶

    抵当権は行政書士試験で最頻出の担保物権。法定地上権・根抵当権・抵当権の効力範囲は毎年出題される。非占有担保が特徴（質権と異なり設定者が使用収益を続けられる）。

    第一節 総則（369条〜372条）¶

    第369条（抵当権の内容）【重要】¶

    日文原文¶

    抵当権者は、債務者又は第三者が占有を移転しないで債務の担保に供した不動産について、他の債権者に先立って自己の債権の弁済を受ける権利を有する。

    ２　地上権及び永小作権も、抵当権の目的とすることができる。この場合においては、この章の規定を準用する。

    中文翻譯¶

    第三百六十九條（抵押權的內容） 抵押權人就債務人或第三人不移轉占有而供為債務擔保之不動產，有優先於其他債權人受自己債權清償之權利。

    ２　地上權及永小作權亦得為抵押權之目的。此情形，準用本章規定。

    白話講解¶

    抵押權的四大特徵： 1. 非占有擔保 — 設定者繼續保有占有，可以繼續使用收益（這是和質權最大的區別） 2. 不動產為目的 — 原則上以不動產（土地、建物）為擔保對象；動產不適用 3. 地上權・永小作權也可以 — 不限於所有權，用益物權也可設定抵押 4. 優先弁済權 — 就擔保不動產的賣出價金，優先於其他債權人受清償

    與質權的比較：質權需要移轉占有（設定者失去使用收益），抵押權不需要，所以企業和個人偏好用抵押來融資。

    第370条（抵当権の効力の及ぶ範囲）【重要】¶

    日文原文¶

    抵当権は、抵当地の上に存する建物を除き、その目的である不動産（以下「抵当不動産」という。）に付加して一体となった物に及ぶ。ただし、設定行為に別段の定めがある場合及び債務者の行為について第四百二十四条第三項に規定する詐害行為取消請求をする場合は、この限りでない。

    中文翻譯¶

    第三百七十條（抵押權效力的及於範圍） 抵押權及於（抵押地上之建物除外）與其目的不動產（以下稱「抵押不動產」）附加而一體化之物。但設定行為有別段約定，或對債務人之行為有第四百二十四條第三項詐害行為撤銷請求之情形，不在此限。

    白話講解¶

    抵押權效力所及的範圍：

    對象 是否及於 說明
    付加一體物 原則上及於 附著在不動產上、成為一體的物（例如：建物內裝、固定式空調）
    抵押地上的建物 原則上不及 土地和建物是獨立的不動產，各自獨立（法定地上權問題）
    從物（371條） 及於 附屬的從物（車庫等）
    果實（債務不履行後）（371條） 及於 天然果實・法定果實（賃料）

    例外：設定行為有別段約定時，或債務人做出詐害行為時，可依詐害行為撤銷請求調整效力範圍。

    第371条（抵当権の効力）¶

    日文原文¶

    抵当権は、その担保する債権について不履行があったときは、その後に生じた抵当不動産の果実に及ぶ。

    中文翻譯¶

    第三百七十一條（抵押權的效力） 抵押權於其擔保債權有不履行時，及於其後抵押不動產所生之果實。

    白話講解¶

    果實（天然果實・法定果實＝賃料）在債務不履行後才被抵押權的效力所及。

    重要連結：賃料への物上代位（304條準用）—— 抵押權人可以對賃料行使物上代位，但必須先辦理差押え（扣押）手續，才能在賃料被支付給第三人前優先受償。

    第372条（留置権等の規定の準用）¶

    日文原文¶

    第二百九十六条、第三百四条及び第三百五十一条の規定は、抵当権について準用する。

    中文翻譯¶

    第三百七十二條（留置權等規定的準用） 第二百九十六條、第三百四條及第三百五十一條之規定準用於抵押權。

    白話講解¶

    準用的三個條文： - 296條（不可分性） — 債權全部清償完畢前，抵押權不消滅 - 304條（物上代位） — 擔保物的替代價值（賣出代金、保險金、賠償金等）上可行使抵押權 - 351條（物上保證人的求償權） — 用自己財產設定抵押卻不是債務人的人（物上保證人），代位清償後可向債務人求償

    第二節 抵当権の効力（373条〜395条）¶

    第373条（抵当権の順位）【重要】¶

    日文原文¶

    同一の不動産について数個の抵当権が設定されたときは、その抵当権の順位は、登記の前後による。

    中文翻譯¶

    第三百七十三條（抵押權的順位） 同一不動產設定數個抵押權時，其抵押權順位依登記之先後定之。

    白話講解¶

    同一不動產上設定多個抵押權時，登記先後決定順位。第1順位（1番抵当権）的人最先受到清償。

    例：不動產價值1,000萬，1番抵当600萬，2番抵当500萬 → 1番先拿600萬，剩400萬給2番（2番只能拿到400萬，無法全額受償）

    第374条（抵当権の順位の変更）¶

    日文原文¶

    抵当権の順位は、各抵当権者の合意によって変更することができる。ただし、利害関係を有する者があるときは、その承諾を得なければならない。

    ２　前項の順位の変更は、その登記をしなければ、効力を生じない。

    中文翻譯¶

    第三百七十四條（抵押權順位的變更） 抵押權之順位得由各抵押權人合意變更之。但有利害關係者時，應得其承諾。

    ２　前項順位之變更，非登記不生效力。

    白話講解¶

    順位變更的三個要件： 1. 各抵押權人全員合意 — 不能僅由部分抵押權人決定 2. 利害關係人的承諾 — 有利害關係的人（例如次順位抵押權人、一般債權人）需要同意 3. 登記 — 登記是效力要件（非對抗要件），未登記則變更不生效力

    第375条（抵当権の被担保債権の範囲）【重要】¶

    日文原文¶

    抵当権者は、利息その他の定期金を請求する権利を有するときは、その満期となった最後の二年分についてのみ、その抵当権を行使することができる。ただし、それ以前の定期金についても、満期後に特別の登記をしたときは、その登記の時からその抵当権を行使することができる。

    ２　前項の規定は、抵当権者が債務の不履行によって生じた損害の賠償を請求する権利を有する場合における最後の二年分についても適用する。

    中文翻譯¶

    第三百七十五條（抵押權被擔保債權的範圍） 抵押權人有請求利息或其他定期金之權利時，就最後二年分屆期者，僅能行使其抵押權。但就此前之定期金，屆期後辦理特別登記時，得自登記時起行使抵押權。

    ２　前項之規定，於抵押權人有請求因債務不履行所生損害賠償之情形，就最後二年分亦適用之。

    白話講解¶

    利息・損害賠償的優先受償範圍限定為最後2年分，理由是保護後順位抵押權人（讓他們可以預測自己的受償範圍）。

    債權種類 優先受償範圍
    元本 全額
    利息・定期金 最後2年分（僅此）
    損害賠償 最後2年分（僅此）

    例外：就2年以前的利息，若在屆期後辦理特別登記（付記登記），亦可對抗後順位抵押權人。

    第376条（抵当権の処分）【重要】¶

    日文原文¶

    抵当権者は、その抵当権を他の債権の担保とし、又は同一の債務者に対する他の債権者の利益のためにその抵当権若しくはその順位を譲渡し、若しくは放棄することができる。

    中文翻譯¶

    第三百七十六條（抵押權的處分） 抵押權人得以其抵押權為他債權之擔保，或為同一債務人之他債權人利益，讓與或放棄其抵押權或其順位。

    白話講解¶

    抵押權人可以對自己的抵押權做以下四種處分：

    處分類型 說明 效果
    転抵当 以自己的抵押權再設定另一個擔保 原抵押權繼續存在，再以其作為擔保
    順位の譲渡 把自己的順位讓給無擔保債權人 受讓人可以用原讓與人的順位受償
    順位の放棄 放棄對無擔保債權人的優先度 讓與人和無擔保債權人按比例均等受償
    抵当権の譲渡・放棄 把整個抵押權讓與或放棄給無擔保債權人 受讓/受益人可優先或均等受償

    對抗要件（377條）：需在抵押權的登記上辦理付記登記，並通知或獲得設定者的承諾。

    第377条（抵当権の処分の対抗要件）¶

    日文原文¶

    前条の場合には、第四百六十七条の規定に従い、主たる債務者に通知をし、又は主たる債務者が承諾をしなければ、これを債務者、保証人、抵当権設定者及びこれらの者の承継人に対抗することができない。

    ２　前項の通知は、抵当権の登記に付記しなければ、その通知後に登記をした抵当権者に対抗することができない。

    中文翻譯¶

    第三百七十七條（抵押權處分的對抗要件） 前條情形，依第四百六十七條之規定，非通知主債務人或主債務人承諾，不得對抗債務人、保證人、抵押權設定者及其繼承人。

    ２　前項通知，非在抵押權登記上辦理付記登記，不得對抗通知後辦理登記之抵押權人。

    白話講解¶

    抵押權處分的對抗要件有兩層： 1. 對債務人・保證人等：需通知主債務人或獲得其承諾（準用467條，採確定日期的通知） 2. 對其後登記的抵押權人：在登記上辦理付記登記才能對抗

    第378条（代価弁済）¶

    日文原文¶

    抵当不動産について所有権又は地上権を買い受けた第三者が、抵当権者の請求に応じてその抵当権者にその代価を弁済したときは、抵当権は、その第三者のために消滅する。

    中文翻譯¶

    第三百七十八條（代價弁済） 買受抵押不動產所有權或地上權之第三人，應抵押權人之請求向抵押權人弁済其代價時，抵押權為該第三人消滅。

    白話講解¶

    代價弁済（378條）vs 抵押權消滅請求（379條）的比較：

    　 代價弁済（378條） 抵押權消滅請求（379條）
    誰先提出 抵押權人向第三取得者請求 第三取得者主動向抵押權人申請
    金額 第三取得者支付的取得代價 第三取得者自己提出的金額（可比代價低）
    抵押權人的選擇 只要接受就消滅 可在2個月內申請競賣來拒絕

    第379条〜第387条（抵当権消滅請求）¶

    白話講解（重點速覽）¶

    抵押權消滅請求（379〜387條）：

    第三取得者可以提出一個金額，請求所有抵押權人在2個月內決定是否接受。抵押權人若不接受，須在2個月內申請競賣（383條）。競賣後若不足清償，第三取得者不需補足差額。

    主要流程： 1. 第三取得者向抵押權人送付書類（抵押權消滅請求書） 2. 各抵押權人有2個月的期間可申請競賣 3. 若無人申請競賣 → 提出金額被承諾，辦理登記後抵押權消滅 4. 若有人申請競賣 → 依競賣結果處理（第三取得者的費用由申請競賣的抵押權人負擔：387條）

    保證人・物上保證人不得提出消滅請求（380條）。

    第388条（法定地上権）【重要】¶

    日文原文¶

    土地及びその上に存する建物が同一の所有者に属する場合において、その土地又は建物につき抵当権が設定され、その実行により所有者を異にするに至ったときは、その建物について、地上権が設定されたものとみなす。この場合において、地代は、当事者の請求により、裁判所が定める。

    中文翻譯¶

    第三百八十八條（法定地上権） 土地及其上存在之建物屬同一所有人時，就該土地或建物設定抵押權，其實行致所有人異其人時，就該建物推定設定地上権。此情形，地代依當事人請求由法院定之。

    白話講解¶

    法定地上権的成立要件（須全部滿足以下4項）：

    1. 抵押權設定時，土地和建物屬於同一所有人
    2. 土地或建物上設定了抵押權（僅一方亦可）
    3. 因抵押權的實行而進行競賣
    4. 競賣結果，土地和建物的所有人分屬不同人

    效果：建物當然視為已設定地上権（保護建物得以繼續存在）。地代如當事人無法合意，由法院決定。

    重要判例¶

    判例 内容
    最判昭52.10.11 抵当権設定時に建物が存在すれば足り、後に建て替えた場合は原則として法定地上権不成立（旧建物の基準で判断）
    最判昭36.2.10 抵当権設定時に更地（建物なし）だった場合は法定地上権不成立（設定後に建物を建てても不成立）
    最判昭47.11.2 土地のみに抵当権設定時、土地抵当権者は建物の存在を認識して設定した → 法定地上権成立
    最判昭51.10.8 建物のみに抵当権設定時 → 土地が売却されても法定地上権成立
    最判平2.1.22 土地と建物が共有の場合、土地共有者の一人が自己の持分のみに抵当権設定 → 法定地上権不成立（他の共有者の利益を害するため）

    出題パターン¶

    • 更地への抵当権設定後に建物が建てられた場合 → 不成立（最頻出）
    • 設定時に建物あり、後に建て替えた → 原則不成立
    • 土地のみに抵当権設定、設定時に建物あり → 成立
    • 共有土地への一部持分の抵当権設定 → 不成立
    • 「建物について」法定地上権が成立する点に注意（土地ではなく建物のため）

    第389条（抵当地の上の建物の競売）¶

    日文原文¶

    抵当権の設定後に抵当地に建物が築造されたときは、抵当権者は、土地とともにその建物を競売することができる。ただし、その優先権は、土地の代価についてのみ行使することができる。

    中文翻譯¶

    第三百八十九條（抵押地上建物的競賣） 抵押權設定後抵押地上築造建物時，抵押權人得連同建物競賣該土地。但其優先權僅能就土地代價行使。

    白話講解¶

    一括競売（389條）：

    抵押權設定後所建造的建物，因不滿足388條的成立要件，法定地上権不成立。 因此，若僅競賣土地，建物的買受人將無法取得使用土地的權利，故允許將土地和建物一併競賣。

    重要限制：抵押權人僅能就土地的賣出代金行使優先受償權。建物的代金（不歸抵押權人）歸屬於建物的所有人。

    第390条〜第391条（第三取得者の費用償還請求）¶

    白話講解（速覽）¶

    • 390條：第三取得者為保存、改良抵押不動產而支出費用時，得如同留置權人般請求費用償還（準用196條）
    • 391條：第三取得者預先支付費用使抵押權代位消滅時的求償關係

    第392条（共同抵当における代価の配当）【重要】¶

    日文原文¶

    債権者が同一の債権の担保として数個の不動産につき抵当権を有する場合において、同時にその代価を配当するときは、その各不動産の価額に応じて、その債権の負担を按分する。

    ２　前項の場合において、ある不動産の代価のみを配当するときは、抵当権者は、その代価から債権の全部の弁済を受けることができる。この場合においても、次順位の抵当権者は、その弁済を受ける抵当権者が前項の規定に従い他の不動産の代価から弁済を受けるべき金額を限度として、その抵当権者に代位して抵当権を行使することができる。

    中文翻譯¶

    第三百九十二條（共同抵押中代價的分配） 債權人就同一債權擔保在數個不動產設定抵押權時，同時分配各代價，按各不動產價額比例分擔債權負擔。

    ２　前項情形，僅分配某一不動產代價時，抵押權人得自該代價就債權全部受弁済。此情形，次順位之抵押權人得在該抵押權人依前項規定應自他不動產代價受弁済之金額限度內，代位行使抵押權人之抵押權。

    白話講解¶

    共同抵当（在複數不動產上擔保同一債權）的分配方法：

    配当方法 規則
    同時配当 按各不動產價額比例按分負擔
    異時配当 先行競賣的不動產之次順位抵押權人，得在「本應自其他不動產代金受清償之金額」限度內，代位行使抵押權人對其他不動產的抵押權

    具體例：甲地（800萬）和乙地（200萬）上設定1,000萬的共同抵押。僅競賣甲地（甲地上有後順位D）。 - 若同時配當，應按比例從甲地分得800萬、乙地分得200萬 - 僅競賣甲地時，D本應從甲地受償400萬，但實際上變成0 → D得為乙地而代位抵押權人，從乙地回收200萬（須辦理393條的付記登記）

    第393条〜第394条（代位と例外弁済）¶

    白話講解（速覽）¶

    • 393條：依392條第2項的代位，若未辦理登記（付記登記），不得對抗後順位抵押權人
    • 394條：抵押不動產的代金不足清償債權全額時，剩餘金額得以一般債權向債務人其他財產追索（抵押權人不失去作為一般債權人的地位）

    第395条（抵当建物使用者の引渡しの猶予）¶

    日文原文¶

    抵当権者に対抗することができない賃貸借により抵当権の目的である建物の使用又は収益をする者であって次に掲げるもの（次項において「抵当建物使用者」という。）は、その建物の競売における買受人の買受けの時から六箇月を経過するまでは、その建物を買受人に引き渡すことを要しない。

    中文翻譯¶

    第三百九十五條（抵押建物使用者的交付猶予） 依不能對抗抵押權之租賃在抵押建物使用收益者……自競賣買受人買受時起六個月內，無須向買受人交付該建物。

    白話講解¶

    抵押權設定後才入住的承租人（無法對抗抵押權），對於競賣後的買受人亦享有6個月的交付猶予期。

    但在這6個月期間，承租人須向買受人支付相當於租金的使用對價（395條第2項）。

    第三節 抵当権の消滅（396条〜398条）¶

    第396条（抵当権の消滅時効）¶

    日文原文¶

    抵当権は、債務者及び抵当権設定者に対しては、その担保する債権と同時でなければ、時効によって消滅しない。

    中文翻譯¶

    第三百九十六條（抵押權的消滅時效） 抵押權，對債務人及抵押權設定者，非與其擔保之債權同時，不因時效消滅。

    白話講解¶

    抵押權的附従性（付従性）的體現：抵押權單獨不會對債務人・設定者發生消滅時效，必須跟著被擔保債權同時消滅。

    例：被擔保債權罹於消滅時效時 → 抵押權亦同時消滅（396條）。

    但需注意，對第三取得者則獨立進行時效（397條）。

    第397条（抵当不動産の時効取得による抵当権の消滅）¶

    日文原文¶

    債務者又は抵当権設定者でない者が抵当不動産について取得時効に必要な要件を具備する占有をしたときは、抵当権は、これによって消滅する。

    中文翻譯¶

    第三百九十七條（因抵押不動產時效取得消滅抵押權） 非債務人或抵押權設定者之人，就抵押不動產具備取得時效要件之占有時，抵押權因此消滅。

    白話講解¶

    當非債務人且非設定者的第三人就抵押不動產完成取得時效時，抵押權因此消滅。

    這是獨立於附従性（396條）之外的另一種消滅事由。即使債務人或設定者本身進行時效取得，抵押權並不消滅；但若是第三人完成取得時效，抵押權則消滅。

    第398条（抵当権の目的物の滅失等）¶

    白話講解（速覽）¶

    抵押權的目的物（不動產）滅失或毀損時，可對設定者所取得的損害賠償請求權或保險金請求權行使物上代位（準用304條）。但須先辦理差押え（扣押）手續。

    第四節 根抵当（398条の2〜398条の22）¶

    第398条の2（根抵当権）【重要】¶

    日文原文¶

    抵当権は、設定行為で定めるところにより、一定の範囲に属する不特定の債権を極度額の限度において担保するためにも設定することができる。

    ２　前項の抵当権（以下「根抵当権」という。）については、第三百七十五条及び第三百七十六条の規定は、適用しない。

    ３　第一項の場合において、債権の範囲に属する元本のほか、利息その他の定期金及び債務の不履行によって生じた損害の賠償の全部を極度額の限度において担保する。

    中文翻譯¶

    第三百九十八條の二（根抵押權） 抵押權依設定行為之約定，亦得設定以擔保屬一定範圍之不特定債權在極度額限度內。

    ２　前項之抵押權（以下稱「根抵押權」），不適用第三百七十五條及第三百七十六條之規定。

    ３　前項情形，除屬債權範圍之元本外，利息等定期金及因債務不履行所生損害賠償之全部，均在極度額限度內擔保。

    白話講解¶

    根抵当権（ねていとうけん）的特徵：

    項目 根抵当権 通常の抵当権
    被担保債権 一定範囲の不特定の将来債権 特定の既存債権
    極度額 あり（担保の上限） なし（被担保債権の全額）
    付従性 弱い（元本確定前は個別債権と切り離す） あり（被担保債権と運命を共にする）
    375條の適用 なし（利息等は全額が極度額内で担保） あり（利息は最後2年分のみ）
    376條の適用 なし（順位の譲渡・放棄不可） あり

    銀行等對企業持續融資時廣泛運用（無需每次融資都重新設定抵押權）。

    第398条の3（根抵当権の被担保債権の範囲）¶

    日文原文¶

    根抵当権は、設定行為で定めるところにより、一定の範囲に属する不特定の債権を担保する。

    中文翻譯¶

    第三百九十八條の三（根抵押權被擔保債權的範圍） 根抵押權依設定行為之約定，擔保屬一定範圍之不特定債權。

    白話講解¶

    被擔保債權的範圍須在設定行為中加以特定。例：「對A銀行之所有銀行交易上的債務」。

    根抵当権雖擔保不特定的債權，但仍須限定在「一定範圍」內（無任何範圍限定者無效）。

    第398条の4（根抵当権の被担保債権の範囲及び債務者の変更）¶

    日文原文¶

    根抵当権の担保すべき債権の範囲及び債務者の変更は、元本の確定前に限り、することができる。

    ２　前項の変更をするには、後順位の抵当権者その他の第三者の承諾を得ることを要しない。

    ３　第一項の変更について元本の確定前に登記をしなければ、その変更は、第三者に対抗することができない。

    中文翻譯¶

    第三百九十八條の四（根抵押權被擔保債權範圍及債務者的變更） 根抵押權擔保債權之範圍及債務者之變更，僅限元本確定前為之。

    ２　為前項變更，不需得後順位抵押權人及其他第三人之承諾。

    ３　前項變更，非於元本確定前辦理登記，不得對抗第三人。

    白話講解¶

    根抵当権彈性的體現：在元本確定前，可變更被擔保債權的範圍和債務人，且無需後順位抵押權人的承諾（這與一般抵押權的順位變更不同）。

    但登記為對抗要件（而非效力要件）。確定前未辦理登記，則不得對抗第三人。

    第398条の5〜18（根抵当権の各種変更・処分）¶

    白話講解（速覽）¶

    条文 内容
    398条の5 極度額の変更は、利害関係人の承諾が必要、登記が効力要件
    398条の6 根抵当権の譲渡（元本確定前）は設定者の承諾が必要
    398条の7 元本確定前の根抵当権の被担保債権の譲渡・引受けは、根抵当権に影響しない（付従性の排除）
    398条の8 根抵当権の共有（共同根抵当権）に関する特則
    398条の9〜10 相続・合併の場合、合意の登記が必要（6ヶ月以内）、未登記なら元本確定
    398条の11〜18 共同根抵当、累積型・純粋型の共同根抵当等の特則

    第398条の19（根抵当権の元本の確定請求）¶

    日文原文¶

    根抵当権設定者は、根抵当権の設定の時から三年を経過したときは、担保すべき元本の確定を請求することができる。この場合において、担保すべき元本は、その請求の時から二週間を経過することによって確定する。

    ２　根抵当権者は、いつでも、担保すべき元本の確定を請求することができる。この場合において、担保すべき元本は、その請求の時に確定する。

    中文翻譯¶

    第三百九十八條の十九（根抵押權元本確定請求） 根抵押權設定者自根抵押權設定時起經三年後，得請求確定應擔保之元本。此情形，應擔保之元本於該請求時起經二週後確定。

    ２　根抵押權人得隨時請求確定應擔保之元本。此情形，應擔保之元本於該請求時確定。

    白話講解¶

    元本確定請求の比較：

    請求者 要件 確定時期
    設定者（債務者・物上保証人等） 設定から3年経過後に請求可 請求から2週間後
    根抵当権者 いつでも請求可 請求したその時

    元本確定後，僅擔保該時點已存在的被擔保債權（此後新發生的債權不再擔保）。確定後與一般抵押權相同方式運作。

    第398条の20（根抵当権の元本の確定事由）【重要】¶

    日文原文¶

    次に掲げる場合には、根抵当権の担保すべき元本は、確定する。

    一　根抵当権者が抵当不動産について競売若しくは担保不動産収益執行又は第三百四条第一項に規定する差押えを申し立てたとき。

    三　根抵当権者又は債務者が破産手続開始の決定を受けたとき。

    五　根抵当権設定者が、根抵当権者の同意を得て抵当不動産につき競売を申し立てたとき。

    中文翻譯¶

    第三百九十八條の二十（根抵押權元本確定事由） 有下列各款情形時，根抵押權應擔保之元本確定：

    一　根抵押權人申請抵押不動產競賣或擔保不動產收益執行或第三百零四條第一項所定差押時。

    三　根抵押權人或債務者受破產手續開始決定時。

    五　根抵押權設定者得根抵押權人同意申請抵押不動產競賣時。

    白話講解¶

    元本の法定確定事由：

    号 事由
    第1号 根抵当権者が競売・差押えを申し立てたとき
    第2号 根抵当権者が物上代位の差押えを申し立てたとき
    第3号 根抵当権者または債務者が破産手続開始決定を受けたとき
    第4号 根抵当権の実行（担保不動産収益執行）が申し立てられたとき
    第5号 設定者が根抵当権者の同意を得て競売申立てしたとき

    元本確定後的效果：附従性恢復，根抵当権與擔保特定債權的一般抵押權相同。此後不得再進行變更或追加設定。

    抵当権 vs 根抵当権 比較總表【考試必背】¶

    比較項目 普通の抵当権 根抵当権
    被担保債権 特定の債権 一定範囲の不特定債権
    極度額 なし あり（必須）
    付従性 あり（強い） 元本確定前は弱い
    随伴性 あり 元本確定前はなし
    利息の範囲 最後2年分のみ（375条） 全額（極度額内）
    順位の譲渡・放棄 可（376条） 不可
    元本確定 概念なし（特定の債権が対象） 確定後に付従性復活
    変更 利害関係人の承諾・登記必要 元本確定前は後順位者承諾不要
    相続・合併 一般原則 6ヶ月以内の合意登記が必要
    主な用途 個別の貸付け担保 継続的取引（銀行融資等）

    抵当権の消滅事由まとめ¶

    消滅事由 説明
    被担保債権の消滅 付従性により抵当権も消滅（396條）
    時効（第三取得者） 第三者が時効取得すると消滅（397條）
    代価弁済 第三取得者が抵当権者の請求に応じ代価を支払う（378條）
    抵当権消滅請求 第三取得者が金額を提示し、抵当権者が競売申立てしない（379條〜）
    混同 所有者と抵当権者が同一人になった場合（179條ただし書の例外あり）
    抵当権の放棄 抵当権者が任意に放棄
