    記述式 答題戰略一頁整理¶

    配點概覽

    3問 × 20點 = 60點（全体の20%）。択一式を落とした時の命綱。絶対に白紙は禁物。

    1. 採点の仕組みを理解する¶

    キーワード採点方式¶

    採点基準は非公開だが、実態は 「キーワードごとに点数加算」 方式。

    模範解答と一字一句同じでなくていい
→ キーワードが含まれていれば部分点がもらえる
→ 1問で 4〜8点の部分点を積み上げることが現実的な戦略

    目標スコア設定¶

    戦略 目標点 備考
    安全圏 40点以上（67%） 合格ライン確保のため必要
    現実的 30〜36点 各問10〜12点の部分点
    最低ライン 20点以上（33%） これ以下だと択一で過負荷

    2. 解答の3ステップ（どの問題にも共通）¶

    Step 1: 事案把握（30秒）
  → 「誰が」「何を」「誰に対して」求めているか
  → 登場人物・処分・法的関係を整理

Step 2: テーマ特定（30秒）
  → 「この事案は何の制度の問題か？」を即座に判断
  → 行政法なら：取消訴訟？義務付け訴訟？差止訴訟？
  → 民法なら：代理？時効？不当利得？不法行為？

Step 3: キーワードを40字に組み立て（2分）
  → 設問が「～は何か」→ 名称を先に書く
  → 設問が「～の要件は」→ 箇条書き的に列挙
  → 設問が「～をせよ」→ 理由付きで結論を書く

    3. 行政法（問44）の頻出パターン¶

    出題傾向¶

    行政事件訴訟法からほぼ毎年出題。

    年度 テーマ
    令和7年 建築確認拒否・審査請求裁決の手続的瑕疵
    令和6年 無線局免許の競願関係・取消訴訟
    令和5年 議会懲罰に対する差止訴訟
    令和4年 義務付け訴訟の訴訟要件
    令和3年 行政指導の法的性質
    令和元年 被告適格・訴訟類型
    平成30年 不作為に対する訴訟類型

    Pattern A：訴訟類型の選択¶

    設問形式：「Aは何という訴訟を提起すべきか。その要件と共に述べよ。」

    解答フォーム： 

    [訴訟名]を提起すべきであり、
[要件①]であり、かつ[要件②]であることが必要である。

    必須キーワード一覧：

    訴訟 キーワード
    取消訴訟 処分性、原告適格、訴えの利益、出訴期間（6ヶ月）、被告（行政庁の所属する国or公共団体）
    義務付け訴訟（申請型） 申請、拒否処分、不作為、取消訴訟との併合提起
    義務付け訴訟（非申請型） 重大な損害、補充性（他に適当な方法がない）
    差止訴訟 重大な損害、補充性、蓋然性
    不作為の違法確認訴訟 法令に基づく申請、相当期間、被告＝国or公共団体
    無効確認訴訟 重大明白な瑕疵、現在の法律関係の確認で目的達成不可

    Pattern B：被告適格¶

    鉄則：

    原則：取消訴訟の被告 = 「処分をした行政庁の所属する国or公共団体」（行訴法11条1項）
　　　※「行政庁」ではなく「国or地方公共団体」が被告！
例外：国の機関の訴訟 = 国が被告

    Pattern C：訴訟要件ビッグ3¶

    要件 チェックポイント
    処分性 公権力の行使、直接・具体的な法的効果
    原告適格 法律上保護された利益、処分の根拠法令の保護範囲
    訴えの利益 取消しにより回復できる法的利益があるか

    4. 民法（問45・問46）の頻出パターン¶

    出題傾向¶

    分野 頻出度 主なテーマ
    債権各論（契約） ★★★ 契約解除の要件・効果、売主の担保責任
    物権 ★★★ 対抗要件、占有、即時取得
    債権総論 ★★☆ 債権譲渡、相殺、弁済
    不法行為 ★★☆ 要件・効果、過失相殺
    代理 ★★☆ 無権代理、表見代理の要件・効果
    総則（法律行為） ★☆☆ 錯誤、詐欺、心裡留保

    Pattern A：テーマ表示型（要件・効果を書け）¶

    設問形式：「Aの債務不履行解除の要件を述べよ。」

    解答フォーム： 

    [テーマ名]が成立するには、[要件①]であり、
[要件②]である必要がある（民法○条）。

    頻出テーマの必須キーワード：

    • 債務不履行（履行遅滞・履行不能・不完全履行）
    • 相当期間を定めた催告（催告解除）または無催告（催告不要の場合）
    • 解除の意思表示（形成権）
    • 効果：原状回復義務・損害賠償は妨げない
    • 故意または過失
    • 他人の権利または法律上保護される利益の侵害
    • 損害の発生
    • 行為と損害の間の因果関係
    • 動産
    • 取引行為（売買等）
    • 前主が無権利者
    • 占有の取得
    • 善意・無過失
    • 基本代理権の存在
    • 権限外の行為
    • 相手方の正当な理由（善意・無過失）

    Pattern B：テーマ未表示型（テーマ名も含めて答える）¶

    設問形式：「Aの主張は何という法律構成に基づくか。要件と共に説明せよ。」

    解答フォーム： 

    Aは[制度名]を主張できる。
この制度は[要件]を満たす場合に[効果]が生じる。

    → 事案からテーマを読み取る力が鍵。択一の学習時に「この事案は何の問題か？」を意識すること。

    5. 「40字で書く」技術¶

    字数感覚の目安¶

    10字：「不作為の違法確認訴訟を提起」
20字：「不作為の違法確認訴訟を提起し、被告は国とする」
30字：「法令に基づく申請後、相当期間内に処分がなければ、不作為の違法確認訴訟が可能」
40字：「法令に基づく申請があり相当期間経過後も処分がなければ、国を被告として不作為の違法確認訴訟を提起できる」

    字数超過・不足の対処法¶

    問題 原因 対処法
    字数不足（30字以下） 知識不足 キーワードを追加。要件を列挙
    字数超過（50字以上） 説明過多 「なぜなら〜」などの理由説明を削る

    書き出しのパターン¶

    設問形式 書き出し例
    「何か」「何というか」 ○○（制度名・条文名）である。
    「要件を述べよ」 ①〜であり、②〜であること。
    「どうすべきか」 ○○訴訟を提起すべきであり、〜を要件とする。
    「理由を述べよ」 〜であるため、○○となる（民法/行訴法○条）。

    6. 試験当日の時間配分¶

    記述式3問の目安：計30〜35分

    問44（行政法）：10〜12分
問45（民法①）：10〜12分
問46（民法②）：10〜12分

    戦略的放棄のライン

    1問で15分以上考えても解けない場合は 部分点だけ書いて次へ。 白紙より「キーワードの羅列」の方が確実に得点できる。

    部分点確保の最終手段

    完全な解答が書けなくても：

    1. 制度名・訴訟名 だけ書く（2〜4点）
    2. 条文番号 を書く（1〜2点）
    3. 関連キーワード を並べる（4〜8点）

    → 合計 8〜10点は確保できる可能性がある

    7. 重点暗記リスト（記述式専用）¶

    行政法 必須フレーズ¶

    被告適格：「処分又は裁決をした行政庁の所属する国又は公共団体」（行訴法11条1項）
出訴期間：「処分又は裁決があったことを知った日から6ヶ月以内」（行訴法14条1項）
義務付け（非申請型）要件：「重大な損害を生ずるおそれ」＋「補充性」（行訴法37条の2第1項）
差止訴訟の要件：「重大な損害を生ずるおそれ」＋「補充性」（行訴法37条の4第1項）

    民法 必須フレーズ¶

    代理効果：「本人に対して直接その効力を生ずる」（民法99条1項）
無権代理の効果：「本人に対してその効力を生じない」（民法113条1項）
表見代理（109条）：「第三者に対してその権限を与えた旨を表示」＋相手方の善意・無過失
即時取得：「平穏・公然・善意・無過失で動産の占有を取得」（民法192条）
解除の効果：「各当事者はその相手方を原状に復させる義務」（民法545条1項）
