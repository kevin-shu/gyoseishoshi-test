    情報通信・個人情報保護 — 練習問題¶

    本章出題傾向

    毎年3～4問出題。個人情報保護法は毎年1問以上。IT用語・デジタル行政関連の出題が増加傾向。法令知識＋最新IT用語の両面対策が必要だが、対策のしやすさと得点効率が最も高い分野。

    択一式¶

    問題 1（令和6年度 問54）¶

    デジタル環境での情報流通に関する次の記述のうち、妥当でないものはどれか。

    1. 生成AIが事実でない情報をあたかも事実であるかのように提供する現象を「アノテーション」という。
    2. 人々の注意力や時間が経済的価値を持つことを「アテンション・エコノミー」という。
    3. プラットフォーム事業者が違法・規約違反コンテンツを削除することを「コンテンツ・モデレーション」という。
    4. 第三者がSNS上の情報の正確性を検証することを「ファクトチェック」という。
    5. アルゴリズムにより自分の関心のある情報だけに囲まれることを「フィルターバブル」という。
    解答・解説

    正解：1

    1. ✗ 生成AIが事実でない情報をあたかも事実であるかのように提供する現象は「ハルシネーション（hallucination）」であり、「アノテーション」ではない。アノテーションとは、AI学習用のデータにラベルを付ける作業（例：画像に「犬」「猫」等のタグをつける）を意味する。
    2. ○ アテンション・エコノミー（注意経済）の正しい定義。
    3. ○ コンテンツ・モデレーションの正しい定義。
    4. ○ ファクトチェックの正しい定義。
    5. ○ フィルターバブルの正しい定義。

    中文要點: 生成式AI產生虛假資訊的現象稱為「幻覺」(hallucination)，非「標註」(annotation)。Annotation是指為AI訓練數據加標籤的作業。其他四個IT用語的定義均正確。

    問題 2（令和6年度 問56）¶

    デジタル庁に関する次の記述のうち、妥当なものはどれか。

    1. デジタル庁は総務省の外局として設置されている。
    2. 個人情報保護委員会はデジタル庁に対して行政指導を行うことができない。
    3. デジタル庁にはサイバーセキュリティ基本法に基づくサイバーセキュリティ戦略本部が置かれている。
    4. デジタル庁は官民データ活用推進基本計画の作成・推進に関する事務をつかさどる。
    5. デジタル庁の所掌事務にはマイナンバー及びマイナンバーカードに関する事項は含まれない。
    解答・解説

    正解：4

    1. ✗ デジタル庁は総務省の外局ではなく、内閣に直属する組織（2021年設置）。
    2. ✗ 個人情報保護委員会はデジタル庁に対して行政指導を行うことができる。実際に2023年9月、マイナンバーの紐付けミスに関してデジタル庁に行政指導が行われた。
    3. ✗ サイバーセキュリティ戦略本部は内閣官房に置かれており、デジタル庁内ではない。
    4. ○ デジタル庁設置法第4条2項2号に基づき、官民データ活用推進基本計画の作成および推進に関する事務をつかさどる。
    5. ✗ マイナンバー及びマイナンバーカードに関する事項はデジタル庁の所掌事務に含まれている（同法第4条2項4号）。

    中文要點: 數位廳直屬內閣（非總務省外局）。個資保護委員會可對數位廳進行行政指導（2023年已有實例）。網路安全戰略本部設於內閣官房（非數位廳）。個人編號相關事務屬數位廳所掌。

    問題 3（令和6年度 問57）¶

    個人情報保護法に関する次の記述のうち、妥当でないものはどれか。

    1. 漏えい等が発生し個人の権利利益を害するおそれが大きい場合、個人情報保護委員会への報告義務がある。
    2. 違法又は不当な行為を助長・誘発するおそれのある方法で個人情報を利用してはならない。
    3. 個人データを第三者に提供する際は、日付・相手方の氏名等を記録しなければならない。
    4. 学術研究機関が研究目的で個人情報を取り扱う場合は、個人情報取扱事業者としての義務が全面的に免除される。
    5. 個人情報保護法は国の行政機関・地方公共団体にも適用される。
    解答・解説

    正解：4

    1. ○ 個人情報保護法26条に基づき、漏えい等が発生した場合の報告義務がある。
    2. ○ 同法19条に基づく不適正利用の禁止。
    3. ○ 同法29条に基づく第三者提供の記録義務。
    4. ✗ 学術研究機関には全面的な適用免除はない。報道機関・著述業者・宗教団体・政治団体には特定目的での義務免除があるが、学術研究機関にはそのような包括的免除は存在しない。一定の義務の適用除外はあるが「全面的に免除」は誤り。
    5. ○ 2021年改正により、国の行政機関・地方公共団体にも個人情報保護法が一元的に適用される。

    中文要點: 學術研究機關沒有全面性的義務免除。有適用除外的是報道機關・著述業者・宗教團體・政治團體（各有特定目的限制）。2021年修法後，個資法統一適用於國家行政機關和地方公共團體。

    問題 4（令和5年度 問54）¶

    日本における行政のデジタル化に関する次のア～オの記述のうち、妥当でないものの組合せはどれか。

    ア. RPA（ロボティック・プロセス・オートメーション）とは、コンピュータ上で行われる業務プロセスや作業を人に代わり自動化する技術をいう。

    イ. ガバメントクラウドとは、政府が共通的に利用するクラウドサービスの利用環境のことをいうが、地方自治体は利用することができない。

    ウ. eLTAXとは、地方税における手続きをインターネットを利用して電子的に行うシステムをいう。

    エ. LGWANとは、地方公共団体を相互に接続する行政専用のネットワーク（閉域ネットワーク）をいう。

    オ. 自治体が保有する各種データ（公共施設の位置情報、統計データ等）について、機械判読に適した形式でのオープンデータ化は、国の方針として禁止されている。

    1. ア・ウ
    2. ア・オ
    3. イ・エ
    4. イ・オ
    5. ウ・エ
    解答・解説

    正解：4（イ・オ）

    ア. ○ RPAの定義として正しい。 イ. ✗ ガバメントクラウドは地方自治体も利用可能。むしろ地方自治体の情報システム標準化においてガバメントクラウドの活用が推進されている。 ウ. ○ eLTAXの定義として正しい。 エ. ○ LGWAN（総合行政ネットワーク）の定義として正しい。 オ. ✗ 自治体のオープンデータ化は禁止されておらず、むしろ国の方針として推奨・推進されている。

    中文要點: 政府雲端（ガバメントクラウド）地方自治體也可以使用。自治體的開放資料化是國家方針所推動的（非禁止）。RPA、eLTAX、LGWAN的定義均正確。

    問題 5（令和5年度 問55）¶

    情報通信用語に関する次の記述のうち、妥当でないものはどれか。

    1. リスクウェアとは、インストール・実行した場合にシステムにリスクをもたらす可能性のあるソフトウェアをいう。
    2. ランサムウェアとは、感染したコンピュータのデータを暗号化してロックし、使えない状態にしたうえで、データを復元する対価として金銭を要求するプログラムをいう。
    3. フリースウェアとは、無料トライアルなどを通して解除方法を知らせないままネットの利用者をサブスクリプションに誘導し、高額の利用料を請求するアプリをいう。
    4. ファームウェアとは、二軍を意味するファームからとられ、優れた性能を持ったアプリケーションのパフォーマンスを劣化させる悪性のプログラムである。
    5. クリッパー・マルウェアとは、感染したコンピュータのクリップボード情報を収集し悪用する機能を持つマルウェアをいい、仮想通貨を狙ったものが多い。
    解答・解説

    正解：4

    1. ○ リスクウェアの定義として正しい。
    2. ○ ランサムウェアの定義として正しい。
    3. ○ フリースウェアの定義として正しい。
    4. ✗ ファームウェア（firmware）とは、PC・スマートフォン等のハードウェアにあらかじめ組み込まれたソフトウェアのことであり、悪意のあるプログラムではない。「ファーム（farm）= 二軍」という語源説明も誤り。firmwareのfirmは「固い」を意味するfirmに由来する。
    5. ○ クリッパー・マルウェアの定義として正しい。

    中文要點: 韌體（firmware）是預先安裝在硬體中的軟體，非惡意程式。firmware的firm意為「固定的」，與farm（農場/二軍）無關。其他四個資安用語的定義均正確。

    問題 6（令和4年度 問56）¶

    情報通信に関する用語を説明した次のア～オの記述のうち、妥当なものの組合せはどれか。

    ア. オプトインとは、あらかじめ利用者の同意・許可を得たうえで広告メール等を送信する方式をいう。

    イ. プラットフォーム事業者とは、検索エンジンやSNS等のデジタルサービスを提供する事業者をいう。

    ウ. デジタルトランスフォーメーション（DX）とは、デジタル技術の活用により各国の政治体制を専制主義に変容させることをいう。

    エ. テレワークとは、テレビ電話システムの構築を基幹として、企業の業務のすべてを在宅で行う方式をいう。

    オ. ベース・レジストリとは、行政機関等において頻繁に利用するウェブサイトをブックマーク登録することをいう。

    1. ア・イ
    2. ア・ウ
    3. イ・エ
    4. ウ・オ
    5. エ・オ
    解答・解説

    正解：1（ア・イ）

    ア. ○ オプトインの定義として正しい。事前同意型の方式。 イ. ○ プラットフォーム事業者の定義として正しい。 ウ. ✗ デジタルトランスフォーメーション（DX）は、デジタル技術を活用して組織のビジネスモデルや業務プロセスを変革すること。政治体制の変容とは無関係。 エ. ✗ テレワークは、ICTを活用して場所や時間にとらわれない柔軟な働き方を指す。テレビ電話に限定されず、「業務のすべてを在宅で行う」ものでもない。 オ. ✗ ベース・レジストリ（基盤データ）とは、行政機関等で共通して参照される基盤的なデータベース（法人番号、住所等）のこと。ブックマーク登録とは無関係。

    中文要點: Opt-in是事前取得用戶同意的方式。DX是透過數位技術變革組織業務模式（非改變政治體制）。遠距工作不限於視訊電話，也非所有業務在家進行。基底登錄冊（ベース・レジストリ）是行政機關共通參照的基礎數據庫。

    問題 7（令和4年度 問57）¶

    個人情報保護制度に関する次の記述のうち、正しいものはどれか。

    1. 個人情報保護は国の制度がすべての地方公共団体に先行して整備された。
    2. 個人情報保護委員会は条例未制定の自治体に是正命令を発出しなければならない。
    3. 個人番号カードは各都道府県が交付している。
    4. 個人情報保護委員会は内閣総理大臣に地方公共団体への指揮監督権限行使を求める意見具申ができる。
    5. 個人情報保護委員会は認定個人情報保護団体に関する事務をつかさどる。
    解答・解説

    正解：5

    1. ✗ 個人情報保護に関しては、一部の地方公共団体の条例が国の法律に先行して整備された。
    2. ✗ 個人情報保護委員会にそのような是正命令の権限は規定されていない。
    3. ✗ 個人番号カード（マイナンバーカード）は都道府県ではなく、市町村長が交付する。
    4. ✗ そのような意見具申制度は存在しない。
    5. ○ 個人情報保護委員会は、認定個人情報保護団体の認定・監督等に関する事務をつかさどる。

    中文要點: 個資保護制度方面，部分地方自治體的條例比國家法律更早制定。個人編號卡由市町村長交付（非都道府縣）。個資保護委員會負責認定個人資訊保護團體的認定和監督事務。
