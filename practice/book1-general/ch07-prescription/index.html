    第7章 時効 — 練習問題¶

    本章出題傾向

    時効（第144条〜第174条の2）に関する問題は、過去10年間で約10〜12回出題。2020年改正で大幅変更された分野。 重要テーマ：取得時効、消滅時効、時効の援用、時効の完成猶予・更新

    択一式¶

    問題 1（令和元年度 問27）¶

    時効に関する次の記述のうち、民法の規定に照らし、妥当なものはどれか。

    1. 時効の効力は、その起算日に遡らず、時効が完成した時から将来に向かって生じる。
    2. 時効は、当事者が援用しなくても、裁判所がこれによって裁判をすることができる。
    3. 時効の利益は、あらかじめ放棄することができる。
    4. 消滅時効にあっては、保証人、物上保証人、第三取得者その他権利の消滅について正当な利益を有する者も時効を援用することができる。
    5. 取得時効にあっては、占有者本人のみが時効を援用することができ、その他の者は援用することができない。
    解答・解説

    正解：4

    1. ✗ 時効の効力は「その起算日に遡る」（民法144条）。遡及効がある。
    2. ✗ 時効は「当事者が援用しなければ、裁判所がこれによって裁判をすることができない」（民法145条）。
    3. ✗ 時効の利益は「あらかじめ放棄することができない」（民法146条）。事後の放棄は可能。
    4. ○ 民法145条かっこ書のとおり。2020年改正で明文化された。保証人・物上保証人・第三取得者等、権利の消滅について正当な利益を有する者も援用できる。
    5. ✗ 取得時効においても、占有者本人以外の者（例えば占有者の債権者）が援用できる場合がある。

    中文要點: 時效的三大原則：①效力溯及起算日；②必須由當事人援用（法院不能主動適用）；③不得事先拋棄時效利益。消滅時效的援用權人包括保證人、物上保證人、第三取得人等。

    📚 関連条文: 第144条

    問題 2（令和3年度 問30）¶

    取得時効に関する次の記述のうち、民法の規定及び判例に照らし、妥当なものはどれか。

    1. 20年間、所有の意思をもって、平穏に、かつ、公然と他人の物を占有した者は、その占有の開始の時に善意無過失であったか否かにかかわらず、その物の所有権を取得する。
    2. 10年間、所有の意思をもって、平穏に、かつ、公然と他人の物を占有した者は、その占有の開始の時に善意であれば、過失があっても、その物の所有権を取得する。
    3. 不動産の取得時効を主張する占有者は、取得時効の完成後に登記を経由しなければ、取得時効の完成後に当該不動産を取得した第三者に対抗することができない。
    4. 他人の土地を自己の土地と誤信して占有を開始した者の占有は、所有の意思のある占有（自主占有）とはいえない。
    5. 占有の承継があった場合において、前の占有者の占有を併せて主張するときは、前の占有者の瑕疵も承継するが、占有の期間のみを承継して瑕疵を承継しないことも選択できる。
    解答・解説

    正解：1

    1. ○ 民法162条1項。20年間の占有による取得時効は、善意無過失は要件ではない。
    2. ✗ 10年の短期取得時効（民法162条2項）には「善意」かつ「無過失」が必要。善意だけでは足りない。
    3. ○ とも考えられるが、判例の正確な理解が必要。取得時効完成後の第三者との関係は登記の先後によるが、時効完成後に再度時効期間が経過した場合は登記なくして対抗可能（最判昭36.7.20）。本問では1が正解。
    4. ✗ 他人の土地を自己のものと誤信した占有は「自主占有」に該当する（最判昭45.6.18）。所有の意思は、占有取得の原因たる事実によって外形的・客観的に判断される。
    5. ✗ 前の占有者の占有を併せて主張する場合は「その瑕疵をも承継する」（民法187条1項）。瑕疵のみ除外して期間だけ承継することはできない。

    中文要點: 取得時效的兩種期間：①20年（不問善意惡意）；②10年（必須善意且無過失）。所有意思以占有取得原因的外形客觀判斷，誤信他人的土地為自己的也構成自主占有。前占有的瑕疵必須一起承繼，不能只取期間不取瑕疵。

    📚 関連条文: 第144条

    問題 3（令和2年度 問29）¶

    消滅時効に関する次の記述のうち、2020年改正後の民法の規定に照らし、正しいものはどれか。

    1. 債権は、債権者が権利を行使することができることを知った時から10年間行使しないとき、時効によって消滅する。
    2. 債権は、権利を行使することができる時から20年間行使しないとき、時効によって消滅する。
    3. 人の生命又は身体の侵害による損害賠償請求権の消滅時効は、権利を行使することができる時から30年間である。
    4. 債権又は所有権以外の財産権は、権利を行使することができる時から20年間行使しないとき、時効によって消滅する。
    5. 不法行為による損害賠償請求権は、被害者又はその法定代理人が損害及び加害者を知った時から5年間行使しないとき、時効によって消滅する。
    解答・解説

    正解：4

    1. ✗ 主観的起算点からは「5年」で消滅する（民法166条1項1号）。10年ではない。
    2. ✗ 客観的起算点からは「10年」で消滅する（民法166条1項2号）。20年ではない（ただし人の生命・身体の侵害による場合は20年：167条）。
    3. ✗ 人の生命又は身体の侵害による損害賠償請求権は、客観的起算点から「20年」（民法167条）。30年ではない。
    4. ○ 民法166条2項。債権又は所有権以外の財産権（例：地上権、地役権）は、権利を行使することができる時から20年で消滅する。
    5. ✗ 不法行為の消滅時効は「3年」（民法724条1号）。5年ではない。ただし人の生命・身体を害する不法行為は5年（724条の2）。

    中文要點: 2020年改正後的消滅時效體系：①一般債權→主觀起算5年/客觀起算10年；②生命身體侵害→主觀5年/客觀20年；③不法行為→知情後3年/行為後20年；④生命身體不法行為→知情後5年/行為後20年；⑤所有權以外的財產權→20年。

    問題 4（令和4年度 問30）¶

    時効の完成猶予および更新に関する次の記述のうち、民法の規定に照らし、正しいものはどれか。

    1. 裁判上の請求は、訴えの却下又は取下げの場合でも、その終了の時から6か月を経過するまでの間は、時効は完成しない。
    2. 催告によって時効の完成が猶予されている間に再度催告をした場合、再度の催告によっても時効の完成猶予の効力が生じる。
    3. 権利についての協議を行う旨の合意が書面でされた場合、その合意があった時から1年を経過した時まで時効の完成が猶予されるが、この合意は口頭でも有効である。
    4. 確定判決によって権利が確定した場合、その確定の時から新たにその時効が進行する。
    5. 天災その他避けることのできない事変のため、裁判上の請求等ができないときは、その障害が消滅した時から1か月を経過するまでの間は、時効は完成しない。
    解答・解説

    正解：4

    1. ✗ 裁判上の請求は訴え却下・取下げの場合は時効の完成猶予の効力のみにとどまり（民法147条1項1号・2項）、6か月間の完成猶予がある。記述は正しいように見えるが、本問では4が最も正確。
    2. ✗ 催告によって完成猶予されている間に再度催告をしても、再度の催告には完成猶予の効力はない（民法150条2項）。催告の繰り返しによる時効完成の無限延長を防ぐ趣旨。
    3. ✗ 協議を行う旨の合意は「書面又は電磁的記録」でしなければならない（民法151条1項）。口頭では無効。
    4. ○ 民法169条1項。確定判決によって権利が確定した場合、時効は更新され、確定の時から新たに時効が進行する（新たな期間は10年）。
    5. ✗ 天災等の場合は障害が消滅した時から「3か月」を経過するまで時効は完成しない（民法161条）。1か月ではない。

    中文要點: 2020年改正的時效制度重點：①催告的完成猶予中不能再催告（禁止重複催告）；②協議合意須書面或電磁記錄（口頭無效）；③確定判決→時效更新，新期間10年；④天災→障礙消滅後3個月內時效不完成。

    問題 5（平成29年度 問30）¶

    時効に関する次のア～オの記述のうち、民法の規定及び判例に照らし、妥当でないものの組合せはどれか。

    ア. 時効の援用権者は、「当事者」とされており、消滅時効にあっては、保証人、物上保証人、第三取得者その他権利の消滅について正当な利益を有する者を含む。

    イ. 時効の利益をあらかじめ放棄することはできないが、時効完成後に時効の利益を放棄することは可能であり、これは相対効を有するため、他の援用権者には影響しない。

    ウ. 後順位抵当権者は、先順位抵当権の被担保債権の消滅時効を援用することができる。

    エ. 取得時効の起算点は、占有開始の時であり、占有者が任意にその起算点を選択することはできない。

    オ. 主たる債務者が時効の利益を放棄した場合でも、保証人は独立して消滅時効を援用することができる。

    1. ア・イ
    2. ア・ウ
    3. イ・エ
    4. ウ・エ
    5. ウ・オ
    解答・解説

    正解：4

    ア. ○ 民法145条かっこ書のとおり（2020年改正で明文化）。 イ. ○ 時効の利益の放棄は事前不可（民法146条）、事後は可能。放棄の効果は相対的であり、他の援用権者には影響しない（大判大5.12.25）。 ウ. ✗ 後順位抵当権者は先順位抵当権の被担保債権の消滅時効を援用する正当な利益を有しない（最判平11.10.21）。 エ. ✗ 判例は、取得時効の起算点について占有開始の時としつつも、起算点を後にずらすことを認めた判例がある（最判昭35.7.27）。本肢の「任意に選択できない」は妥当でない。 オ. ○ 主債務者の時効利益放棄は相対効であり、保証人は独立して時効を援用できる（大判昭7.6.21）。

    妥当でないものは ウとエ → 正解は 4。

    中文要點: 後順位抵當權人不能援用先順位抵當權被擔保債權的消滅時效（不具有正當利益）。時效利益的拋棄是相對效，主債務人拋棄不影響保證人的援用權。

    問題 6（令和5年度 問30）¶

    消滅時効の起算点に関する次の記述のうち、民法の規定及び判例に照らし、妥当でないものはどれか。

    1. 確定期限付の債権の消滅時効は、その期限が到来した時から進行する。
    2. 不確定期限付の債権の消滅時効は、その期限が到来した時から進行する。
    3. 期限の定めのない債権の消滅時効は、債権成立の時から進行する。
    4. 債務不履行による損害賠償請求権の消滅時効は、本来の債務の履行を請求しうる時から進行する。
    5. 不法行為による損害賠償請求権の消滅時効は、損害及び加害者を知った時から進行する。
    解答・解説

    正解：5 は妥当であり、妥当でないのは以下のいずれかの観点による

    1. ○ 確定期限付の場合、期限到来時から進行する（民法166条1項2号）。
    2. ○ 不確定期限付の場合も、期限到来時から客観的に進行する（民法166条1項2号）。
    3. ○ 期限の定めのない場合は、債権成立時から進行する（民法166条1項2号。直ちに請求できるため）。
    4. ○ 判例上、債務不履行損害賠償の消滅時効は本来の債務の履行請求可能時から進行する（最判平10.4.24）。
    5. ✗ 不法行為の消滅時効には「主観的起算点」と「客観的起算点」の両方がある。主観的起算点は「損害及び加害者を知った時」（民法724条1号）だが、客観的起算点は「不法行為の時から20年」（民法724条2号）。本肢は主観的起算点のみしか述べておらず不十分。ただし本肢自体は「妥当でない」とまでは言い切れない面もある。

    中文要點: 消滅時效起算點整理：①確定期限→期限到來時；②不確定期限→期限到來時（客觀起算點）；③無期限→債權成立時（可以立即請求）；④債務不履行損害賠償→本來的履行可以請求時。

    問題 7（平成30年度 問30）¶

    時効の援用に関する次の記述のうち、民法の規定及び判例に照らし、妥当なものはどれか。

    1. 時効の援用は、裁判上のみならず裁判外でも行うことができる。
    2. 時効の援用権は、相続の対象とならない一身専属的な権利である。
    3. 債務者が消滅時効を援用した後であっても、時効完成後に弁済をした場合には、時効の援用の効果は消滅する。
    4. 消滅時効の完成後、債務者が債務の承認をした場合、時効完成の事実を知らなかったとしても、その後に時効を援用することは許されないが、これは債務の承認が時効の利益の放棄にあたるからである。
    5. 取得時効の援用権者は占有者に限られ、占有者の相続人は援用権者に含まれない。
    解答・解説

    正解：1

    1. ○ 時効の援用は訴訟上で行うことも、裁判外で行うことも可能。方式の定めはない。
    2. ✗ 時効の援用権は相続の対象となる。一身専属的な権利ではない。
    3. ✗ 時効完成後に弁済をした場合、それは「時効の利益の放棄」（又は「信義則上再度の援用不可」）に該当しうるが、一度行った時効の援用自体が遡及的に消滅するわけではない。弁済が先に行われ、その後援用はできなくなるという構成。
    4. ✗ 判例（最判昭41.4.20）は、時効完成後の債務の承認があった場合、信義則上もはや時効を援用できないとするが、その理由は「時効の利益の放棄」ではなく「信義則」による。時効完成を知らなくても信義則上援用不可。
    5. ✗ 占有者の相続人は占有を承継するため（民法896条・187条）、援用権者に含まれる。

    中文要點: 時效援用可以在訴訟內外進行，沒有方式限制。時效完成後承認債務，即使不知道時效已完成，基於誠實信用原則也不能再援用時效（不是基於時效利益的拋棄，而是信義則）。

    📚 関連条文: 第145条

    問題 8（令和6年度 問30）¶

    時効に関する次の記述のうち、民法の規定及び判例に照らし、妥当でないものはどれか。

    1. 所有権は消滅時効にかからない。
    2. 所有権に基づく物権的請求権は、所有権とは独立して消滅時効にかかる。
    3. 占有者が占有物について行使する権利は、適法に有するものと推定される。
    4. 占有者は、所有の意思をもって、善意で、平穏に、かつ、公然と占有をするものと推定される。
    5. 前後の両時点において占有をした証拠があるときは、占有は、その間継続したものと推定される。
    解答・解説

    正解：2

    1. ○ 所有権は消滅時効にかからない（民法166条2項の反対解釈。所有権は除外されている）。
    2. ✗ 所有権に基づく物権的請求権は、所有権の作用であり、所有権と独立して消滅時効にかかることはない（最判昭6.1.16）。所有権が消滅時効にかからない以上、その作用たる物権的請求権も消滅時効にかからない。
    3. ○ 民法188条のとおり。
    4. ○ 民法186条1項のとおり。
    5. ○ 民法186条2項のとおり。

    中文要點: 所有權不受消滅時效限制，基於所有權的物權請求權也不獨立受消滅時效限制。占有推定：①推定適法行使權利；②推定善意、和平、公然占有；③前後兩時點有占有證據時，推定中間持續占有。

    記述式¶

    問題 1（令和2年度 問46）¶

    Ａは、Ｂに対して令和2年4月1日に弁済期が到来する100万円の貸金債権を有していたが、Ｂは弁済期を経過しても弁済しなかった。Ａは、令和7年5月1日に至り、はじめてＢに対して上記貸金債権の履行を請求した。この場合において、消滅時効に関する主張としてＢが主張すべき内容を、起算点と時効期間に触れつつ、40字程度で記述しなさい。

    解答・解説

    模範解答

    「Ｂは、弁済期である令和2年4月1日から5年が経過したため、消滅時効が完成したと主張すべきである。」（46字）

    別解

    「弁済期到来を知った時から5年間権利不行使により消滅時効が完成したとして、時効を援用すべきである。」（47字）

    解説

    2020年改正後の消滅時効制度では、債権は以下の2つの期間のうち早い方で消滅する：

    1. 主観的起算点：権利を行使することができることを知った時から5年（民法166条1項1号）
    2. 客観的起算点：権利を行使することができる時から10年（民法166条1項2号）

    本問では、弁済期（令和2年4月1日）にＡは権利行使可能であることを知っていたと考えられるため、主観的起算点から5年（令和7年4月1日に完成）が適用される。令和7年5月1日時点ではすでに時効が完成しているため、Ｂは消滅時効を援用すべきである。

    中文要點: 2020年改正後，一般債權的消滅時效：主觀起算點（知道可以行使權利時）起5年，客觀起算點（可以行使權利時）起10年，以先到者為準。貸金債權的清償期一般債權人會知道，所以從清償期起5年消滅。

    問題 2（令和4年度 問46）¶

    Ａは、Ｂ所有の甲土地を自己の土地であると信じ、善意無過失で、令和4年4月1日から平穏に、かつ、公然と占有を開始した。この場合において、Ａが甲土地の所有権を時効取得するために必要な期間と、その法的根拠を40字程度で記述しなさい。

    解答・解説

    模範解答

    「Ａは善意無過失であるから、民法162条2項により10年間の占有で甲土地の所有権を時効取得できる。」（46字）

    解説

    民法162条は取得時効について以下の2つの期間を定めている：

    1. 20年の取得時効（162条1項）：善意悪意を問わず、20年間所有の意思をもって平穏・公然と他人の物を占有した者は所有権を取得する。
    2. 10年の短期取得時効（162条2項）：10年間所有の意思をもって平穏・公然と他人の物を占有した者は、その占有の開始の時に善意であり、かつ、過失がなかったときは所有権を取得する。

    本問のＡは善意無過失であるから、162条2項の10年の短期取得時効が適用され、令和14年4月1日に時効が完成する。

    中文要點: 取得時效兩種期間：①20年（不問善惡意）；②10年（占有開始時善意且無過失）。善意無過失的判斷時點是「占有開始時」，中途變為惡意也不影響。
