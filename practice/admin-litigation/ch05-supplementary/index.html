    第5章 補則 — 練習問題¶

    本章出題傾向

    補則（第44条〜第46条）に関する問題は、過去10年間で約2〜3回出題。 重要テーマ：仮処分の排除（第44条）の趣旨と代替制度、教示制度（第46条）の対象と内容。

    択一式¶

    問題 1（令和元年度 問19）¶

    行政事件訴訟法第44条の仮処分の排除に関する次の記述のうち、妥当でないものはどれか。

    1. 行政庁の処分その他公権力の行使に当たる行為については、民事保全法に規定する仮処分をすることができない。
    2. 仮処分が排除される理由の一つは、行政行為の公定力により、処分が取り消されるまでは有効と扱われるため、仮処分による暫定的な停止を認めると行政の安定性が害されるからである。
    3. 仮処分が排除される代わりに、行政事件訴訟法は、執行停止（第25条）、仮の義務付け（第37条の5第1項）および仮の差止め（第37条の5第2項）という暫定的救済制度を設けている。
    4. 第44条の仮処分の排除は、公権力の行使に当たる行為についてのみ適用されるため、国を当事者とする私法上の法律関係に関する訴訟については、民事保全法に基づく仮処分が認められる。
    5. 第44条の仮処分の排除は、仮の義務付けおよび仮の差止めが2004年改正で新設されたことにより廃止され、現在は民事保全法に基づく仮処分も認められている。
    解答・解説

    正解：5

    1. ○ 第44条のとおり。公権力の行使にあたる行為については仮処分が排除される。
    2. ○ 行政行為の公定力との整合性が仮処分排除の重要な理由の一つである。仮処分を認めると、行政処分の効力を事実上否定することになり、行政の安定性が損なわれる。
    3. ○ 仮処分の排除に対応する暫定的救済制度として、執行停止（取消訴訟の場合）、仮の義務付け（義務付けの訴えの場合）、仮の差止め（差止めの訴えの場合）が設けられている。
    4. ○ 第44条は「行政庁の処分その他公権力の行使に当たる行為」について適用されるものであり、国が私人と同様の立場で行う私法上の行為には適用されない。したがって、国を当事者とする私法上の法律関係については仮処分が認められる。
    5. ✗ 第44条は現在も有効であり、廃止されていない。仮の義務付け・仮の差止めが新設されたのは、仮処分の排除を維持しつつ、行政訴訟固有の暫定的救済制度を充実させるためである。民事保全法の仮処分は依然として排除されている。

    中文要點: 第44條（假處分之排除）至今仍然有效，2004年改正並未廢止此條。改正是在維持假處分排除的前提下，新設行政訴訟固有的暫時權利保護制度（執行停止、假義務付、假差止）。私法上的法律關係不受第44條限制，仍可適用民事保全法的假處分。

    📚 関連条文: 第44条

    問題 2（平成28年度 問19）¶

    行政事件訴訟法第46条の教示制度に関する次の記述のうち、正しいものはどれか。

    1. 行政庁は、取消訴訟を提起することができる処分をする場合には、処分の相手方に対し、当該処分に係る取消訴訟の被告とすべき者、出訴期間および審査請求前置の有無を書面で教示しなければならない。
    2. 教示の対象となる事項には、取消訴訟の被告とすべき者および出訴期間が含まれるが、審査請求に関する事項は行政不服審査法の教示に委ねられているため含まれない。
    3. 教示制度は、処分の相手方のみならず、処分の相手方以外の利害関係者に対しても教示することを義務づけている。
    4. 行政庁が教示を怠った場合、当該処分は違法となり取り消される。
    5. 教示制度は、2004年改正により新設されたものであり、行政不服審査法における教示制度（不服申立てに関する教示）とは別の制度である。
    解答・解説

    正解：5

    1. ✗ 教示事項は、①取消訴訟の被告とすべき者、②取消訴訟の出訴期間、③法律に審査請求前置の定めがあるときはその旨である（第46条1項）。記述は概ね正しいが、「審査請求前置の有無」ではなく、審査請求前置の定めがあるときにその旨を教示する（定めがないときは教示不要）。
    2. ✗ 審査請求前置の定めがあるときはその旨も教示事項に含まれる（第46条1項3号）。行政不服審査法の教示とは別に、行政事件訴訟法独自の教示として規定されている。
    3. ✗ 教示義務は処分の相手方に対してのみ課されており（第46条1項）、利害関係者に対する教示義務は定められていない。
    4. ✗ 教示を怠ったことは処分自体の違法事由とはならない。教示の懈怠は処分の効力に影響しないが、出訴期間の徒過について「正当な理由」（第14条1項ただし書）が認められる余地がある。
    5. ○ 行政事件訴訟法の教示制度（第46条）は2004年改正で新設された。行政不服審査法の教示制度（不服申立ての方法や期間の教示）とは別個の制度であり、取消訴訟に関する被告適格・出訴期間等を処分の相手方に教示するものである。

    中文要點: 教示制度（第46條）是2004年改正新設的，要求行政廳在作成可提起撤銷訴訟的處分時，向處分相對人書面教示：①被告、②出訴期間、③有審查請求前置規定時其旨。教示義務僅對處分相對人，不包括利害關係人。怠於教示不影響處分的效力，但可能構成出訴期間經過的「正當理由」。
