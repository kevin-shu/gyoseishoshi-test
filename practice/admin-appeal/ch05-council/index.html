    第5章 行政不服審査会等 — 練習問題¶

    本章出題傾向

    行政不服審査会等（第67条〜第81条）は、平成26年改正で新設された第三者機関。過去10年間で約2〜3回出題。 重要テーマ：審査会の設置（総務省）、委員の人数（9人）・任命（両議院同意→総務大臣）・任期（3年）、合議体の構成（3人）

    択一式¶

    問題 1（令和2年度 問16）¶

    行政不服審査法に規定する行政不服審査会に関する次の記述のうち、妥当なものはどれか。

    1. 行政不服審査会は、内閣府に設置される。
    2. 行政不服審査会は、委員12人をもって組織される。
    3. 委員は、両議院の同意を得て、内閣総理大臣が任命する。
    4. 委員の任期は5年であり、再任されることができる。
    5. 行政不服審査会は、委員のうちから会長が互選により選任され、会長は会務を総理し、審査会を代表する。
    解答・解説

    正解：5

    1. ✗ 行政不服審査会は「総務省」に設置される（第67条第1項）。内閣府ではない。
    2. ✗ 行政不服審査会は「委員9人」をもって組織される（第68条第1項）。12人ではない。
    3. ✗ 委員は、両議院の同意を得て「総務大臣」が任命する（第69条第1項）。内閣総理大臣ではない。審査会が総務省に設置されているため、総務大臣が任命権者。
    4. ✗ 委員の任期は「3年」であり、5年ではない（第69条第4項）。再任は可能（同条第5項）。
    5. ○ 第70条のとおり。会長は委員の互選により選任され、会務を総理し審査会を代表する。

    中文要點: 行政不服審查會的基本組織：設於總務省、委員9人、兩議院同意後由總務大臣任命、任期3年（可連任）、原則非常勤（最多3人常勤）、會長由委員互選產生。考試常考的陷阱：設置機關（總務省而非內閣府）、任命權者（總務大臣而非內閣總理大臣）。

    📚 関連条文: 第67条、第68条、第69条、第70条

    問題 2（平成30年度 問16）¶

    行政不服審査法に規定する行政不服審査会の組織及び審査手続に関する次の記述のうち、正しいものはどれか。

    1. 行政不服審査会の委員は、すべて常勤でなければならない。
    2. 行政不服審査会は、委員全員による合議体で審査を行わなければならない。
    3. 行政不服審査会の委員は、在任中、政党その他の政治的団体の役員となり、又は積極的に政治運動をしてはならない。
    4. 行政不服審査会の委員が職務上知ることができた秘密を漏らした場合の罰則は定められていない。
    5. 行政不服審査会の委員を罷免するには、総務大臣の判断のみで足り、両議院の同意は必要ない。
    解答・解説

    正解：3

    1. ✗ 委員は原則「非常勤」とされている（第68条第2項本文）。ただし、そのうち3人以内は常勤とすることができる（同項ただし書）。全員常勤ではない。
    2. ✗ 行政不服審査会は、委員のうちから会長が指名する3人をもって構成する合議体で審査を行う（第71条第1項）。委員全員（9人）で審査するのではない。ただし、会長が指名する場合には全員で審査することも可能（同条第2項）。
    3. ○ 第69条第9項のとおり。委員は在任中の政治的中立性が求められ、政党等の役員就任や積極的政治運動が禁止されている。
    4. ✗ 第87条により、委員が秘密を漏らした場合は「1年以下の拘禁刑又は50万円以下の罰金」に処する旨の罰則が定められている。
    5. ✗ 第69条第7項により、委員の罷免には「両議院の同意」が必要。任命と同様の手続を要する。総務大臣の判断のみでは罷免できない。

    中文要點: 審查會的組織細節：委員原則非常勤（最多3人常勤）、合議體由3人構成（非全員9人）、委員須政治中立（禁止擔任政黨幹部、積極政治運動）、洩密有刑事罰則（1年以下拘禁或50萬日圓以下罰金）、罷免需兩議院同意（與任命同）。

    📚 関連条文: 第68条、第69条、第71条
