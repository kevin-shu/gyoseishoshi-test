    第3章 再調査の請求 — 練習問題¶

    本章出題傾向

    再調査の請求（第54条〜第61条）は、処分庁に対する簡易な不服申立て手段。過去10年間で約2〜3回出題。 重要テーマ：再調査の請求期間（第54条）、審査請求との競合処理（第56条）、決定と裁決の区別、審理員が不要であること

    択一式¶

    問題 1（令和3年度 問15）¶

    行政不服審査法に規定する再調査の請求に関する次の記述のうち、妥当でないものはどれか。

    1. 再調査の請求は、法律に再調査の請求をすることができる旨の定めがある場合に限り、処分庁に対してすることができる。
    2. 再調査の請求は、処分があったことを知った日の翌日から起算して3月を経過したときは、することができないが、正当な理由があるときはこの限りでない。
    3. 再調査の請求においては、処分庁が自ら再調査を行うため、審理員を指名する必要はない。
    4. 再調査の請求についての処分庁の判断は「裁決」と呼ばれ、審査請求の場合と同一の名称が用いられている。
    5. 再調査の請求をした者は、当該再調査の請求についての決定を経ないでも、審査請求をすることができる場合がある。
    解答・解説

    正解：4

    1. ○ 第5条第1項のとおり。再調査の請求は、法律に特別の定めがある場合に限られる。一般的に認められる審査請求とは異なる。
    2. ○ 第54条第1項のとおり。主観的期間は3月（審査請求と同じ）。客観的期間も1年で同じ（第54条第2項）。
    3. ○ 再調査の請求は処分庁自身が審理するため、審理員の指名は不要。第61条の準用規定でも第9条は準用されていない。
    4. ✗ 再調査の請求についての処分庁の判断は「決定」であり、「裁決」ではない（第58条・第59条参照）。「裁決」は審査請求及び再審査請求について用いられる用語である。
    5. ○ 第5条第2項ただし書により、再調査の請求をした日の翌日から起算して3月を経過しても決定がないとき等には、決定を経ないで審査請求が可能。

    中文要點: 再調查之請求的重要特徵：限於法律有規定時才能提起、由處分機關自行審理（不需審理員）、判斷結果稱為「決定」而非「裁決」。期間與審查請求相同（主觀3個月/客觀1年）。超過3個月無決定時，可不經決定直接提起審查請求。

    📚 関連条文: 第54条、第56条

    問題 2（平成29年度 問14）¶

    行政不服審査法に規定する再調査の請求と審査請求の関係に関する次の記述のうち、正しいものはどれか。

    1. 再調査の請求をした者は、当該再調査の請求についての決定を経た後でなければ、審査請求をすることができない。
    2. 再調査の請求をした者が、当該再調査の請求についての決定を経ないで審査請求をした場合、当該再調査の請求は当然に取り下げられたものとみなされる。
    3. 再調査の請求をした者が、決定を経ないで審査請求をした場合であっても、処分庁が審査請求がされた日以前に処分を取り消す旨の決定書の謄本を発しているときは、当該審査請求が取り下げられたものとみなされる。
    4. 処分庁は、再調査の請求がされた日の翌日から起算して1年を経過しても決定をしていないときは、書面で審査請求ができる旨を教示しなければならない。
    5. 再調査の請求と審査請求が同時に係属している場合、審査庁は再調査の請求の決定を待って裁決をしなければならない。
    解答・解説

    正解：3

    1. ✗ 第5条第2項ただし書により、再調査の請求についての決定を経ないでも審査請求をすることができる場合がある（3月経過して決定がないとき等）。再調査の請求前置主義ではない。
    2. ✗ 第56条本文により、決定を経ないで審査請求がされたときは、再調査の請求が取り下げられたものとみなされるのが原則であるが、「当然に」ではなく、例外がある（同条ただし書）。
    3. ○ 第56条ただし書のとおり。処分庁が審査請求がされた日以前に既に処分を取り消す旨の決定書の謄本を発している場合等には、逆に審査請求の方が取り下げられたものとみなされる。先に有利な決定が出ていればそれを優先する趣旨。
    4. ✗ 第57条により、教示義務が生じるのは「3月」を経過しても再調査の請求が係属しているときであり、1年ではない。
    5. ✗ そのような規定はない。第56条により、審査請求がされた場合には原則として再調査の請求が取り下げられたものとみなされるため、同時に係属し続けることは通常ない。

    中文要點: 再調查之請求與審查請求的競合處理（第56條）：未經決定即提起審查請求時，再調查之請求視為撤回。但若處分機關已先作出撤銷處分的決定，則審查請求視為撤回。3個月後未決定時，處分機關有義務教示可直接提起審查請求（第57條）。

    📚 関連条文: 第56条、第57条
