    第4章 再審査請求 — 練習問題¶

    本章出題傾向

    再審査請求（第62条〜第66条）は、裁決に対するさらなる不服申立て。過去10年間で約1〜2回出題。 重要テーマ：再審査請求期間（主観1月 — 審査請求より短い！）、取消しのみ可能（変更不可）、行政不服審査会への諮問不要

    択一式¶

    問題 1（令和4年度 問15）¶

    行政不服審査法に規定する再審査請求に関する次の記述のうち、妥当なものはどれか。

    1. 再審査請求は、審査請求の裁決に不服がある場合に、行政不服審査法の規定に基づき当然にすることができる。
    2. 再審査請求は、原裁決があったことを知った日の翌日から起算して3月を経過したときは、することができない。
    3. 再審査請求においては、再審査庁は、原裁決を取り消すことはできるが、これを変更することはできない。
    4. 再審査請求においても、審査庁は行政不服審査会に諮問しなければならない。
    5. 再審査請求において、原裁決が違法又は不当であるが撤銷により公益に著しい障害を生ずる場合であっても、事情裁決をすることはできない。
    解答・解説

    正解：3

    1. ✗ 第6条第1項により、再審査請求は「法律に再審査請求をすることができる旨の定めがある場合」に限りすることができる。行政不服審査法の規定に基づき当然に認められるわけではない。
    2. ✗ 再審査請求の主観的期間は「1月」であり、3月ではない（第62条第1項）。審査請求の3月よりも短い点に注意。
    3. ○ 第65条第1項により、再審査請求が理由がある場合には、再審査庁は原裁決等の全部又は一部を「取り消す」のみ。審査請求の場合（第46条第1項）と異なり、変更する権限はない。
    4. ✗ 第66条第1項の準用規定において、第43条（行政不服審査会への諮問）の規定のうち第1項は除外されていないが、実質的に第66条第1項の括弧書きにより第45条〜第49条が除外されており、再審査請求には独自の裁決規定（第64条・第65条）が適用される。なお、再審査請求については第43条は準用されない（第66条第1項の括弧書きで第四節が除外）。
    5. ✗ 第64条第4項により、再審査請求においても事情裁決をすることができる。原裁決等が違法又は不当であっても、取消しにより公益に著しい障害を生ずる場合には、棄却しつつ裁決主文で違法又は不当であることを宣言する。

    中文要點: 再審查請求的三大特徵：(1) 限法律有規定時才能提起（非當然權利）。(2) 主觀期間僅1個月（最短！）。(3) 再審查機關只能取消，不能變更原裁決。客觀期間仍為1年。不需向行政不服審查會諮問。

    📚 関連条文: 第62条、第65条

    問題 2（平成28年度 問15）¶

    行政不服審査法の不服申立て期間に関する次の記述のうち、正しいものの組合せはどれか。

    ア. 審査請求の主観的期間は、処分があったことを知った日の翌日から起算して3月である。

    イ. 再調査の請求の主観的期間は、処分があったことを知った日の翌日から起算して1月である。

    ウ. 再審査請求の主観的期間は、原裁決があったことを知った日の翌日から起算して1月である。

    エ. 審査請求・再調査の請求・再審査請求のいずれも、客観的期間は1年である。

    1. ア・イ
    2. ア・ウ
    3. イ・ウ
    4. ア・ウ・エ
    5. ア・イ・エ
    解答・解説

    正解：4

    ア. ○ 第18条第1項のとおり。審査請求の主観的期間は3月。 イ. ✗ 再調査の請求の主観的期間は「3月」であり、1月ではない（第54条第1項）。審査請求と同じ。 ウ. ○ 第62条第1項のとおり。再審査請求の主観的期間は1月。最も短い。 エ. ○ 審査請求（第18条第2項）、再調査の請求（第54条第2項）、再審査請求（第62条第2項）のいずれも、客観的期間は1年。

    期間比較表：

    類型 主観的期間 客観的期間
    審査請求 3月 1年
    再調査の請求 3月 1年
    再審査請求 1月 1年
    取消訴訟（行訴法） 6月 1年

    中文要點: 各類型不服申訴的主觀期間比較：審查請求3個月、再調查之請求3個月、再審查請求1個月（最短）、取消訴訟6個月（最長）。客觀期間全部相同為1年。再調查之請求容易被誤認為1個月，需注意與審查請求相同為3個月。

    📚 関連条文: 第18条、第54条、第62条
