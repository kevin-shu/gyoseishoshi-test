    第6章 補則 — 練習問題¶

    本章出題傾向

    補則（第82条〜第87条）は、教示制度が超頻出。過去10年間で約3〜5回出題。 重要テーマ：教示制度（第82条）— 処分相手方への主動教示 vs 利害関係人への被動教示、未教示の場合の救済（第83条）、行訴法46条との比較

    択一式¶

    問題 1（令和5年度 問15）¶

    行政不服審査法に規定する教示に関する次の記述のうち、妥当でないものはどれか。

    1. 行政庁は、審査請求等をすることができる処分をする場合には、処分の相手方に対し、不服申立てをすることができる旨等を書面で教示しなければならない。
    2. 処分を口頭でする場合には、書面による教示は不要である。
    3. 利害関係人から教示を求められた場合には、行政庁は当該事項を教示しなければならない。
    4. 利害関係人が書面による教示を求めたときであっても、行政庁は口頭で教示をすれば足りる。
    5. 教示すべき事項には、不服申立てをすべき行政庁及び不服申立てをすることができる期間が含まれる。
    解答・解説

    正解：4

    1. ○ 第82条第1項本文のとおり。処分の相手方に対する主動的教示義務。教示事項は、(1)不服申立てができる旨、(2)不服申立てをすべき行政庁、(3)不服申立ての期間の3つ。
    2. ○ 第82条第1項ただし書のとおり。処分を口頭でする場合には、書面による教示は不要。
    3. ○ 第82条第2項のとおり。利害関係人から求めがあったときは被動的教示義務が生じる。処分の相手方に限らず、利害関係人にも教示しなければならない。
    4. ✗ 第82条第3項により、教示を求めた者が「書面による教示を求めたとき」は、当該教示は書面でしなければならない。口頭では足りない。
    5. ○ 第82条第1項のとおり。教示事項は、不服申立てができる旨、不服申立てをすべき行政庁、不服申立てをすることができる期間である。

    中文要點: 教示制度的核心區別：對處分相對人→主動教示義務（不需對方請求，原則書面）。對利害關係人→被動教示義務（對方請求時才需教示）。利害關係人要求書面教示時，必須以書面回應。口頭處分時不需書面教示。

    📚 関連条文: 第82条

    問題 2（令和元年度 問16）¶

    行政不服審査法第83条に規定する教示をしなかった場合の不服申立てに関する次の記述のうち、正しいものはどれか。

    1. 行政庁が教示をしなかった場合には、当該処分について不服がある者は、直接裁判所に取消訴訟を提起しなければならない。
    2. 行政庁が教示をしなかった場合には、当該処分について不服がある者は、当該処分庁に不服申立書を提出することができる。
    3. 教示をしなかった場合の救済として不服申立書が処分庁に提出されたとき、処分庁はこれを審理し自ら裁決をしなければならない。
    4. 処分庁に提出された不服申立書が他の行政庁に送付された場合であっても、不服申立ての効力は送付された時点から発生する。
    5. 教示をしなかった場合の救済は、処分の相手方にのみ認められ、利害関係人には認められない。
    解答・解説

    正解：2

    1. ✗ 第83条は、教示がなかった場合に処分庁に不服申立書を提出できるという救済を定めており、直接裁判所に訴訟を提起しなければならないわけではない。
    2. ○ 第83条第1項のとおり。行政庁が第82条の教示をしなかった場合、「当該処分について不服がある者」は、当該処分庁に不服申立書を提出することができる。
    3. ✗ 第83条第3項により、当該処分が処分庁以外の行政庁に対し審査請求をすることができる処分であるときは、処分庁は速やかに不服申立書を当該行政庁に送付しなければならない。処分庁が自ら裁決するのではない。
    4. ✗ 第83条第4項により、不服申立書が送付されたときは「初めから」当該行政庁に審査請求がされたものとみなされる。送付時点からではなく、最初の提出時点に遡って効力が生じる。
    5. ✗ 第83条第1項は「当該処分について不服がある者」と規定しており、処分の相手方に限定していない。利害関係人にも認められると解される。

    中文要點: 未教示時的救濟（第83條）：不服者可向處分機關提出不服申訴書→處分機關判斷正確管轄機關後轉送→視為自始向正確機關提起。重點：效力追溯至最初提出時（而非轉送時），不限於處分相對人。

    📚 関連条文: 第82条、第83条

    問題 3（平成29年度 問16）¶

    行政不服審査法の教示制度と行政事件訴訟法の教示制度の比較に関する次の記述のうち、妥当なものはどれか。

    1. 行政不服審査法の教示制度は処分の相手方のみを対象とするのに対し、行政事件訴訟法の教示制度は利害関係人も対象とする。
    2. 行政不服審査法においては、利害関係人から教示を求められた場合にも教示義務が生じるが、行政事件訴訟法においては、取消訴訟を提起することができる処分をする場合に、処分の相手方に対して教示義務が生じる。
    3. 行政不服審査法も行政事件訴訟法も、教示をしなかった場合の救済措置として、処分庁に不服申立書等を提出することができる旨を定めている。
    4. 行政事件訴訟法は、処分の相手方に対する教示に加えて、利害関係人から求めがあった場合の教示についても規定している。
    5. 行政不服審査法の教示事項は不服申立てをすべき行政庁及び不服申立ての期間であるが、行政事件訴訟法の教示事項には被告とすべき者及び出訴期間のほか、裁決主義が定められている場合にはその旨も含まれる。
    解答・解説

    正解：5

    1. ✗ 逆である。行政不服審査法は処分の相手方（第82条第1項）及び利害関係人（同条第2項）を対象とする。行政事件訴訟法第46条は処分の相手方に対する教示のみを規定している。
    2. ✗ 前段は正しい（行審法第82条第2項）。しかし後段について、行訴法第46条は「取消訴訟を提起することができる処分又は裁決をする場合」に教示義務が生じるとしているが、利害関係人からの求めに対する教示規定は行訴法にはない。本肢の記述自体は正しいように見えるが、この肢は行審法と行訴法の教示の対象範囲の違いを正確に示していない。
    3. ✗ 行政不服審査法第83条は教示をしなかった場合の救済措置を定めているが、行政事件訴訟法には教示をしなかった場合の同様の救済措置は定められていない。
    4. ✗ 行政事件訴訟法第46条は処分の相手方に対する教示のみを規定しており、利害関係人から求めがあった場合の教示については規定していない。これは行政不服審査法（第82条第2項）にのみある規定。
    5. ○ 行政不服審査法の教示事項は、不服申立てができる旨、不服申立てをすべき行政庁、期間の3点。行政事件訴訟法第46条の教示事項は、被告とすべき者、出訴期間、裁決主義が定められている場合にはその旨（法律に当該処分についての審査請求に対する裁決を経た後でなければ訴えを提起できない旨の定めがあるときは、その旨）。

    中文要點: 行審法 vs 行訴法教示制度比較：行審法教示對象較廣（含利害關係人），行訴法僅限處分相對人。行審法有未教示的救濟措施（第83條），行訴法無。行訴法教示事項含被告、出訴期間、裁決前置主義之旨；行審法教示事項含申訴機關、期間。

    📚 関連条文: 第82条、第83条
