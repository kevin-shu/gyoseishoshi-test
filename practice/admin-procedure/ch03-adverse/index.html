    第3章 不利益処分 — 練習問題¶

    本章出題傾向

    不利益処分（第12条〜第31条）は行政手続法で最も出題頻度が高い章。過去10年間で毎年1〜3問出題。 超頻出テーマ：聴聞と弁明の機会の付与の振分け（第13条）、理由の提示（第14条）と一級建築士懲戒事件判例、聴聞手続の詳細（通知・文書閲覧・主宰者・調書）、聴聞と弁明の比較問題。 記述式では第14条の理由の提示の程度が繰り返し出題されている。

    択一式¶

    問題 1（令和元年度 問11）¶

    行政手続法に規定する聴聞と弁明の機会の付与に関する次の記述のうち、正しいものはどれか。

    1. 許認可等を取り消す不利益処分をしようとするときは、弁明の機会の付与の手続を執らなければならない。
    2. 名あて人の資格又は地位を直接にはく奪する不利益処分をしようとするときは、聴聞の手続を執らなければならない。
    3. 聴聞は書面によることが原則であり、行政庁が口頭ですることを認めたときに限り口頭で行うことができる。
    4. 弁明は口頭によることが原則であり、行政庁が書面ですることを認めたときに限り書面で行うことができる。
    5. 聴聞と弁明の機会の付与のいずれの手続においても、当事者は文書等の閲覧を求めることができる。
    解答・解説

    正解：2

    1. ✗ 許認可等を取り消す不利益処分は聴聞の対象である（第13条第1項第1号イ）。弁明ではない。
    2. ○ 第13条第1項第1号ロのとおり。資格・地位の直接はく奪は聴聞の対象。
    3. ✗ 聴聞は口頭が原則（出頭して意見を述べる。第20条第2項）。書面（陳述書）は出頭に代えて提出可能だが原則ではない。
    4. ✗ 弁明は書面が原則（弁明書の提出。第29条第1項）。行政庁が認めたときに限り口頭で行える。選択肢は原則と例外が逆。
    5. ✗ 文書等の閲覧（第18条）は聴聞手続にのみ認められ、弁明の機会の付与には認められない。

    中文要點: 聽證與辯明的區分是行政手續法最頻出的考點。聽證＝口頭為原則（重大處分）、辯明＝書面為原則（輕微處分）。文書閱覽權僅存在於聽證程序中。許認可等之取消、資格地位之剝奪、法人役員之解任命令都是聽證的對象。

    📚 関連条文: 第13条、第29条

    問題 2（令和4年度 問11）¶

    行政手続法に規定する聴聞手続に関する次の記述のうち、妥当なものはどれか。

    1. 行政庁は、聴聞を行うに当たっては、聴聞を行うべき期日までに相当な期間をおいて、不利益処分の名あて人となるべき者に対し、口頭又は書面により通知しなければならない。
    2. 聴聞の期日における審理は、公開が原則であり、行政庁が非公開とすることを相当と認めるときに限り非公開とすることができる。
    3. 当事者又は参加人は、主宰者の許可を得ることなく、行政庁の職員に対し質問を発することができる。
    4. 主宰者は、聴聞の終結後速やかに、不利益処分の原因となる事実に対する当事者等の主張に理由があるかどうかについての意見を記載した報告書を作成し、聴聞調書とともに行政庁に提出しなければならない。
    5. 行政庁は、聴聞を経てされる不利益処分の決定をするときは、主宰者の報告書に記載された意見に拘束される。
    解答・解説

    正解：4

    1. ✗ 聴聞の通知は書面によらなければならない（第15条第1項）。「口頭又は書面」ではない。
    2. ✗ 聴聞の審理は非公開が原則であり、行政庁が公開を相当と認めるときに限り公開する（第20条第6項）。原則と例外が逆。
    3. ✗ 行政庁の職員に対する質問は、主宰者の許可を得なければならない（第20条第2項）。
    4. ○ 第24条第3項のとおり。主宰者は聴聞終結後速やかに報告書を作成し、聴聞調書とともに行政庁に提出する。
    5. ✗ 行政庁は主宰者の意見を「十分に参酌」しなければならないが、拘束はされない（第26条）。

    中文要點: 聽證通知必須以書面為之、審理原則非公開、質問須經主持人許可、行政機關不受主持人意見之拘束（僅充分斟酌）。這些都是常見的考試陷阱。報告書由主持人作成，記載其對當事人主張是否有理由的意見。

    📚 関連条文: 第15条、第20条、第24条、第26条

    問題 3（平成28年度 問12）¶

    行政手続法に規定する聴聞と弁明の機会の付与の比較に関する次の記述のうち、妥当でないものはどれか。

    1. 聴聞においては、当事者は代理人を選任することができ、弁明の機会の付与においても、第16条の準用により代理人を選任することができる。
    2. 聴聞においては、当事者及び参加人は文書等の閲覧を求めることができるが、弁明の機会の付与においては文書等の閲覧は認められていない。
    3. 聴聞の手続に基づく処分又はその不作為については審査請求をすることができないが、弁明の機会の付与の手続にはこのような制限はない。
    4. 聴聞においては、当事者以外の者であっても利害関係を有する者は参加人として手続に参加できるが、弁明の機会の付与にはそのような参加人の制度はない。
    5. 聴聞の通知は書面によらなければならないが、弁明の機会の付与の通知は口頭で行うこともできる。
    解答・解説

    正解：5

    1. ○ 第16条により聴聞では代理人選任が可能。弁明の機会の付与でも第31条により第16条が準用され、代理人の選任が可能。
    2. ○ 文書等の閲覧（第18条）は聴聞にのみ認められ、弁明の機会の付与には準用されない。
    3. ○ 第27条は聴聞の節に規定され、聴聞手続に基づく処分等について審査請求を制限している。弁明の機会の付与にはこの制限はない。
    4. ○ 第17条により聴聞には参加人制度があるが、弁明の機会の付与にはこの制度がない。
    5. ✗ 弁明の機会の付与の通知も書面によらなければならない（第30条）。口頭で行うことはできない。

    中文要點: 聽證與辯明的主要差異：(1)文書閱覽權——僅聽證、(2)參加人制度——僅聽證、(3)審查請求限制——僅聽證、(4)代理人——兩者皆可（辯明準用第16條）、(5)通知方式——兩者皆須書面。常見陷阱是將辯明的通知誤認為可口頭進行。

    📚 関連条文: 第13条、第18条、第27条、第31条

    問題 4（令和2年度 問12）¶

    不利益処分の理由の提示に関する次の記述のうち、最高裁判所の判例に照らし、妥当でないものはどれか。

    1. 不利益処分をする場合には、その名あて人に対し、同時に、当該不利益処分の理由を示さなければならないのが原則である。
    2. 不利益処分を書面でするときは、理由も書面により示さなければならない。
    3. 一級建築士に対する懲戒処分において、処分の理由として根拠法条のみを示すことは、行政手続法第14条第1項本文の趣旨に照らし、十分な理由の提示とはいえない。
    4. 理由の提示の程度は、当該処分の性質・内容、理由の提示を命じた法規の趣旨・目的等を総合考慮して判断すべきであり、いかなる処分においても常に同一の程度が要求されるわけではない。
    5. 理由を示さないで処分をすべき差し迫った必要がある場合であっても、事後に理由を示す必要はない。
    解答・解説

    正解：5

    1. ○ 第14条第1項本文のとおり。不利益処分の理由の提示は「同時に」が原則。
    2. ○ 第14条第3項のとおり。書面処分には書面で理由を示す。
    3. ○ 一級建築士懲戒事件（最判平23.6.7）。処分基準の適用関係を示すことなく、単に根拠法条と処分内容を示すのみでは、処分の名あて人において、いかなる事実関係に基づきいかなる法規を適用して処分がされたかを、その記載自体から了知し得るものとはいえず、理由の提示として不十分。
    4. ○ 最判平23.6.7の判示。理由の提示の程度は処分の性質・内容等を総合考慮して判断される。
    5. ✗ 第14条第2項により、差し迫った必要がある場合に理由を示さず処分した場合でも、名あて人の所在が判明しなくなったとき等を除き、処分後相当の期間内に理由を示さなければならない。

    中文要點: 一級建築士懲戒事件（最判平23.6.7）是理由提示相關最重要的判例。理由提示必須讓相對人能從記載本身了知係基於何種事實、適用何種法規而受處分。僅記載法條而未記載具體事實和處分基準的適用關係是不充分的。即使因急迫必要而未提示理由，事後仍須補提示。

    📚 関連条文: 第14条

    問題 5（平成27年度 問11）¶

    行政手続法に規定する聴聞・弁明の機会の付与に関する判例の次の記述のうち、妥当でないものはどれか。

    1. 個人タクシー事件判決（最判昭46.10.28）は、多数の申請者のうちから少数の特定の者を免許の相手方として選択する処分において、申請人に聴聞その他公正な手続を行うことが必要であるとした。
    2. 群馬中央バス事件判決（最判昭50.5.29）は、路線の免許申請に対する運輸審議会の公聴会において、反対意見の者に対し十分な反論の機会が与えられなかったことは、審議会の審理手続に瑕疵があるとした。
    3. 一級建築士懲戒事件判決（最判平23.6.7）は、不利益処分における理由の提示は、処分基準の存在を前提として、当該基準の適用関係が示されなければならないとした。
    4. 個人タクシー事件判決は、行政手続法の制定以前の判決であるが、行政手続法はこの判決の趣旨を立法化したものと解されている。
    5. 行政手続法上、聴聞又は弁明の機会の付与が必要となる場合であっても、公益上緊急に不利益処分をする必要があるときは、これらの手続を省略することが認められている。
    解答・解説

    正解：3

    1. ○ 個人タクシー事件（最判昭46.10.28）は、多数の競願者の中から選択する免許処分において、申請人に聴聞その他公正な手続を経るべきことを要求した先駆的判例。
    2. ○ 群馬中央バス事件（最判昭50.5.29）は、公聴会における審理手続の瑕疵を認め、運輸大臣の免許拒否処分を違法とした。
    3. ✗ 一級建築士懲戒事件（最判平23.6.7）は、処分基準の適用関係が示されるべきとしたが、処分基準が設定されていない場合でも理由の提示自体は必要であり、「処分基準の存在を前提として」という限定は不正確。判例は処分基準の有無にかかわらず、処分の原因事実と法令の適用関係が了知しうる程度の理由提示を求めている。
    4. ○ 個人タクシー事件は昭和46年の判決であり、行政手続法は平成5年に制定された。同法は判例法理を立法化した側面がある。
    5. ○ 第13条第2項第1号により、公益上緊急の必要がある場合は聴聞・弁明の手続を省略できる。

    中文要點: 個人計程車事件與群馬中央巴士事件是行政手續法制定前的重要判例，確立了行政處分中公正程序的必要性。一級建築士懲戒事件判例要求的理由提示，並不以處分基準的存在為前提——即使未制定處分基準，也必須提示足以讓相對人了知處分根據的理由。

    📚 関連条文: 第13条、第14条

    問題 6（令和3年度 問44）¶

    一級建築士であるAは、建築士法に基づく懲戒処分として業務停止処分を受けた。当該処分の通知書には、処分の根拠法条と処分内容が記載されていたが、処分基準の適用関係や具体的な非違事実の詳細な摘示はなされていなかった。Aが当該処分の取消しを求めて訴訟を提起した場合、最高裁判所の判例に照らし、行政手続法第14条第1項本文の定める理由の提示としてどのような問題があるか。40字程度で記述しなさい。

    解答・解説

    解答例: 「いかなる事実関係に基づきいかなる法規を適用したかを、記載自体から了知し得ず、理由提示が不十分である。」（48字）

    別解: 「処分基準の適用関係が示されておらず、名あて人が処分の根拠を記載自体から了知し得ないため違法である。」（47字）

    中文要點: 一級建築士懲戒事件（最判平23.6.7）的核心論點：不利益處分的理由提示，必須達到讓相對人能從通知書的記載本身了知「係基於何種事實關係、適用何種法規」而受處分的程度。僅記載根據法條和處分內容，未記載處分基準的適用關係及具體違規事實者，屬於理由提示不充分，該處分違法。

    ポイント整理:

    • 最判平23.6.7の判旨：「行政手続法14条1項本文が、不利益処分をする場合に同時にその理由を名あて人に示さなければならないとしているのは、名あて人に直接に義務を課し又はその権利を制限するという不利益処分の性質に鑑み、行政庁の判断の慎重と合理性を担保してその恣意を抑制するとともに、処分の理由を名あて人に知らせて不服の申立てに便宜を与える趣旨に出たもの」
    • 理由提示の程度は「処分の原因となる事実及び処分の根拠法条に加えて、本件処分基準の適用関係が示されなければ」十分とはいえない
    • 処分基準が設定・公表されている場合、その適用関係の記載も求められる

    📚 関連条文: 第14条
