    第7〜8章 留置権・先取特権 — 練習問題¶

    本章出題傾向

    留置権・先取特権に関する問題は、過去10年間で約2〜3回出題。 重要テーマ：留置権の成立要件（牽連性・善管注意義務）・消滅原因、先取特権の追及力消滅（引渡しと占有改定）

    択一式¶

    問題 1（令和3年度 問30）¶

    留置権に関する次の記述のうち、民法の規定および判例に照らし、妥当なものはどれか。

    1. 留置権者は善良な管理者の注意をもって留置物を占有すべきであるが、その注意は自己財産に対するのと同一の注意より軽減されたものである。
    2. 留置権者は債務者の承諾を得なければ留置物について使用・賃貸・担保供与をなすことができず、承諾なく使用した場合、留置権は直ちに消滅する。
    3. 建物賃借人が賃料不払いで契約解除された後に有益費を支出した場合、賃貸人の明渡請求に対し、賃借人は有益費償還請求権を被担保債権として建物を留置することはできない。
    4. 売主Aが買主Bに不動産を売却したが、所有権移転登記未了の段階で当該不動産をBからCに転売しCが登記を具備した場合、AはBに対する代金未払債権を被担保債権としてCに対し留置権を行使できる。
    5. A所有の建物をBに売却し引渡した後、代金の支払いを受けていない段階で当該建物が他者Cに転売されBが登記を失った場合、BはAに対する売買不履行に基づく損害賠償請求権を被担保債権として留置権を行使できる。
    解答・解説

    正解：3

    1. ✗ 留置権者は「善良な管理者の注意（善管注意義務）」をもって留置物を占有する義務を負う（第298条第1項）。善管注意義務は自己の財産に対する注意より重い義務であり、「軽減されたもの」は誤り。
    2. ✗ 第298条第2項により、留置権者は債務者の承諾なく留置物を使用・賃貸・担保供与することができない。しかし、承諾なく使用した場合に留置権が「直ちに消滅する」とは規定されておらず、消滅請求（第298条第3項）の手続きが必要。
    3. ○ 判例（最判昭46.7.16）により、債務不履行（賃料不払い）による契約解除後は無権原で占有することになり、その後に支出した有益費については、第295条第2項を類推適用して留置権を行使することはできないとされている。
    4. ✗ 留置権における「物と債権の牽連関係」の要件から、AはCに対して当初物の占有を有していたわけではなく、AのBに対する代金債権とCが占有する物との間に牽連関係が認められない。
    5. ✗ Bが物の占有を失った後は、留置権を行使するための占有継続の要件を欠く。また損害賠償請求権と物との牽連関係も認められない。

    中文要點: 留置権の成立要件：①他人の物の占有、②被担保債権と物の牽連関係、③債権の弁済期到来。 契約解除後の無権原占有者が支出した費用 → 留置権行使不可（第295条第2項類推）。 善管注意義務（善良な管理者の注意）= 通常人の注意基準 → 自己の財産より重い基準。

    📚 関連条文: 第295条〜第302条

    問題 2（令和2年度 問28 関連）¶

    先取特権に関する次の記述のうち、民法の規定および判例に照らし、妥当なものはどれか。

    1. 先取特権は法律の規定によって当然に生じる法定担保物権であり、設定行為を必要としないため、その成立のためにも登記は必要ない。
    2. 動産の先取特権は、その目的動産が債務者の手を離れて第三取得者に引き渡された場合、追及力を失うが、この「引渡し」には現実の引渡しのみが含まれ、占有改定は含まれない。
    3. 不動産の先取特権（不動産保存の先取特権）は、登記をすれば、その不動産に設定された抵当権に対しても優先して弁済を受けることができる。
    4. 先取特権の目的物上に抵当権が設定されている場合、先取特権者は常に抵当権者に優先して弁済を受けることができる。
    5. 一般の先取特権は、特別の先取特権に対して常に優先する。
    解答・解説

    正解：3

    1. ✗ 先取特権は法定担保物権であり設定行為は不要だが、不動産の先取特権は登記を要する。動産先取特権は登記が不要なものが多いが、不動産保存・不動産工事・不動産売買の先取特権は登記が必要（第337条〜第340条）。「一切登記不要」は誤り。
    2. ✗ 第333条（先取特権の追及力消滅）の「引渡し」には、判例（最判昭54.2.15）により、占有改定も含まれる。占有改定による引渡しがあれば先取特権の動産への追及力は消滅する。
    3. ○ 第339条により、不動産の保存の先取特権（工事代金等）は、保存行為後直ちに登記をすれば、その登記前に設定された抵当権にも優先する。不動産の先取特権の中で最も強い効力を持つ。
    4. ✗ 先取特権と抵当権の優先順位は第339条等の規定に基づき判断される。先取特権が「常に」抵当権に優先するわけではない。
    5. ✗ 第335条第2項により、一般の先取特権は不動産については特別の先取特権に劣後する。「常に優先する」は誤り。

    中文要點: 先取特権の追及力消滅（第333条）のポイント：目的動産が第三取得者に「引き渡された」場合 → 追及できない。この「引渡し」には占有改定も含む（最判昭54.2.15）。 不動産保存の先取特権（第339条）は最優先 → 登記があれば既存の抵当権にも優先。
