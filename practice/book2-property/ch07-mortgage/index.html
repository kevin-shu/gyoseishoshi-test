    第10章 抵当権 — 練習問題¶

    本章出題傾向

    抵当権に関する問題は、過去10年間で毎年1〜2問出題される最頻出分野。 重要テーマ：抵当権の効力の範囲（従物・附加一体物）、物上代位（第304条・第372条）、法定地上権（第388条）、一括競売（第389条）、根抵当権

    択一式¶

    問題 1（平成30年度 問30）¶

    抵当権の効力に関する次の記述のうち、民法の規定および判例に照らし、妥当なものはどれか。

    1. 抵当権の効力は抵当不動産の従物にも及ぶが、抵当不動産とは別個に従物について対抗要件を具備しなければ、その旨を第三者に対して対抗することができない。
    2. 借地上の建物に抵当権が設定された場合において、その建物の抵当権の効力は、特段の合意がない限り借地権には及ばない。
    3. 買戻特約付売買の買主が目的不動産について買主の債権者のために抵当権を設定し、その旨の登記がなされたところ、その後、売主が買戻権を行使した場合、買主が売主に対して有する買戻代金債権につき、上記抵当権者は物上代位権を行使することができる。
    4. 抵当不動産が転貸された場合、抵当権者は、原則として、転貸料債権（転貸賃料請求権）に対して物上代位権を行使することができる。
    5. 抵当権者が、被担保債権について利息および遅延損害金を請求する権利を有するときは、抵当権者は、原則として、それらの全額について優先弁済権を行使することができる。
    解答・解説

    正解：3

    1. ✗ 判例（最判平2.4.19）により、抵当権設定時に抵当不動産の従物であれば、設定登記により従物にも抵当権の効力が及び、従物について別個の対抗要件を具備する必要はない。
    2. ✗ 判例（最判昭40.5.4）により、借地上の建物に抵当権が設定された場合、その借地権にも原則として抵当権の効力が及ぶ（建物と借地権は一体として把握）。「及ばない」は誤り。
    3. ○ 判例（最判平11.11.30）により、買戻特約付売買において、買主が設定した抵当権者は、買戻権行使後に買主が取得する買戻代金債権に対して物上代位権を行使することができる。
    4. ✗ 判例（最判平12.4.14）により、転貸賃料債権に対しては、原則として物上代位権を行使することができない（転貸人である賃借人と転借人の間の契約関係に直接介入できない）。通常の賃料債権とは異なる扱い。
    5. ✗ 第375条により、抵当権者が優先弁済を受けられる利息・遅延損害金は、満期となった最後の2年分に限られる（利息等の制限）。全額ではない。

    中文要點: 物上代位（第372条・第304条）の重要ポイント： - 通常の賃料債権 → 物上代位OK（差押え前に取立て等がなければ） - 転貸賃料債権 → 原則として物上代位不可（最判平12.4.14） - 利息の優先弁済 → 最後の2年分のみ（第375条）

    📚 関連条文: 第369条〜第375条

    問題 2（平成23年度 問30）¶

    法定地上権に関する次の記述のうち、民法の規定および判例に照らし、妥当なものはどれか。

    1. Aが自己所有の更地（建物のない土地）に抵当権を設定した後にその土地上に建物を建築したが、弁済できなくなって抵当権が実行された場合、建物のために法定地上権が成立し、一括競売はすることができない。
    2. Aが借地上の建物に甲抵当権を設定したところ、その後AがBから当該土地を取得して乙抵当権を設定し、甲抵当権が弁済により消滅した後に乙抵当権が実行されて買受人Cが土地を取得した場合、建物のために法定地上権は成立しない。
    3. Aが借地上の建物に甲抵当権を設定したところ、Aが当該土地を取得して乙抵当権を設定し、甲抵当権が実行されて買受人Cが建物を取得した場合、建物のために法定地上権は成立しない。
    4. Aが土地と建物に共同抵当権を設定したところ、建物が滅失したためAがその後新たに建物を建築した場合において、再築された建物について共同抵当権者が同順位の共同抵当権の設定を受けたなどの特段の事情がない限り、再築建物のために法定地上権は成立しない。
    5. Aが単独所有する敷地上にAとBが共有する建物が存在する場合において、AがBの同意なく土地に抵当権を設定し、その後抵当権が実行されて第三者Cが土地を取得したとき、建物のために法定地上権は成立する。
    解答・解説

    正解：4

    1. ✗ 抵当権設定時に更地（建物なし）であれば、法定地上権の成立要件（設定時に土地上に建物が存在すること）を欠くため、法定地上権は成立しない。ただし、一括競売（第389条）は任意で行うことができる（義務ではない）。
    2. ✗ 乙抵当権設定時にはAが土地と建物を所有する状態（同一人帰属）になっており、乙抵当権実行時の観点では同一人帰属が成立している。判例（最判昭52.10.11）により、法定地上権は成立する。「成立しない」は誤り。
    3. ✗ 甲抵当権（建物のみ）設定時、Aは借地権によって土地を使用していた。その後Aが土地を取得し乙抵当権を設定。甲抵当権実行時、建物取得者Cのために法定地上権が成立する（最判昭37.9.4）。「成立しない」は誤り。
    4. ○ 判例（最判平9.2.14）により、土地・建物に共同抵当権設定後に建物が滅失し新築された場合、「再築建物について共同抵当権者が同順位の共同抵当権設定を受けたなどの特段の事情がない限り、新建物のために法定地上権は成立しない」とされている。
    5. ✗ 土地が単独所有でも、建物が共有の場合に土地のみに抵当権を設定すると、法定地上権の成立は複雑な判断となる。判例（最判昭44.11.4）は原則として成立を認めない立場をとるケースもある。一律に「成立する」とは断言できない。

    中文要點: 法定地上権（第388条）の成立3要件： ①抵当権設定時に建物が存在する（更地への設定 → 法定地上権なし） ②抵当権設定時に土地と建物が同一人に帰属する ③抵当権実行後、土地と建物が異なる所有者になる

    共同抵当権設定後の建物滅失・再築 → 特段の事情なければ法定地上権不成立（最判平9.2.14）。

    📚 関連条文: 第388条〜第389条

    問題 3（平成26年度 問30）¶

    根抵当権に関する次の記述のうち、民法の規定および判例に照らし、妥当でないものはどれか。

    1. 根抵当権の担保すべき元本の確定前においては、根抵当権者と設定者は、合意によって根抵当権の被担保債権の範囲を変更することができるが、この変更は、後順位担保権者の承諾がなくても、その登記をすることによって効力を生ずる。
    2. 根抵当権の元本の確定前に根抵当権者から被担保債権の範囲に属する個別の債権を譲渡された場合、当該債権の譲受人は根抵当権を取得しない。
    3. 根抵当権の担保すべき元本が確定した後は、被担保債権の範囲及び債務者を変更することはできない。
    4. 根抵当権の担保すべき元本の確定前においては、根抵当権は、根抵当権者が有するすべての債権を担保しているため、当事者の合意によっても、被担保債権の範囲を限定することはできない。
    5. 根抵当権設定者は、元本の確定期日の定めがない場合、根抵当権の設定の時から3年を経過したときは、担保すべき元本の確定を請求することができる。
    解答・解説

    正解：4

    1. ○ 第398条の4により、元本確定前は、根抵当権者と設定者の合意（後順位担保権者の承諾不要）によって被担保債権の範囲・債務者を変更できる。ただし、登記をしなければ第三者に対抗できない。
    2. ○ 第398条の7第1項により、元本確定前に被担保債権を譲渡しても、根抵当権は随伴しない（付従性の緩和）。個別の債権の譲受人は根抵当権を取得しない。
    3. ○ 第398条の4第3項により、元本確定後は被担保債権の範囲・債務者の変更はできない。
    4. ✗ 根抵当権は「被担保債権の範囲」を定めて設定するものであり（第398条の2第1項）、根抵当権者のすべての債権を担保するわけではない。被担保債権の範囲は設定時に限定され、かつ変更も可能（合意＋登記）。「すべての債権を担保しているため限定できない」は誤り。
    5. ○ 第398条の19第1項により、確定期日の定めがない根抵当権は、設定の時から3年を経過すれば設定者が確定請求できる。

    中文要點: 根抵当権（第398条の2〜第398条の22）の特徴： - 普通抵当権と違い、「特定の債権」ではなく「一定の範囲の不特定の債権」を担保 - 付従性の緩和：被担保債権が全額消滅しても根抵当権は消滅しない（元本確定前） - 随伴性なし：元本確定前に個別債権を譲渡しても根抵当権は随伴しない（第398条の7）

    📚 関連条文: 第398条の2〜第398条の22
