    第1章 総則（物権変動・登記） — 練習問題¶

    本章出題傾向

    物権変動・登記に関する問題は、過去10年間で約3〜4回出題。 重要テーマ：不動産の二重譲渡と登記の対抗力（第177条）、動産の即時取得（第178条）、物権的請求権

    択一式¶

    問題 1（平成29年度 問31）¶

    物権的請求権等に関する次の記述のうち、民法の規定および判例に照らし、妥当なものはどれか。

    1. A所有の土地上にBが権原なく建物を建設しCに譲渡した場合、AはBに対してのみ建物の取去を求めることができる。
    2. 第三者が抵当不動産を不法占有する場合、抵当権は占有を目的としないため、抵当権者は占有者に直接明渡しを求めることはできない。
    3. 占有者が占有を奪われた場合、占有回収の訴えにより返還請求はできるが、損害賠償請求はできない。
    4. 賃借人は、対抗要件の有無を問わず、不法占有者に対して妨害排除請求ができる。
    5. D所有の土地上にEが権原なく建物を建設・登記後Fに譲渡しても、登記がE名義のままである場合、DはEに建物収去を求めることができる。
    解答・解説

    正解：5

    1. ✗ Aは建物の現在の所有者（C）に対しても建物収去・土地明渡しを求めることができる（最判昭35.6.17）。
    2. ✗ 判例（最大判平11.11.24）により、抵当権者は、抵当不動産を不法占有する者に対して、抵当権に基づく妨害排除請求として明渡しを求めることができる。
    3. ✗ 占有回収の訴え（第200条）では、返還請求に加えて損害賠償請求も認められている（第201条第1項）。
    4. ✗ 賃借人は、対抗要件（登記等）を具備している場合に限り、不法占有者に対して妨害排除請求ができる（最判昭28.12.18）。
    5. ○ 土地所有者Dは、建物の登記名義人Eに対して建物収去・土地明渡しを求めることができる（最判昭35.6.17）。登記名義人は登記上の所有者として責任を負う。

    中文要點: 地上物（建物）的撤除請求，可以對建物登記名義人提出。登記名義人即使已將建物讓渡給他人，只要登記仍在其名下，土地所有人就可以對其主張撤除請求。

    📚 関連条文: 第175条〜第179条

    問題 2（平成30年度 問29）¶

    Aが登記簿上の所有名義人である甲土地をBが買い受ける旨の契約をA・B間で締結した場合に関する次のア〜オの記述のうち、民法の規定および判例に照らし、妥当なものの組合せはどれか。

    ア　甲土地は実際にはCの所有に属していたが、CがAに無断で甲土地の所有名義人をAとしていた場合において、Aがその事情を知らないBとの間で本件売買契約を締結したときであっても、BはCに対して甲土地の引渡しを求めることができない。

    イ　甲土地はAの所有に属していたところ、Aの父であるDが、Aに無断でAの代理人と称して本件売買契約を締結し、その後Dが死亡してAがDを単独で相続したときは、Aは、Dの法律行為の追認を拒絶することができ、また、損害賠償の責任を免れる。

    ウ　甲土地が相続によりAおよびEの共有に属していたところ、AがEに無断でAの単独所有名義の登記をしてBとの間で本件売買契約を締結し、Bが所有権移転登記をした場合において、Bがその事情を知らず、かつ、過失がないときは、Bは甲土地の全部について所有権を取得する。

    エ　甲土地はAの所有に属していたところ、本件売買契約が締結され、B名義での所有権移転の仮登記がされた場合において、Aが甲土地をその事情を知らないFに売却し所有権移転登記をしたときは、Bは本登記をしない限りFに対して所有権の取得を対抗することができない。

    オ　甲土地はAの所有に属していたところ、GがAに無断で甲土地上に建物を築造し、その建物の所有権保存登記をした場合において、本件売買契約により甲土地の所有者となったBは、Gが当該建物の所有権を他に譲渡していたとしても、登記名義がGにある限り、Gに対して当該建物の収去および土地の明渡しを求めることができる。

    1. ア・ウ
    2. ア・オ
    3. イ・ウ
    4. イ・エ
    5. エ・オ
    解答・解説

    正解：5（エ・オ）

    ア　✗ 不動産登記に公信力はないが、真の権利者が登記名義を他人に無断で移転されていた場合、善意の第三者（B）は、第94条第2項類推等として保護されうる。Bが保護されない旨の断定は誤り。

    イ　✗ 無権代理人を相続した本人（A）は、無権代理行為を追認したものとみなされる場合があり（最判昭40.6.18）、損害賠償責任も免れない。

    ウ　✗ 不動産の登記には公信力がない。共有者の一人（A）が単独所有名義で登記していても、Bはその真実の持分（E分）を超えて所有権を取得することはできない。

    エ　○ 仮登記には対抗力がなく、本登記をしない限り第三者Fに対して所有権取得を対抗できない。

    オ　○ 土地所有者Bは、建物の登記名義人Gに対して建物収去・土地明渡しを求めることができる（最判昭35.6.17）。

    中文要點: 不動産登記沒有公信力（不像動産的善意取得），真正所有人可以對抗善意第三人。仮登記（假登記）只有順位保全效力，沒有對抗效力，本登記前不能對抗第三人。

    📚 関連条文: 第177条〜第178条

    問題 3（平成29年度 問29）¶

    物権の成立に関する次のア〜オの記述のうち、民法の規定および判例に照らし、妥当でないものの組合せはどれか。

    ア　他人の土地の地下または空間の一部について、工作物を所有するため、上下の範囲を定めて地上権を設定することは認められない。

    イ　一筆の土地の一部について、所有権を時効によって取得することは認められる。

    ウ　構成部分の変動する集合動産について、一括して譲渡担保の目的とすることは認められない。

    エ　土地に生育する樹木について、明認方法を施した上で、土地とは独立した目的物として売却することは認められる。

    オ　地役権は、継続的に行使され、かつ、外形上認識することができるものに限り、時効によって取得することができる。

    1. ア・イ
    2. ア・ウ
    3. イ・エ
    4. ウ・エ
    5. エ・オ
    解答・解説

    正解：2（ア・ウ）

    ア　✗ 第269条の2第1項により、「地下または空間は、工作物を所有するため、上下の範囲を定めて地上権の目的とすることができる」と規定されている。地下・空間の地上権（区分地上権）は認められる。

    イ　○ 判例（最判昭35.7.27）により、一筆の土地の一部についても時効取得は可能とされている。ただし、分筆登記なしでの対抗はできない。

    ウ　✗ 判例（最判昭54.2.15）により、構成部分が変動する集合動産であっても、「その種類、所在場所及び量的範囲を指定するなど何らかの方法で目的物の範囲が特定」されれば、譲渡担保の目的とすることができる。

    エ　○ 明認方法（立木等に氏名等を書き記す公示方法）を施すことで、樹木を土地から独立した物として取引することができる（慣習法上の制度）。

    オ　○ 第283条のとおり。地役権の時効取得には「継続的に行使され、かつ、外形上認識することができる」ことが必要。

    中文要點: ア（第269条の2：地下・空間地上権は認められる）とウ（最判昭54.2.15：集合動産の譲渡担保は可能）が誤り。「認められない」と書いてあるが、実際には「認められる」。

    📚 関連条文: 第175条〜第179条, 第269条の2
